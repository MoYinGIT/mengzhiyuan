# 编程数学基础

> 适合阶段：Scratch L1-L2 · 6-9岁

---

## 🎯 本章目标

学习编程中常用的数学概念：
- 坐标系（位置定位）
- 角度与方向（旋转控制）
- 随机数（游戏不确定性）
- 正负数（比较与判断）

## 1️⃣ 坐标系

### 生活中的坐标

| 场景 | 坐标表示 |
|:---|:---|
| 电影院座位 | 5排8号 |
| 地图位置 | 东经120°，北纬30° |
| 棋盘位置 | C4（C列4行） |

### Scratch 坐标系

Scratch 舞台使用直角坐标系：

**舞台尺寸**：宽480像素，高360像素

| 位置 | x 坐标 | y 坐标 |
|:---|:---:|:---:|
| 中心点 | 0 | 0 |
| 最左边 | -240 | 任意 |
| 最右边 | 240 | 任意 |
| 最上边 | 任意 | 180 |
| 最下边 | 任意 | -180 |

### 练习：画正方形

```scratch
当绿旗被点击
落笔
移到 x: (-50) y: (-50)

// 第一条边
移动 (100) 步

// 第二条边
右转 (90) 度
移动 (100) 步

// 第三条边
右转 (90) 度
移动 (100) 步

// 第四条边
右转 (90) 度
移动 (100) 步

抬笔
```

**思考问题**：为什么每次要右转90度？

## 2️⃣ 角度与方向

### 方向表示

| 角度 | 方向 |
|:---:|:---|
| 0° | 向上 |
| 90° | 向右 |
| 180° | 向下 |
| -90° 或 270° | 向左 |

### 顺时针 vs 逆时针

- **右转**：顺时针（角度增加）
- **左转**：逆时针（角度减少）

### 练习：画五角星

```scratch
当绿旗被点击
落笔
移到 x: (0) y: (0)

重复执行 (5) 次
    移动 (100) 步
    右转 (144) 度  // 为什么是144度？
结束

抬笔
```

**数学原理**：五角星的每个外角是144度。

## 3️⃣ 随机数

### 什么是随机数？

随机数是在一定范围内**不确定**的数。

### 应用场景

| 场景 | 随机数范围 |
|:---|:---|
| 骰子 | 1-6 |
| 猜数字游戏 | 1-100 |
| 抽奖 | 根据参与人数 |
| 游戏敌人出现位置 | 舞台范围内 |

### Scratch 中的随机数

```scratch
// 产生1-10的随机数
随机数取 (1) 到 (10)

// 应用：猜数字游戏
当绿旗被点击
将 目标数字 设为 随机数取 (1) 到 (100)
说 "我想了一个1-100的数字，你猜是多少？"
```

### 练习：随机画圆

```scratch
当绿旗被点击
重复执行 (10) 次
    落笔
    移到 x: (随机数取 (-200) 到 (200)) y: (随机数取 (-150) 到 (150))
    将 画笔颜色 设为 随机数取 (1) 到 (200)
    重复执行 (36) 次
        移动 (10) 步
        右转 (10) 度
    结束
    抬笔
结束
```

## 4️⃣ 正负数

### 正负数的意义

| 场景 | 正数 | 负数 |
|:---|:---|:---|
| 温度 | 零上 | 零下 |
| 楼层 | 地上 | 地下 |
| 坐标 | 右/上 | 左/下 |
| 存款 | 存入 | 取出 |

### 比较大小

在数轴上，越往右的数越大：

```
←────────────────────────────────→
-100  -50   0   50   100
      小 ←———→ 大
```

### 编程应用

**判断角色在舞台左侧还是右侧**：

```scratch
当绿旗被点击
重复执行
    如果 <(x坐标) > (0)> 那么
        说 "我在右边" 持续 (1) 秒
    否则
        说 "我在左边" 持续 (1) 秒
    结束
结束
```

## 🎮 综合项目：小猫走迷宫

### 项目目标

让小猫从迷宫入口走到出口，练习坐标和方向控制。

### 数学概念

- 坐标定位
- 方向控制
- 距离计算

### 提示

1. 先规划路线，确定每个转折点的坐标
2. 使用「移动」和「转向」积木
3. 可以加上「碰到边缘就反弹」防止走出舞台

## 📝 练习题

### 练习1：坐标计算

小猫在 (50, 100)，它需要：
1. 向右移动100步，新位置是？(____, ____)
2. 再向下移动50步，新位置是？(____, ____)
3. 最后向左移动150步，新位置是？(____, ____)

### 练习2：角度计算

一个三角形，三个内角分别是60°、60°、60°。
如果要用画笔在Scratch中画出这个三角形，每次需要转多少度？

**提示**：外角 = 180° - 内角

### 练习3：随机数概率

掷一个六面骰子，掷出偶数的概率是多少？

## 🏛️ 三经典智慧

### 《易经》· 象数思维

《易经》用数字和图像描述世界变化：
- 阴阳 = 二进制（0和1）
- 八卦 = 3位二进制
- 六十四卦 = 6位二进制

**编程就是现代的象数之学**。

### 《道德经》· 有无相生

0不是没有意义：
- 0是正数和负数的分界
- 0是加法的单位元
- 在计算机中，0和1构成了整个世界
