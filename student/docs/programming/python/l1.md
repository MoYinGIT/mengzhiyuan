## 📚 模块一：Python 环境搭建

### 1.1 安装 Python

**Windows 安装步骤**：

1. 访问 [python.org/downloads](https://www.python.org/downloads/)
2. 点击下载 Python 3.x（最新版）
3. 运行安装程序，**重要**：勾选「Add Python to PATH」
4. 选择「Install Now」
5. 等待安装完成

**Mac 安装步骤**：

1. 打开终端（Terminal）
2. 输入：`brew install python3`（需先安装 Homebrew）
3. 或从官网下载安装包

**验证安装**：

打开命令提示符（Windows）或终端（Mac），输入：
```bash
python --version
```

应显示类似：`Python 3.11.0`

### 1.2 第一个 Python 程序

**使用 IDLE（Python 自带编辑器）**：

1. 打开 IDLE
2. 输入：
```python
print("你好，世界！")
```
3. 按 `F5` 运行

**使用命令行**：

1. 创建文件 `hello.py`
2. 输入代码：
```python
print("你好，世界！")
```
3. 在命令行运行：
```bash
python hello.py
```

**输出结果**：
```
你好，世界！
```

🎉 恭喜你，写出了第一个 Python 程序！

## 📚 模块二：基础语法

### 2.1 注释

注释是给人看的，程序不会执行。

```python
# 这是单行注释

"""
这是多行注释
可以写很多行
"""

print("Hello")  # 注释也可以写在代码后面
```

### 2.2 变量

**变量就是给数据起个名字**。

```python
# 创建变量
name = "小明"      # 字符串（文字）
age = 10          # 整数
height = 1.45     # 浮点数（小数）
is_student = True # 布尔值（真/假）

# 使用变量
print(name)
print(age)
print(name, "今年", age, "岁")
```

**输出**：
```
小明
10
小明 今年 10 岁
```

**变量命名规则**：
- ✅ 可以用：字母、数字、下划线
- ✅ 必须以字母或下划线开头
- ❌ 不能用：空格、特殊符号
- ❌ 不能用：Python 关键字（如 if、for、print）

```python
# 好的命名
user_name = "张三"
score_1 = 85
MAX_SIZE = 100

# 不好的命名
2name = "李四"    # 数字开头，错误！
my name = "王五"  # 有空格，错误！
```

### 2.3 数据类型

| 类型 | 说明 | 示例 |
|:---|:---|:---|
| `int` | 整数 | `10`, `-5`, `0` |
| `float` | 浮点数 | `3.14`, `-0.5` |
| `str` | 字符串 | `"Hello"`, `'世界'` |
| `bool` | 布尔 | `True`, `False` |

**类型转换**：

```python
# 字符串转整数
age_str = "12"
age = int(age_str)  # 12

# 整数转字符串
num = 100
num_str = str(num)  # "100"

# 字符串转浮点数
price = float("19.99")  # 19.99
```

### 2.4 运算符

**算术运算符**：

```python
a = 10
b = 3

print(a + b)   # 13（加）
print(a - b)   # 7（减）
print(a * b)   # 30（乘）
print(a / b)   # 3.333...（除）
print(a // b)  # 3（整除）
print(a % b)   # 1（取余）
print(a ** b)  # 1000（幂运算，10的3次方）
```

**比较运算符**：

```python
a = 5
b = 3

print(a > b)   # True（大于）
print(a < b)   # False（小于）
print(a == b)  # False（等于）
print(a != b)  # True（不等于）
print(a >= b)  # True（大于等于）
print(a <= b)  # False（小于等于）
```

**字符串运算**：

```python
# 字符串拼接
first = "Hello"
second = "World"
print(first + " " + second)  # Hello World

# 字符串重复
print("哈" * 3)  # 哈哈哈
```

## 📚 模块三：输入输出

### 3.1 输出 print

```python
# 基本输出
print("Hello")

# 输出多个内容
print("姓名：", "小明", "年龄：", 10)

# 自定义分隔符
print("2024", "03", "23", sep="-")  # 2024-03-23

# 自定义结尾（默认换行）
print("Loading", end="")

```

### 3.2 输入 input

```python
# 获取用户输入
name = input("请输入你的名字：")
print("你好，", name)

# 输入的是字符串，需要转换
age = input("请输入你的年龄：")
age = int(age)  # 转成整数
print("明年你就", age + 1, "岁了")

# 简写
age = int(input("请输入你的年龄："))
```

**完整示例：个人信息录入**

```python
print("===== 个人信息录入 =====")
name = input("姓名：")
age = int(input("年龄："))
height = float(input("身高（米）："))

print("\n===== 你的信息 =====")
print("姓名：", name)
print("年龄：", age, "岁")
print("身高：", height, "米")
print("明年", age + 1, "岁")
```

## 📚 模块四：条件判断

### 4.1 if 语句

```python
score = 85

if score >= 90:
    print("优秀！")
elif score >= 80:
    print("良好")
elif score >= 60:
    print("及格")
else:
    print("不及格")
```

**注意**：
- `if`、`elif`、`else` 后面要加冒号 `:`
- 缩进很重要（通常4个空格）
- `elif` 可以有很多个

### 4.2 嵌套条件

```python
age = 15
has_ticket = True

if age >= 18:
    if has_ticket:
        print("请进！")
    else:
        print("请先购票")
else:
    print("未成年人不能进入")
```

### 4.3 逻辑运算符

```python
# and（与）：两个条件都满足
age = 25
income = 5000
if age >= 18 and income >= 3000:
    print("符合申请条件")

# or（或）：满足其中一个
day = "周六"
if day == "周六" or day == "周日":
    print("今天是周末")

# not（非）：条件反过来
is_raining = False
if not is_raining:
    print("不下雨，可以出去玩")
```

## 📚 模块五：循环

### 5.1 for 循环

```python
# 循环 5 次
for i in range(5):
    print(i)  # 输出 0, 1, 2, 3, 4

# 从 1 到 5
for i in range(1, 6):
    print(i)  # 输出 1, 2, 3, 4, 5

# 步长为 2
for i in range(0, 10, 2):
    print(i)  # 输出 0, 2, 4, 6, 8
```

**range 函数**：
- `range(5)` → 0, 1, 2, 3, 4
- `range(1, 6)` → 1, 2, 3, 4, 5
- `range(0, 10, 2)` → 0, 2, 4, 6, 8

### 5.2 while 循环

```python
# 数到 5
count = 1
while count <= 5:
    print(count)
    count = count + 1
```

**示例：猜数字游戏**

```python
import random

secret = random.randint(1, 100)
count = 0

while True:
    guess = int(input("猜一个 1-100 的数字："))
    count = count + 1
    
    if guess < secret:
        print("太小了！")
    elif guess > secret:
        print("太大了！")
    else:
        print(f"恭喜你猜对了！用了 {count} 次")
        break
```

### 5.3 循环控制

```python
# break：提前结束循环
for i in range(10):
    if i == 5:
        break  # 遇到 5 就停止
    print(i)  # 输出 0, 1, 2, 3, 4

# continue：跳过当前循环
for i in range(10):
    if i % 2 == 0:
        continue  # 跳过偶数
    print(i)  # 输出 1, 3, 5, 7, 9
```

## 📚 模块六：Turtle 绘图

### 6.1 Turtle 入门

```python
import turtle

# 创建画笔
pen = turtle.Turtle()

# 画一条线
pen.forward(100)  # 前进 100 像素

# 保持窗口显示
turtle.done()
```

### 6.2 常用命令

```python
import turtle

pen = turtle.Turtle()

# 移动
pen.forward(100)   # 前进
pen.backward(50)   # 后退
pen.right(90)      # 右转 90 度
pen.left(45)       # 左转 45 度

# 画笔控制
pen.penup()        # 抬笔（移动不画）
pen.pendown()      # 落笔
pen.pensize(5)     # 画笔粗细
pen.pencolor("red") # 画笔颜色

# 形状
pen.circle(50)     # 画圆（半径 50）
pen.dot(20)        # 画点（直径 20）

# 填充颜色
pen.fillcolor("yellow")
pen.begin_fill()
# ... 画图形 ...
pen.end_fill()

turtle.done()
```

### 6.3 画正方形

```python
import turtle

pen = turtle.Turtle()

for i in range(4):
    pen.forward(100)
    pen.right(90)

turtle.done()
```

### 6.4 画五角星

```python
import turtle

pen = turtle.Turtle()
pen.pencolor("red")
pen.pensize(3)

for i in range(5):
    pen.forward(100)
    pen.right(144)  # 五角星外角 144 度

turtle.done()
```

### 6.5 画彩色螺旋

```python
import turtle

colors = ["red", "orange", "yellow", "green", "blue", "purple"]

pen = turtle.Turtle()
pen.speed(0)  # 最快速度

for i in range(100):
    pen.pencolor(colors[i % 6])
    pen.forward(i * 2)
    pen.right(59)  # 螺旋角度

turtle.done()
```

## 🎮 综合项目

### 项目一：计算器

```python
def calculator():
    print("===== 简易计算器 =====")
    
    num1 = float(input("请输入第一个数："))
    op = input("请输入运算符（+ - * /）：")
    num2 = float(input("请输入第二个数："))
    
    if op == "+":
        result = num1 + num2
    elif op == "-":
        result = num1 - num2
    elif op == "*":
        result = num1 * num2
    elif op == "/":
        if num2 == 0:
            print("错误：不能除以 0")
            return
        result = num1 / num2
    else:
        print("错误：未知的运算符")
        return
    
    print(f"结果：{num1} {op} {num2} = {result}")

calculator()
```

### 项目二：九九乘法表

# 九九乘法表
for i in range(1, 10):

        print(f"{j}×{i}={i*j}", end="\t")

### 项目三：绘制雪花

```python
import turtle

def draw_branch(pen, length):
    if length < 5:
        return
    
    pen.forward(length)
    pen.right(45)
    draw_branch(pen, length / 2)
    pen.left(90)
    draw_branch(pen, length / 2)
    pen.right(45)
    pen.backward(length)

pen = turtle.Turtle()
pen.speed(0)
pen.left(90)
pen.penup()
pen.backward(100)
pen.pendown()

draw_branch(pen, 100)

turtle.done()
```
