## 📚 模块一：Scratch vs C++ 对照

### 1.1 基本结构对照

| 概念 | Scratch | C++ | 说明 |
|:---|:---|:---|:---|
| 程序开始 | 绿旗被点击 | `int main()` | 入口点 |
| 输出 | 说 "Hello" | `cout << "Hello"` | 打印输出 |
| 输入 | 询问并等待 | `cin >> x` | 获取输入 |
| 变量 | 变量块 | `int x = 10` | 存储数据 |

### 1.2 变量与数据类型

**Scratch**：
```scratch
将 [分数 v] 设为 [0]
将 [分数 v] 增加 [10]
```

**C++ 对应**：
```cpp
int score = 0;    // 整数
score = score + 10;
// 或
score += 10;
```

**数据类型对比**：
| Scratch | C++ | 说明 |
|:---|:---|:---|
| 数字 | `int` | 整数 |
| 数字 | `double` | 小数 |
| 字符串 | `string` | 文本 |
| 布尔 | `bool` | 真/假 |

### 1.3 控制结构对照

#### 条件语句

**Scratch**：
```scratch
如果 <(年龄) > (18)> 那么
    说 [成年了！]
否则
    说 [未成年]
结束
```

**C++**：
```cpp
if (age > 18) {
    cout << "成年了！";
} else {
    cout << "未成年";
}
```

#### 循环语句

**Scratch - 重复执行**：
```scratch
重复执行 (10) 次
    移动 (10) 步
结束
```

**C++ - for 循环**：
```cpp
for (int i = 0; i < 10; i++) {
    // 移动10步的代码
}
```

**Scratch - 重复执行直到**：
```scratch
重复执行直到 <碰到边缘？>
    移动 (10) 步
结束
```

**C++ - while 循环**：
```cpp
while (!碰到边缘) {
    // 移动10步的代码
}
```

## 📚 模块二：用 Scratch 学算法

### 2.1 冒泡排序

**算法思想**：两两比较，大的往后冒泡

**Scratch 实现**：
```scratch
定义 [冒泡排序 v]
重复执行 (长度 of [数据 v]) 次
    重复执行 ((长度 of [数据 v]) - (当前次数)) 次
        如果 <(item (i) of [数据 v]) > (item (i+1) of [数据 v])> 那么
            // 交换
            将 [临时 v] 设为 (item (i) of [数据 v])
            replace item (i) of [数据 v] with (item (i+1) of [数据 v])
            replace item (i+1) of [数据 v] with (临时)
        结束
    结束
结束
```

**C++ 对应**：
```cpp
void bubble_sort(int arr[], int n) {
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                swap(arr[j], arr[j + 1]);
            }
        }
    }
}
```

### 2.2 线性查找

**算法思想**：逐个检查，找到目标

**Scratch 实现**：
```scratch
定义 [线性查找 v] (目标)
将 [i v] 设为 [1]
重复执行 (长度 of [数据 v]) 次
    如果 <(item (i) of [数据 v]) = (目标)> 那么
        说 (连接 [找到了！位置是：] 和 (i))
        停止 [这个脚本 v]
    结束
    将 [i v] 增加 [1]
结束
说 [没找到]
```

**C++ 对应**：
```cpp
int linear_search(int arr[], int n, int target) {
    for (int i = 0; i < n; i++) {
        if (arr[i] == target) {
            return i;  // 找到了
        }
    }
    return -1;  // 没找到
}
```

### 2.3 二分查找

**算法思想**：每次排除一半，快速定位

**前提**：数据必须是有序的

**Scratch 实现**：
```scratch
定义 [二分查找 v] (目标)
将 [左 v] 设为 [1]
将 [右 v] 设为 (长度 of [数据 v])
重复执行直到 <(左) > (右)>
    将 [中 v] 设为 ((左) + (右)) / (2)
    如果 <(item (中) of [数据 v]) = (目标)> 那么
        说 (连接 [找到了！位置是：] 和 (中))
        停止 [这个脚本 v]
    结束
    如果 <(item (中) of [数据 v]) < (目标)> 那么
        将 [左 v] 设为 ((中) + (1))
    否则
        将 [右 v] 设为 ((中) - (1))
    结束
结束
说 [没找到]
```

**C++ 对应**：
```cpp
int binary_search(int arr[], int n, int target) {
    int left = 0, right = n - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (arr[mid] == target) return mid;
        if (arr[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}
```

## 📚 模块三：列表（数组）进阶

### 3.1 列表操作对比

| 操作 | Scratch | C++ |
|:---|:---|:---|
| 添加 | `add x to list` | `arr[n++] = x` |
| 插入 | `insert x at i` | 需要移动元素 |
| 删除 | `delete i of list` | 需要移动元素 |
| 长度 | `length of list` | 需要额外变量记录 |
| 访问 | `item i of list` | `arr[i]` |

### 3.2 求最大值

**Scratch**：
```scratch
将 [最大值 v] 设为 (item (1) of [数据 v])
将 [i v] 设为 [2]
重复执行 ((长度 of [数据 v]) - (1)) 次
    如果 <(item (i) of [数据 v]) > (最大值)> 那么
        将 [最大值 v] 设为 (item (i) of [数据 v])
    结束
    将 [i v] 增加 [1]
结束
```

**C++**：
```cpp
int max_val = arr[0];
for (int i = 1; i < n; i++) {
    if (arr[i] > max_val) {
        max_val = arr[i];
    }
}
```

### 3.3 求平均值

**Scratch**：
```scratch
将 [总和 v] 设为 [0]
重复执行 (长度 of [数据 v]) 次
    将 [总和 v] 增加 (item (i) of [数据 v])
结束
将 [平均值 v] 设为 ((总和) / (长度 of [数据 v]))
```

**C++**：
```cpp
double sum = 0;
for (int i = 0; i < n; i++) {
    sum += arr[i];
}
double avg = sum / n;
```

### 4.1 定义函数对比

**Scratch**：

### 4.2 带返回值的函数

**Scratch**：
```scratch
定义 [求和 v] (a) (b)
报告 ((a) + (b))
结束定义

将 [结果 v] 设为 (求和 [3] [5])  // 结果是8
```

**C++**：
```cpp
int add(int a, int b) {
    return a + b;
}

int result = add(3, 5);  // 结果是8
```

### 4.3 递归思想（初步）

**Scratch - 阶乘**：
```scratch
定义 [阶乘 v] (n)
如果 <(n) = (1)> 那么
    报告 [1]
否则
    报告 ((n) * (阶乘 ((n) - (1))))
结束
结束定义
```

**C++**：
```cpp
int factorial(int n) {
    if (n == 1) return 1;
    return n * factorial(n - 1);
}
```

## 🎮 综合项目：简易计算器

**功能**：支持加减乘除四则运算

**Scratch 实现框架**：
```scratch
当 [绿旗 v] 被点击
询问 [请输入第一个数：] 并等待
将 [a v] 设为 (回答)
询问 [请输入运算符(+ - * /)：] 并等待
将 [op v] 设为 (回答)
询问 [请输入第二个数：] 并等待
将 [b v] 设为 (回答)

如果 <(op) = [+]> 那么
    说 (连接 [结果是：] 和 ((a) + (b)))
结束
如果 <(op) = [-]> 那么
    说 (连接 [结果是：] 和 ((a) - (b)))
结束
如果 <(op) = [*]> 那么
    说 (连接 [结果是：] 和 ((a) * (b)))
结束
如果 <(op) = [/]> 那么
    说 (连接 [结果是：] 和 ((a) / (b)))
结束
```

**C++ 对应代码**：
```cpp
#include <iostream>
using namespace std;

int main() {
    double a, b;
    char op;
    
    cout << "请输入第一个数：";
    cin >> a;
    cout << "请输入运算符(+ - * /)：";
    cin >> op;
    cout << "请输入第二个数：";
    cin >> b;
    
    if (op == '+') cout << "结果是：" << a + b;
    else if (op == '-') cout << "结果是：" << a - b;
    else if (op == '*') cout << "结果是：" << a * b;
    else if (op == '/') cout << "结果是：" << a / b;
    
    return 0;
}
```

## 📋 Scratch → C++ 语法速查

| Scratch | C++ | 备注 |
|:---|:---|:---|
| `将x设为10` | `int x = 10;` | 声明+初始化 |
| `将x增加5` | `x += 5;` | 累加 |
| `如果...那么...` | `if (...) {...}` | 条件 |
| `如果...那么...否则` | `if (...) {...} else {...}` | 分支 |
| `重复执行10次` | `for(int i=0; i<10; i++)` | 定次循环 |
| `重复执行直到` | `while (...)` | 条件循环 |
| `说...` | `cout << ...` | 输出 |
| `询问并等待` | `cin >> ...` | 输入 |
| `列表第i项` | `arr[i]` | 数组访问 |
| `列表长度` | 需要变量记录 | C++数组不自带长度 |
| `定义...` | `void func()` | 函数定义 |
| `报告...` | `return ...` | 返回值 |
