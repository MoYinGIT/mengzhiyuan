# 示例：迷宫冒险

> 经典的迷宫探索游戏

## 🎮 游戏效果

- 控制角色在迷宫中移动
- 碰到墙壁会返回起点
- 找到出口（金色大门）获胜
- 计时挑战，看谁用时最短

## 📝 完整代码

### 玩家角色代码

```scratch
// 变量：计时器（适用于所有角色）

// 初始化位置
当绿旗被点击
移到 x: (-200) y: (-150)  // 迷宫入口
将计时器归零

// 键盘控制
当按下 [向上 v] 箭头
面向 (0) 方向
移动 (5) 步
检查碰撞

当按下 [向下 v] 箭头
面向 (180) 方向
移动 (5) 步
检查碰撞

当按下 [向左 v] 箭头
面向 (-90) 方向
移动 (5) 步
检查碰撞

当按下 [向右 v] 箭头
面向 (90) 方向
移动 (5) 步
检查碰撞

// 碰撞检测
定义 检查碰撞
如果 <碰到颜色 [#000000]？> 那么  // 黑色是墙壁
    播放声音 [Boop v]
    移动 (-5) 步  // 退回来
结束
如果 <碰到 [出口 v]？> 那么  // 到达出口
    播放声音 [Cheer v]
    说 (连接 你赢了！用时：和计时器) 2秒
    停止 [全部 v]
结束
```

### 计时器代码

```scratch
// 显示用时
当绿旗被点击
重复执行
    说 (计时器) 持续 (0.1) 秒
结束
```

### 出口（金色大门）代码

```scratch
// 出口一直在固定位置
当绿旗被点击
移到 x: (200) y: (150)  // 迷宫出口
显示
```

## 🎨 舞台设计

**绘制迷宫**：
1. 点击「背景」→「造型」→「编辑」
2. 选择黑色画笔，绘制墙壁
3. 在右上角绘制一个金色区域作为出口
4. 左下角为起点

**迷宫示例**：

## 💡 要点解析

| 知识点 | 说明 |
|:---|:---|
| 颜色检测 | 用`碰到颜色`判断墙壁 |
| 后退机制 | 碰撞后反向移动实现"撞墙"效果 |
| 计时器 | Scratch内置计时器记录用时 |
| 胜利条件 | 碰到出口角色触发胜利 |

## 🎯 扩展挑战

1. **收集金币**：在迷宫中放置金币，收集加分
2. **移动敌人**：添加巡逻的敌人，碰到返回起点
3. **多关卡**：设计多个迷宫，通关后进入下一关
4. **提示系统**：按空格显示最短路径提示
