# 示例：小猫追蝴蝶

> 一个经典的追逐游戏示例

## 🎮 游戏效果

- 用方向键控制小猫移动
- 蝴蝶在舞台上随机飞舞
- 小猫碰到蝴蝶得分
- 蝴蝶被碰到后飞到新位置

## 📝 完整代码

### 小猫代码

```scratch
// 初始化
当绿旗被点击
移到 x: (0) y: (0)
面向 (90) 方向

// 键盘控制移动
当按下 [向上 v] 箭头
面向 (0) 方向
移动 (5) 步

当按下 [向下 v] 箭头
面向 (180) 方向
移动 (5) 步

当按下 [向左 v] 箭头
面向 (-90) 方向
移动 (5) 步

当按下 [向右 v] 箭头
面向 (90) 方向
移动 (5) 步

// 碰到边缘反弹
当绿旗被点击
重复执行
    如果 <碰到边缘？> 那么
        碰到边缘就反弹
    结束
结束
```

### 蝴蝶代码

```scratch
// 变量：得分（适用于所有角色）

// 初始化
当绿旗被点击
将得分设为 (0)
重复执行
    移到 x: (随机数 (-200) 到 (200)) y: (随机数 (-150) 到 (150))
    等待 (2) 秒
结束

// 被小猫碰到
当绿旗被点击
重复执行
    如果 <碰到 [小猫 v]？> 那么
        将得分增加 (1)
        播放声音 [Chirp v]
        移到 x: (随机数 (-200) 到 (200)) y: (随机数 (-150) 到 (150))
        等待 (1) 秒
    结束
结束

// 显示得分
当绿旗被点击
重复执行
    说 (得分) 持续 (0.5) 秒
结束
```

## 💡 要点解析

| 知识点 | 说明 |
|:---|:---|
| 方向控制 | 四个方向键分别对应 0°、90°、180°、-90° |
| 随机位置 | 使用随机数让蝴蝶出现在舞台任意位置 |
| 变量计分 | 用变量记录得分，碰到时增加 |
| 重复执行 | 用循环持续检测碰撞和移动 |

## 🎯 扩展挑战

1. **增加难度**：蝴蝶移动速度越来越快
2. **添加时间**：60秒倒计时，看能得多少分
3. **多条生命**：小猫碰到边缘减生命
4. **多个蝴蝶**：用克隆技术创建多个蝴蝶
