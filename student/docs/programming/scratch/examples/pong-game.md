# Pong 游戏（乒乓游戏）

> 经典街机游戏 · 双人对战 · 物理碰撞

---

## 🎯 游戏目标

- 控制挡板击打乒乓球
- 不让球落到自己这边
- 先得 10 分者获胜

---

## 🎮 游戏玩法

- **玩家1（左边）**：W 键上移，S 键下移
- **玩家2（右边）**：↑ 键上移，↓ 键下移
- 球碰到挡板会反弹
- 球出界则对方得分

### 角色3：球

```scratch
当 绿旗被点击
    移到 x:0 y:0
    将 大小 设为 15
    面向 随机数从 1 到 360 方向
    将 玩家1分数 设为 0
    将 玩家2分数 设为 0
    重复执行
        移动 5 步
        
        // 碰到上下边界反弹
        如果 碰到 边缘 那么
            面向 180 - 方向 方向
        结束如果
        
        // 碰到玩家1挡板
        如果 碰到 颜色 [#FF0000] 那么  // 假设玩家1是红色
            面向 随机数从 -60 到 60 方向
            播放声音 啵
        结束如果
        
        // 碰到玩家2挡板
        如果 碰到 颜色 [#0000FF] 那么  // 假设玩家2是蓝色
            面向 随机数从 120 到 240 方向
            播放声音 啵
        结束如果
        
        // 球出界 - 玩家2得分
        如果 x 坐标 < -240 那么
            将 玩家2分数 增加 1
            说 "玩家2得分！" 持续 1 秒
            移到 x:0 y:0
            面向 随机数从 -45 到 45 方向
        结束如果
        
        // 球出界 - 玩家1得分
        如果 x 坐标 > 240 那么
            将 玩家1分数 增加 1
            说 "玩家1得分！" 持续 1 秒
            移到 x:0 y:0
            面向 随机数从 135 到 225 方向
        结束如果
        
        // 检查获胜
        如果 玩家1分数 = 10 那么
            说 "玩家1获胜！" 持续 3 秒
            停止 全部
        结束如果
        如果 玩家2分数 = 10 那么
            说 "玩家2获胜！" 持续 3 秒
            停止 全部
        结束如果
    结束重复
结束
```

## 🔧 进阶功能

### 1. 增加球速

```scratch
// 每反弹5次增加速度
将 反弹次数 设为 0
将 速度 设为 5

// 在球的代码中
如果 碰到挡板 那么
    将 反弹次数 增加 1
    如果 反弹次数 除以 5 的余数 = 0 那么
        将 速度 增加 1
    结束如果
结束如果
```

### 2. 挡板击中位置影响反弹角度

```scratch
// 根据击中挡板的位置计算反弹角度
将 相对位置 设为 (球的y - 挡板的y) / 挡板高度
将 反弹角度 设为 相对位置 * 60  // -60到60度
面向 反弹角度 方向
```

### 3. 单人模式（对战电脑）

```scratch
// 电脑挡板自动跟随球
当 绿旗被点击
    重复执行

            将 y 坐标 增加 8  // 电脑速度稍慢，给人机会
        结束如果
        如果 y 坐标 > 球的y 那么
            将 y 坐标 增加 -8
        结束如果
    结束重复
结束
```

## 🧠 编程思维

### 碰撞检测

```
Pong 游戏的核心是碰撞检测：
1. 球与上下边界碰撞 → 垂直反弹
2. 球与挡板碰撞 → 水平反弹
3. 球与左右边界碰撞 → 得分/重置

这用到了：
- 条件判断（如果...那么）
- 坐标系统（x, y）
- 运动控制（移动、面向）
```

### 游戏循环

```
游戏的核心循环：
1. 更新玩家位置（输入处理）
2. 更新球位置（游戏逻辑）
3. 检测碰撞（游戏逻辑）
4. 更新分数（游戏状态）
5. 渲染画面（显示）
6. 回到第1步

这就是游戏编程的基本框架！
```
