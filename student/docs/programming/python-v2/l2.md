## 📚 知识探索

### 🔍 探索1：条件判断——程序的分岔路口

**生活类比**：你每天上学都会遇到选择：
- 如果下雨 → 带雨伞
- 如果出太阳 → 戴帽子
- 否则 → 正常出门

程序也需要做选择，这就是**条件判断**。

**最简单的if语句**：

```python
age = 12

if age >= 10:
    print("你可以参加这个活动！")
```

**运行结果**：
```
你可以参加这个活动！
```

**⚠️ 注意**：
- `if` 后面要加冒号 `:`
- 下一行要**缩进**（通常是4个空格）

### 🔍 探索2：比较运算符——判断的"尺子"

条件判断需要"尺子"来比较大小：

| 运算符 | 含义 | 示例 | 结果 |
|:---:|:---:|:---:|:---:|
| `==` | 等于 | `5 == 5` | `True` |
| `!=` | 不等于 | `5 != 3` | `True` |
| `>` | 大于 | `5 > 3` | `True` |
| `<` | 小于 | `5 < 3` | `False` |
| `>=` | 大于等于 | `5 >= 5` | `True` |
| `<=` | 小于等于 | `5 <= 3` | `False` |

**⚠️ 注意**：`==` 是两个等号，表示"等于判断"；`=` 是一个等号，表示"赋值"。

```python
# 正确
if age == 10:
    print("你10岁了")

# 错误（会报错）
if age = 10:      # 这里用了赋值，不是判断！
    print("你10岁了")
```

### 🔍 探索3：逻辑运算符——组合条件

有时候需要同时满足多个条件：

**and（并且）**：两个条件都要满足
```python
age = 15
has_ticket = True

if age >= 12 and has_ticket:
    print("请进！")
```

**or（或者）**：满足其中一个即可
```python
day = "周六"

if day == "周六" or day == "周日":
    print("今天是周末！")
```

**not（不是）**：条件反过来
```python
is_raining = False

if not is_raining:
    print("不下雨，可以出去玩！")
```

**试一试**：写一个程序，判断一个人是否能买儿童票（年龄 < 12 或 身高 < 1.4米）。

<summary>✅ 参考答案</summary>

```python

height = float(input("请输入身高（米）："))

if age < 12 or height < 1.4:
    print("可以买儿童票")
else:
    print("需要买成人票")
```

### 🔍 探索4：for循环——重复做事的好帮手

**生活类比**：体育老师说："绕操场跑5圈！" 你会一圈一圈地跑，直到跑完5圈。

程序中，让代码重复执行就是**循环**。

**最简单的for循环**：

```python
for i in range(5):
    print("第", i, "次循环")
```

**输出**：
```
第 0 次循环
第 1 次循环
第 2 次循环
第 3 次循环
第 4 次循环
```

**发现规律了吗？** 🤔
- `range(5)` 生成 0, 1, 2, 3, 4
- 循环变量 `i` 每次取一个值
- 循环体内的代码重复执行5次

**range的更多用法**：

```python
# range(开始, 结束) —— 包含开始，不包含结束
for i in range(1, 6):
    print(i)        # 输出 1, 2, 3, 4, 5

# range(开始, 结束, 步长)
for i in range(0, 10, 2):
    print(i)        # 输出 0, 2, 4, 6, 8

# 倒着数
for i in range(5, 0, -1):
    print(i)        # 输出 5, 4, 3, 2, 1
```

### 🔍 探索5：while循环——当条件满足时继续

**for循环**适合知道要循环多少次的情况。
**while循环**适合不知道循环多少次的情况。

**基本结构**：

```python
count = 1

while count <= 5:
    print("计数：", count)
    count = count + 1   # 别忘了增加计数器！

print("循环结束！")
```

**⚠️ 危险警告**：如果忘记写 `count = count + 1`，程序会永远循环下去！这叫"死循环"。

**猜数字游戏**：

```python
import random

secret = random.randint(1, 100)   # 随机生成1-100的数字
count = 0

while True:                        # 无限循环
    guess = int(input("猜一个1-100的数字："))
    count = count + 1
    
    if guess < secret:
        print("太小了！")
    elif guess > secret:
        print("太大了！")
    else:
        print("恭喜你猜对了！用了", count, "次")
        break                       # 猜对了，跳出循环
```

**break语句**：立刻跳出循环，不再执行后面的代码。

### 🔍 探索6：列表——一个盒子装多个东西

**生活类比**：你的书包里可以装很多本书，列表就像是程序中的"书包"。

**创建列表**：

```python
# 创建一个空列表
fruits = []

# 创建一个有元素的列表
fruits = ["苹果", "香蕉", "橙子", "葡萄"]

# 列表可以装不同类型的数据
mixed = [10, "hello", 3.14, True]
```

**访问列表元素**：

```python
fruits = ["苹果", "香蕉", "橙子"]

print(fruits[0])    # 苹果（索引从0开始！）
print(fruits[1])    # 香蕉
print(fruits[2])    # 橙子
```

**⚠️ 注意**：Python中索引从 **0** 开始，不是从1！

**修改和添加元素**：

```python
fruits = ["苹果", "香蕉"]

# 修改元素
fruits[0] = "西瓜"
print(fruits)       # ["西瓜", "香蕉"]

# 添加元素
fruits.append("橙子")   # 添加到末尾
print(fruits)       # ["西瓜", "香蕉", "橙子"]

# 插入元素
fruits.insert(1, "葡萄")  # 在索引1的位置插入
print(fruits)       # ["西瓜", "葡萄", "香蕉", "橙子"]
```

**获取列表长度**：

```python
fruits = ["苹果", "香蕉", "橙子"]
print(len(fruits))   # 3
```

**遍历列表**：

```python
fruits = ["苹果", "香蕉", "橙子"]

# 方法1：使用索引
for i in range(len(fruits)):
    print(fruits[i])

# 方法2：直接遍历（推荐）
for fruit in fruits:
    print(fruit)
```

### 🟡 练习4：寻找最大值（进阶）

**要求**：输入5个数字，找出其中的最大值。

**提示**：先假设第一个数最大，然后逐个比较。

<details>
<summary>✅ 参考答案</summary>

```python
numbers = []

# 输入5个数字
for i in range(5):
    n = int(input(f"请输入第{i+1}个数字："))
    numbers.append(n)

# 找最大值
max_num = numbers[0]    # 先假设第一个最大

for num in numbers:
    if num > max_num:
        max_num = num   # 发现更大的，更新最大值

print("\n输入的数字：", numbers)
print("最大值是：", max_num)
```
</details>

### 🔴 练习5：猜数字游戏升级版（挑战）

**要求**：
1. 随机生成1-100的数字
2. 让用户猜，提示"太大"或"太小"
3. 限制最多猜7次
4. 猜对显示"恭喜"，用完次数显示"游戏结束"

<details>
<summary>✅ 参考答案</summary>

```python
import random

secret = random.randint(1, 100)
chances = 7

print("===== 猜数字游戏 =====")
print("我想了一个1-100的数字，你有7次机会！")

for i in range(chances):
    guess = int(input(f"\n第{i+1}次猜测："))
    
    if guess < secret:
        print("太小了！")
    elif guess > secret:
        print("太大了！")
    else:
        print(f"恭喜你猜对了！答案就是 {secret}")
        break
else:
    # for循环正常结束（没有被break）时执行
    print(f"\n游戏结束！正确答案是 {secret}")
```
</details>

### 🔴 练习6：购物车统计（挑战）

**要求**：
1. 输入5件商品的价格，存入列表
2. 计算总价
3. 计算平均价格
4. 找出最贵和最便宜的商品
5. 如果总价超过100元，显示"预算超支"

<details>
<summary>✅ 参考答案</summary>

```python
prices = []

# 输入5件商品价格
for i in range(5):
    price = float(input(f"请输入第{i+1}件商品的价格："))
    prices.append(price)

# 计算
 total = sum(prices)           # Python内置函数求和
average = total / len(prices)
max_price = max(prices)       # Python内置函数求最大值
min_price = min(prices)       # Python内置函数求最小值

# 输出结果
print("\n===== 购物车统计 =====")
print("商品价格：", prices)
print(f"总价：{total:.2f} 元")
print(f"平均：{average:.2f} 元")
print(f"最贵：{max_price:.2f} 元")
print(f"最便宜：{min_price:.2f} 元")

if total > 100:
    print("⚠️ 预算超支！")
else:
    print("✅ 预算内")
```
</details>

## 🧩 综合应用

### 题目：班级成绩管理系统

**背景**：老师需要统计班级成绩。

**功能要求**：
1. 输入班级人数
2. 依次输入每个同学的成绩
3. 计算班级平均分
4. 统计及格人数（>=60分）
5. 找出最高分和最低分
6. 给出班级评价：
   - 平均分 >= 90：优秀班级
   - 平均分 >= 80：良好班级
   - 平均分 >= 60：合格班级
   - 平均分 < 60：需努力

**问题分解**：
- Step 1：获取班级人数，创建空列表
- Step 2：用循环输入每个成绩，存入列表
- Step 3：计算平均分、最高分、最低分
- Step 4：用循环统计及格人数
- Step 5：根据平均分输出评价

<details>
<summary>✅ 参考答案</summary>

```python
print("===== 班级成绩管理系统 =====")

# 输入班级人数
n = int(input("请输入班级人数："))

# 输入成绩
scores = []
for i in range(n):
    score = int(input(f"请输入第{i+1}位同学的成绩："))
    scores.append(score)

# 计算统计数据
total = sum(scores)
average = total / n
max_score = max(scores)
min_score = min(scores)

# 统计及格人数
pass_count = 0
for score in scores:
    if score >= 60:
        pass_count = pass_count + 1

# 输出结果
print("\n===== 成绩统计 =====")
print(f"班级人数：{n} 人")
print(f"平均分：{average:.1f} 分")
print(f"最高分：{max_score} 分")
print(f"最低分：{min_score} 分")
print(f"及格人数：{pass_count} 人")
print(f"及格率：{pass_count/n*100:.1f}%")

# 班级评价
print("\n===== 班级评价 =====")
if average >= 90:
    print("🌟 优秀班级！")
elif average >= 80:
    print("👍 良好班级！")
elif average >= 60:
    print("✅ 合格班级")
else:
    print("💪 需继续努力")
```
</details>

### 🐛 常见错误自查

运行出错时，检查：

| 错误 | 原因 | 解决 |
|:---|:---|:---|
| `IndentationError` | 缩进错误 | 确保对齐，用4个空格 |
| `SyntaxError` | if/for后面没加冒号 | 检查 `:` |
| `IndexError` | 列表索引超出范围 | 检查索引是否 < len(列表) |
| 死循环 | while条件永远为真 | 确保循环内有改变条件的代码 |
| `=` 和 `==` 混淆 | 判断用了赋值 | 判断用 `==`，赋值用 `=` |

|:---|:---|:---|
