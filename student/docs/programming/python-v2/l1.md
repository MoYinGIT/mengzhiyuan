### 🔍 探索1：第一个程序——和计算机打招呼

**想象一下**：你刚认识一个新朋友，会说什么？

在编程世界，我们和计算机的"第一句话"是这样的：

```python
print("你好，世界！")
```

**试试看**：打开Python，输入上面这行代码，按回车运行。

**你看到什么？**

```
你好，世界！
```

🎉 恭喜你！这是你的第一个Python程序！

### 🤔 思考题

**问题1**：如果把双引号去掉，写成 `print(你好，世界！)` 会怎样？

<details>
<summary>💡 需要提示？</summary>

**提示**：想一想，计算机怎么知道"你好，世界"是文字，还是指令呢？引号就像是给文字穿上"外衣"，告诉计算机："这是文字，直接显示出来！"
</details>

**问题2**：下面的代码会输出什么？
```python
print(3 + 5)

```

<details>
<summary>💡 需要提示？</summary>

**提示**：第一个没有引号，Python会计算 `3+5`；第二个有引号，Python会直接显示文字。
</details>

### 🔍 探索2：变量——程序中的「标签盒子」

**生活类比**：想象你有一个玩具收纳箱，里面有很多带标签的盒子：
- 贴着「年龄」的盒子里放着数字 `10`
- 贴着「名字」的盒子里放着文字 `"小明"`

在Python中，这些带标签的盒子就是**变量**。

**创建你的第一个变量**：

```python
age = 10           # 创建一个标签为age的盒子，里面放10
name = "小明"       # 创建一个标签为name的盒子，里面放"小明"

print(name, "今年", age, "岁")
```

**输出**：
```
小明 今年 10 岁
```

**发现规律了吗？** 🤔
- `=` 不是"等于"，而是"放入"（把右边的值放进左边的盒子）
- 变量名就是盒子的标签
- `print()` 可以输出多个内容，用逗号隔开

### 🔍 探索3：变量类型——盒子能装什么？

不同的盒子装不同的东西：

| 类型 | 名称 | 能装什么 | 示例 |
|:---:|:---:|:---|:---|
| `int` | 整数 | 没有小数的数字 | `10`, `-5`, `0` |
| `float` | 浮点数 | 有小数的数字 | `3.14`, `1.75` |
| `str` | 字符串 | 文字 | `"Hello"`, `'世界'` |

```python
age = 10              # 整数 int
height = 1.45         # 浮点数 float
name = "小明"          # 字符串 str

print(type(age))      # 输出: <class 'int'>
print(type(height))   # 输出: <class 'float'>
print(type(name))     # 输出: <class 'str'>
```

**试一试**：创建几个不同类型的变量，用 `type()` 看看它们是什么类型！

### 🔍 探索4：输入——让用户「填写盒子」

上面的例子中，年龄和名字是写在代码里的。如果想让用户自己输入呢？

**使用 `input()`**：

```python
name = input("请输入你的名字：")
print("你好，", name, "！欢迎来到Python世界！")
```

**运行程序**，输入你的名字，看看会发生什么！

**⚠️ 注意**：`input()` 得到的内容都是**字符串**（文字）。如果想进行数学计算，需要转换类型：

```python
# 错误示范（会得到奇怪的结果）
age = input("请输入你的年龄：")
print("明年你", age + 1, "岁")     # 报错！不能文字+数字

# 正确做法：先转换类型
age = input("请输入你的年龄：")
age = int(age)                    # 把文字转换成整数
print("明年你", age + 1, "岁")     # 正常输出

# 或者简写
age = int(input("请输入你的年龄："))
print("明年你", age + 1, "岁")
```

### 🔍 探索5：基本计算——Python计算器

Python可以像计算器一样进行数学运算：

```python
a = 10
b = 3

print(a + b)    # 13（加法）
print(a - b)    # 7（减法）
print(a * b)    # 30（乘法）
print(a / b)    # 3.333...（除法）
print(a // b)   # 3（整除，只留整数部分）
print(a % b)    # 1（取余，10除以3余1）
print(a ** b)   # 1000（幂运算，10的3次方）
```

**试一试**：编写程序，输入两个数，输出它们的和、差、积、商。

<details>
<summary>💡 提示</summary>

```python
a = int(input("请输入第一个数："))
b = int(input("请输入第二个数："))

print("和：", a + b)
print("差：", a - b)
# 继续补充...
```
</details>

### 🟡 练习2：简易计算器（进阶）

**要求**：输入两个数，输出它们的和、差、积、商、余数。

**Step 1 分解**：先获取两个数字
```python
num1 = int(input("请输入第一个数："))
num2 = int(input("请输入第二个数："))
```

**Step 2 分解**：计算并输出各种结果
```python
print("和：", num1 + num2)
print("差：", num1 - num2)
# 继续补充乘、除、余数...
```

**思考**：如果第二个数输入 `0`，除法会发生什么？

<details>
<summary>✅ 参考答案</summary>

```python
num1 = int(input("请输入第一个数："))
num2 = int(input("请输入第二个数："))

print("===== 计算结果 =====")
print("和：", num1 + num2)
print("差：", num1 - num2)
print("积：", num1 * num2)
print("商：", num1 / num2)
print("整除：", num1 // num2)
print("余数：", num1 % num2)
```

**注意**：如果 `num2` 为0，除法会产生错误（ZeroDivisionError）。我们会在L2学习如何避免这种情况。

### 🔴 练习3：购物小票（挑战）

**要求**：模拟超市收银，输入商品单价和数量，计算总价并打印小票。

**问题分解**：

<summary>✅ 参考答案</summary>

## 🧩 综合应用

### 题目：BMI健康计算器

**背景**：BMI（身体质量指数）是衡量身体健康的一个指标。

**公式**：BMI = 体重(kg) ÷ 身高(m)²

**任务**：编写程序，输入身高和体重，计算并输出BMI值。

**参考标准**：
- BMI < 18.5：偏瘦
- 18.5 ≤ BMI < 24：正常
- BMI ≥ 24：偏胖

**问题分解**：
- Step 1：输入身高（米）和体重（公斤）
- Step 2：计算 BMI = weight / (height ** 2)
- Step 3：输出BMI值

<details>
<summary>✅ 参考答案</summary>

```python
print("===== BMI健康计算器 =====")

height = float(input("请输入你的身高（米）："))
weight = float(input("请输入你的体重（公斤）："))

bmi = weight / (height ** 2)

print("\n你的BMI值是：", round(bmi, 2))
print("（保留两位小数）")

if bmi < 18.5:
    print("体型偏瘦，多吃点哦！")
elif bmi < 24:
    print("身材很棒，继续保持！")
else:
    print("体型偏胖，多运动哦！")
```

**注意**：这里用到了`if`语句，我们将在L2详细学习。
</details>

### 🐛 常见错误自查

运行出错时，检查：

| 错误 | 原因 | 解决 |
|:---|:---|:---|
| `SyntaxError` | 语法错误，如漏了冒号或引号 | 检查每行是否完整 |
| `NameError` | 变量名拼写错误 | 检查大小写是否一致 |
| `TypeError` | 类型不匹配，如字符串+数字 | 用 `int()` 或 `float()` 转换 |
| `ValueError` | 转换失败，如 `int("abc")` | 确保输入的是数字 |
