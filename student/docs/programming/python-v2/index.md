## 🎉 v2.0 全新升级

本套教程采用**全球IOI金牌教练**推崇的五大思维模型重构，专为**9-12岁**学习者设计。

| 原则 | 说明 |
|:---|:---|
| **引导而不灌输** | 不给答案，只给提示，让你自己发现知识 |
| **分层而不跳跃** | 每课都有前置检查，确保基础扎实再进阶 |
| **分解而不畏惧** | 复杂问题拆成小步骤，一步一步解决 |
| **练习而不盲目** | 每个知识点都有针对性练习，即学即练 |

## ✨ v2.0 新特性

### 1. 引导式学习
不再是"知识灌输"，而是"探索发现"：
- 用问题开场，引发思考
- 用生活类比，建立直觉
- 用填空练习，主动参与
- 用 `&lt;details&gt;` 隐藏提示和答案，自主探索

### 2. 分层设计
- 🟢 **基础**：语法巩固
- 🟡 **进阶**：综合应用
- 🔴 **挑战**：思维拓展

### 3. 反思闭环
- 三省吾身自检
- 错题本模板
- 常见错误自查表

## 💡 Python vs C++ 对比

| 特性 | Python | C++ |
|:---|:---|:---|
| 语法复杂度 | 简单 | 较复杂 |
| 运行速度 | 较慢 | 快 |
| 内存管理 | 自动 | 手动 |
| 适用场景 | 快速开发、AI | 竞赛、系统编程 |
| 学习曲线 | 平缓 | 陡峭 |

**建议**：
- 想快速入门编程 → 学Python
- 目标信息学奥赛 → 学C++
- 时间充裕 → 先Python后C++
