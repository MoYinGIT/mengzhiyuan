| 原则 | 说明 |

## ✨ v2.0 新特性

### 1. 引导式学习
不再是"知识灌输"，而是"探索发现"：
- 用问题开场，引发思考
- 用生活类比，建立直觉
- 用填空练习，主动参与
- 用 `&lt;details&gt;` 标签隐藏提示和答案，自主探索

### 2. 分层设计
- 🟢 **基础**：语法巩固
- 🟡 **进阶**：综合应用
- 🔴 **挑战**：思维拓展

### 3. 问题分解
复杂题目拆成Step 1/2/3，不再畏惧难题

### 4. 即时反馈
每个知识点后紧跟练习，趁热打铁

### 5. 反思闭环
- 三省吾身自检
- 错题本模板
- 常见错误自查表

## 📝 学习方法

### 课前
- 完成**课前自检**，确认前置知识已掌握
- 如果没有掌握，返回前一课复习

### 课中
- 按照**知识探索**的顺序学习
- 先自己思考，需要时再点开**提示**
- 完成所有**填空练习**和**思考题**

### 课后
- 完成**动手实践**的所有练习
- 填写**错题本**
- 完成**三省吾身**自检
