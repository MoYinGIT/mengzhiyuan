A. 0 1 2 3 4

**2. 如果希望程序输出 `0 1 2 3 4`，应该在第5行前添加（ ）**

A. `i = 0;`  
B. `int i = 0;` 把第4行改成这行  
C. A和B都可以  
D. 不需要添加，程序本身正确

A. 输出 10 11 12 13 14  
B. 输出空行（什么都没有）  
C. 进入无限循环  
D. 编译错误

### 二、判断题

**4. 局部变量如果不初始化，其值是不确定的（垃圾值）。**  
（ ）正确 （ ）错误

（ ）正确 （ ）错误

（ ）正确 （ ）错误

**7. 循环变量应该在 ______（循环外/循环内）初始化，否则可能导致______。**

**8. 如果将代码改为 `int i = 5; while (i < 5)`，循环体执行______次。**

**9. 安全的编程习惯是：定义变量时 ______（立即初始化/稍后赋值）。**

---

**解析关键点**：
- 局部变量未初始化时值不确定

**2. `break` 语句会跳出（ ）**

A. 所有循环  
B. 最内层的while循环  
C. 最外层的while循环  
D. 整个程序

**3. 如果希望在内层j==2时直接结束整个程序，应该用（ ）**

A. `break;`  
B. `return 0;`  
C. `continue;`  
D. `exit(0);`

### 二、判断题

**4. break只能用于循环内部，不能用于if语句单独使用。**  
（ ）正确 （ ）错误

**5. break跳出的是包含它的最内层循环。**  
（ ）正确 （ ）错误

（ ）正确 （ ）错误

### 三、填空题

**7. break语句的作用是 ______ 本层循环，程序继续执行 ______ 的代码。**

**8. 上述程序中，内层循环每次执行 ______ 次（j=1时正常执行，j=2时break）。**

## 【大题10】while循环的"综合应用陷阱"

```cpp

int main() {
    int n = 1221;

        n = n / 10;

    if (original == reversed) {
        cout << original << "是回文数" << endl;

        cout << original << "不是回文数" << endl;
```

| 4 | A,B,B | 错,对,对 | 跳出/终止,循环后,1,goto/标志 |
| 5 | B,C,C | 错,对,对 | 本次循环剩余,下一次循环,循环变量更新,终止,跳过 |
| 6 | C,A,B | 对,对,错 | 读入,验证,cin.clear,cin.ignore,do-while |
| 7 | B,D,A | 对,对,对 | 初始化为0,累加,n*(n+1)/2,快,1,0 |
| 8 | B,A,C | 对,对,对 | 除以10/舍弃最后一位,0,复制/备份,绝对值 |
| 9 | B,B,B | 对,对,对 | %10,/10,4,3,digit |
| 10 | A,B,B | 对,对,对 | *,+,逆序,0 |

---

**总结：while循环核心要点**

1. **三要素不可少**：初始化、条件判断、更新语句
2. **防死循环**：确保条件会趋向于假
3. **边界注意**：< 还是 <=，从0还是从1开始
4. **变量保存**：循环会改变的值需要备份
5. **break与continue**：跳出 vs 跳过，都只影响当前层
