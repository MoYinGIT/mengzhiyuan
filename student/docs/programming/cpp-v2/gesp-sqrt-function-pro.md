## 第10题：综合应用

**程序代码**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    // 求一元二次方程的根：x² - 5x + 6 = 0
    double a = 1, b = -5, c = 6;
    double delta = b*b - 4*a*c;
    
    if (delta >= 0) {
        double x1 = (-b + sqrt(delta)) / (2*a);
        double x2 = (-b - sqrt(delta)) / (2*a);
        cout << x1 << " " << x2 << endl;
    }
    
    // 求三点形成的三角形面积（海伦公式）
    double side1 = 3, side2 = 4, side3 = 5;
    double s = (side1 + side2 + side3) / 2;
    double area = sqrt(s * (s-side1) * (s-side2) * (s-side3));
    cout << area << endl;
    
    return 0;
}
```

## 知识点总结

| 考点 | 说明 |
|:---|:---|
| 头文件 | `#include <cmath>` 或 `<bits/stdc++.h>` |
| 返回值类型 | 始终为 `double` |
| 参数要求 | 非负数，负数返回 `nan` |

| 实际应用 | 勾股定理、距离公式、圆面积逆运算、海伦公式 |
| 常见错误 | 忘记括号、参数为负、直接比较浮点数相等 |
