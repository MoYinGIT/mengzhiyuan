## 📚 知识探索

### 🔍 探索1：第一个程序

**想象一下**：你刚刚学会一门外语，想说的第一句话是什么？

在编程世界，我们的"第一句话"是：

```cpp
Hello, World!
```

**试试看**：打开你的编程软件，创建一个新文件，输入以下内容：

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}
```

**点击运行**，你看到了什么？

### 🔍 探索2：变量——程序中的「盒子」

**生活类比**：想象你有几个盒子，分别贴着标签：「年龄」「身高」「姓名」。

在C++中，我们也有这样的「盒子」，叫做**变量**。

**创建你的第一个变量**：

```cpp
#include <iostream>
using namespace std;

int main() {
    int age = 15;           // 创建一个叫age的盒子，里面放整数15
    cout << "我的年龄是：" << age << endl;
    return 0;
}
```

**试一试**：把 `15` 改成你自己的年龄，运行看看！

---

### 📝 填空练习

**请补全代码**：创建一个变量存储你的身高（厘米），并输出。

```cpp
#include <iostream>
using namespace std;

int main() {
    int height = ______;    // 填空：你的身高
    cout << "我的身高是：" << ______ << "厘米" << endl;
    return 0;
}
```

<details>
<summary>✅ 答案对照</summary>

```cpp
int height = 170;           // 假设身高170厘米
cout << "我的身高是：" << height << "厘米" << endl;
```
</details>

---

### 🔍 探索3：输入——让用户「填盒子」

上面的例子中，年龄和身高是写在代码里的。如果想让用户自己输入呢？

**使用 `cin`**：

```cpp
#include <iostream>
using namespace std;

int main() {
    int age;
    cout << "请输入你的年龄：";
    cin >> age;                    // 等待用户输入，存到age盒子里
    cout << "哇，你" << age << "岁了！" << endl;
    return 0;
}
```

**运行程序**，输入你的年龄，看看会发生什么！

**发现规律了吗？**
- `cout <<` ：输出（把内容显示出来）
- `cin >>` ：输入（把用户打字的内容存到变量）

箭头方向就是数据流动的方向！

### 🟡 练习2：简易计算器（进阶）

**要求**：输入两个数，输出它们的和、差、积、商。

### 🔴 练习3：温度转换器（挑战）

**要求**：输入摄氏温度，输出华氏温度。

**公式**：华氏 = 摄氏 × 9/5 + 32

**提示**：先写出步骤
1. 输入摄氏温度（存到变量c）
2. 计算华氏温度（存到变量f）
3. 输出结果

<details>
<summary>✅ 参考答案</summary>

```cpp
#include <iostream>
using namespace std;

int main() {
    double c;                           // 摄氏温度（用小数）
    cout << "请输入摄氏温度：";
    cin >> c;
    
    double f = c * 9 / 5 + 32;          // 计算华氏温度
    cout << "华氏温度是：" << f << endl;
    
    return 0;
}
```
</details>
