## 【大题1】循环嵌套的"执行顺序陷阱"

**考察点**：外层循环执行1次，内层循环执行完整的一轮

```cpp
#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 2; i++) {
        cout << "外层i=" << i << " ";
        for (int j = 1; j <= 3; j++) {
            cout << "内层j=" << j << " ";
        }
        cout << endl;
    }
    return 0;
}
```

### 一、选择题

**1. 程序的输出结果是（ ）**

A. 外层i=1 内层j=1 内层j=2 内层j=3  
   外层i=2 内层j=1 内层j=2 内层j=3  
B. 外层i=1 内层j=1 外层i=2 内层j=2 ...  
C. 外层i=1 外层i=2 内层j=1 内层j=2 内层j=3  
D. 什么都不输出

**2. 内层循环一共执行了多少次（ ）**

A. 2次  
B. 3次  
C. 5次  
D. 6次

**3. 外层循环的cout << endl执行了几次（ ）**

A. 1次  
B. 2次  
C. 3次  
D. 6次

### 二、判断题

**4. 外层循环执行1次，内层循环会执行完整的一轮（从初值到结束）。**  
（ ）正确 （ ）错误

**5. 内层循环的j每次都会从1重新开始。**  
（ ）正确 （ ）错误

**6. 外层循环变量i在内层循环中可以使用。**  
（ ）正确 （ ）错误

### 三、填空题

**7. 外层循环执行______次，每次内层循环执行______次，总共执行______次。**

**8. 执行顺序是：i=1时，j=1,2,3；然后i=2时，j=______。**

**9. 如果把内层循环改成j<=i，当i=2时j执行______次。**

---

**解析关键点**：
- 外层1次，内层完整执行一轮
- 内层变量每次重新初始化
- 外层变量在内层可见

C. 什么都不输出  
D. 编译错误

**2. 如果把int j = 1移到第一个for循环内部（int i = 1后面），结果会（ ）**

（ ）正确 （ ）错误

**学习建议**：
1. **执行顺序**：外层1次，内层完整执行一轮
2. **变量初始化**：内层变量应在外层体内初始化
3. **矩形打印**：外层控制行，内层控制列
4. **三角形打印**：内层条件依赖外层变量（j <= i）
5. **金字塔**：需要两个内层循环（空格+数字）
6. **break/continue**：只作用于最内层循环
7. **时间复杂度**：双层循环通常为O(n²)
8. **变量作用域**：内层可以访问外层变量
