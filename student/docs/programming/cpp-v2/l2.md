### 🔍 探索1：分支——程序中的「选择题」

**生活场景**：
- 如果明天不下雨，就去公园
- 如果考试成绩≥90，就奖励一个玩具
- 否则，继续努力

程序也需要做这样的「选择题」。

---

#### 🎯 挑战：成绩评定

**场景**：输入一个成绩，判断是"及格"还是"不及格"（60分及格）。

**想一想**：如果你来判断，你会怎么做？

**Step 1**：判断条件是什么？
```
成绩 >= 60 → 及格
成绩 < 60  → 不及格
```

**Step 2**：用代码实现

```cpp
#include <iostream>
using namespace std;

int main() {
    int score;
    cout << "请输入成绩：";
    cin >> score;
    
    if (score >= 60) {
        cout << "及格！" << endl;
    } else {
        cout << "不及格，继续努力！" << endl;
    }
    
    return 0;
}
```

**试一试**：输入50、60、90，分别观察输出！

---

#### 🤔 思考题

**问题1**：如果要把成绩分成"优秀"(≥90)、"良好"(80-89)、"及格"(60-79)、"不及格"(<60)四档，怎么写？

<details>
<summary>💡 需要提示？</summary>

**提示**：用 `else if` 连接多个条件，像梯子一样一级一级判断。
</details>

<details>
<summary>✅ 参考答案</summary>

```cpp
if (score >= 90) {
    cout << "优秀！" << endl;
} else if (score >= 80) {
    cout << "良好！" << endl;
} else if (score >= 60) {
    cout << "及格！" << endl;
} else {
    cout << "不及格！" << endl;
}
```
**注意**：条件要从高到低判断！
</details>

### 🔍 探索2：循环——重复做一件事

**生活场景**：
- 每天背10个单词（重复7天）
- 跑步，直到跑满1000米
- 洗碗，直到所有碗洗完

程序中，用 **循环** 来处理重复的事情。

---

#### 🎯 挑战：打印5次"Hello"

**方法一：写5行代码（笨方法）**
```cpp
cout << "Hello" << endl;
cout << "Hello" << endl;
cout << "Hello" << endl;
cout << "Hello" << endl;
cout << "Hello" << endl;
```

**如果打印100次呢？** 😱

---

#### 🎯 挑战：用循环打印5次"Hello"

**使用 `for` 循环**：

```cpp
for (int i = 0; i < 5; i++) {
    cout << "Hello" << endl;
}
```

**分解理解**：
```cpp
for (初始化; 条件; 更新) {
    // 循环体（重复执行的代码）
}
```

| 部分 | 作用 | 本例 |
|:---|:---|:---|
| `int i = 0` | 初始化：从0开始 | 设置计数器 |
| `i < 5` | 条件：满足就继续 | 做5次（i=0,1,2,3,4）|
| `i++` | 更新：每次加1 | 计数器+1 |

**试一试**：把 `i < 5` 改成 `i < 10`，会发生什么？

---

#### 📝 填空练习

**打印1到10**：补全代码

```cpp
for (int i = ____; i <= ____; i____) {
    cout << i << endl;
}
```

<details>
<summary>✅ 答案</summary>

```cpp
for (int i = 1; i <= 10; i++) {
    cout << i << endl;
}
```
</details>

### 🔍 探索3：累加——循环的经典应用

**挑战**：计算 1+2+3+...+100 = ?

**分解思考**：
1. 需要一个变量存"总和"，初始值是？
2. 循环从1到100，每次做什么？
3. 最后输出总和

**代码框架**：
```cpp
int sum = ____;          // 初始值？
for (int i = 1; i <= 100; i++) {
    sum = sum + i;       // 或者 sum += i;
}
cout << "总和是：" << sum << endl;
```

<details>
<summary>💡 初始值提示</summary>

**提示**：总和初始应该是0，就像空盒子，等着往里加东西。
</details>

### 🟢 练习1：判断奇偶数（基础）

**要求**：输入一个整数，判断是奇数还是偶数。

**提示**：
- 偶数 ÷ 2 的余数是 0
- 取余运算符是 `%`

**填空**：
```cpp
int n;
cin >> n;
if (n ____ 2 == ____) {
    cout << "偶数" << endl;
} else {
    cout << "奇数" << endl;
}
```

<details>
<summary>✅ 答案</summary>

```cpp
if (n % 2 == 0) {
    cout << "偶数" << endl;
} else {
    cout << "奇数" << endl;
}
```
</details>

---

### 🟡 练习2：九九乘法表（进阶）

**要求**：打印九九乘法表。

**分解**：
1. 先打印一行：`1×1=1 1×2=2 ... 1×9=9`
2. 再打印9行

**提示**：需要**两个循环**，一个控制行，一个控制列。

<details>
<summary>✅ 参考答案</summary>

```cpp
for (int i = 1; i <= 9; i++) {          // 外层：控制行
    for (int j = 1; j <= 9; j++) {      // 内层：控制列
        cout << i << "×" << j << "=" << i*j << "\t";
    }
    cout << endl;                       // 每行结束换行
}
```
</details>

### 🔴 练习3：猜数字游戏（挑战）

**要求**：程序想一个1-100的数，用户猜，程序告诉"大了"或"小了"，直到猜中。

**分解**：
1. 生成随机数（先假设目标是50）
2. 循环：让用户输入猜测
3. 判断：告诉用户大了/小了/猜中了
4. 猜中了才结束循环

**提示**：用 `while` 循环，条件是没猜中就一直循环。

<details>
<summary>✅ 参考答案</summary>

```cpp
#include <iostream>
using namespace std;

int main() {
    int target = 50;        // 假设目标是50
    int guess;
    
    while (true) {          // 无限循环，猜中了break
        cout << "请猜一个1-100的数：";
        cin >> guess;
        
        if (guess > target) {
            cout << "大了！" << endl;
        } else if (guess < target) {
            cout << "小了！" << endl;
        } else {
            cout << "猜中了！" << endl;
            break;          // 猜中了，跳出循环
        }
    }
    
    return 0;
}
```
</details>

### 🐛 常见错误自查

| 错误 | 原因 | 解决 |
|:---|:---|:---|
| `if (a = 5)` | 用了赋值`=`而不是比较`==` | 判断相等用`==` |
| `for (i = 0; i < 5; i++);` | 循环后多了分号 | 分号导致循环体为空 |
| 死循环 | 循环条件永远为真 | 检查条件是否能变成假 |
