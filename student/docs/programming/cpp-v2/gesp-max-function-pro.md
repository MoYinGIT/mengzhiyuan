## 第10题：综合应用

**程序代码**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    // 计算多个矩形的最大面积
    int w1 = 3, h1 = 4;
    int w2 = 5, h2 = 2;
    int w3 = 4, h3 = 4;
    
    int area1 = w1 * h1;
    int area2 = w2 * h2;
    int area3 = w3 * h3;
    
    int max_area = max(max(area1, area2), area3);
    
    // 判断三角形是否能构成（两边之和大于第三边）
    int a = 3, b = 4, c = 5;
    bool valid = (a + b > c) && (a + c > b) && (b + c > a);
    int max_side = max(max(a, b), c);
    
    // 计算最大公约数的一部分（辗转相减法）
    int m = 48, n = 18;
    while (m != n) {
        if (m > n) m = m - n;
        else n = n - m;
    }
    int gcd_result = m;
    
    cout << max_area << " " << valid << " " << max_side << " " << gcd_result << endl;
    return 0;
}
```

## 知识点总结

| 考点 | 说明 |
|:---|:---|
| 基本功能 | 返回两个值中的较大值 |
| 头文件 | `<algorithm>` 或 `<bits/stdc++.h>` |
| 参数类型 | 两个参数必须类型相同 |
| 返回值 | 较大值的原类型 |
| 嵌套使用 | 可以嵌套比较多个值 |
| 常见错误 | 参数个数错误、类型不匹配、返回值未使用 |
| 实际应用 | 找最大值、计算范围、限制最小值、排序辅助 |
| 对比函数 | `min()` 返回较小值 |
