## 📚 知识探索

### 🔍 探索1：图——比树更复杂的关系

**树 vs 图**：
- 树：层次关系，没有环
- 图：任意关系，可以有环

**生活中的图**：
- 地图（城市是点，道路是边）
- 社交网络（人是点，关系是边）
- 电路图（元件是点，连线是边）

**图的存储**：

**邻接矩阵**：二维数组
```cpp
int g[100][100];  // g[i][j] = 1 表示i到j有边
```

**邻接表**： vector数组（推荐）
```cpp
vector<int> g[100];  // g[i]存储与i相连的所有点
// 添加边：g[1].push_back(2); 表示1→2有边
```

---

### 🔍 探索2：BFS——层层推进

**思想**：从起点开始，先访问所有邻居，再访问邻居的邻居...

**类比**：
- 你发朋友圈（第1层）
- 朋友转发（第2层）
- 朋友的朋友转发（第3层）
- ...层层扩散

**代码**：
```cpp
void bfs(int start) {
    queue<int> q;
    bool visited[100] = {false};  // 标记是否访问过
    
    q.push(start);
    visited[start] = true;
    
    while (!q.empty()) {
        int u = q.front();
        q.pop();
        cout << u << " ";  // 访问u
        
        // 访问u的所有邻居
        for (int v : g[u]) {
            if (!visited[v]) {
                visited[v] = true;
                q.push(v);
            }
        }
    }
}
```

**应用**：
- 最短路径（无权图）
- 连通性判断
- 迷宫问题

---

### 🔍 探索3：DFS——深度探索

**思想**：从一个点出发，一直走到底，走不通再回头

**类比**：走迷宫
- 选一个方向走到底
- 遇到死胡同，退回上一个岔路口
- 换另一个方向继续

**代码（递归版）**：
```cpp
bool visited[100];

void dfs(int u) {
    visited[u] = true;
    cout << u << " ";  // 访问u
    
    for (int v : g[u]) {
        if (!visited[v]) {
            dfs(v);  // 递归访问邻居
        }
    }
}
```

**应用**：
- 连通块计数
- 拓扑排序
- 判断是否有环

### 🔍 探索4：最短路径——Dijkstra算法

**问题**：从A城市到B城市，怎么走最快？

**Dijkstra思想**：
1. 从起点开始，维护到每个点的最短距离
2. 每次选距离最小的未访问点
3. 用这个点更新邻居的距离
4. 重复直到所有点都访问

**代码框架**：
```cpp
int dist[100];      // dist[i] = 起点到i的最短距离
bool visited[100];

void dijkstra(int start) {
    // 初始化
    for (int i = 0; i < n; i++) dist[i] = INF;
    dist[start] = 0;
    
    for (int i = 0; i < n; i++) {
        // 找未访问中距离最小的
        int u = -1;
        for (int j = 0; j < n; j++) {
            if (!visited[j] && (u == -1 || dist[j] < dist[u])) {
                u = j;
            }
        }
        
        visited[u] = true;
        
        // 更新邻居
        for (auto &edge : g[u]) {
            int v = edge.first;   // 邻居
            int w = edge.second;  // 边权
            if (dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
            }
        }
    }
}
```

**时间复杂度**：O(n²)，用优先队列可优化到O(m log n)

### 🔍 探索5：动态规划——最优子结构

**核心思想**：
- 大问题分解为小问题
- 小问题有重叠（不是分治）
- 从小问题推到大问题

**经典例子：爬楼梯**

**问题**：有n级台阶，每次走1步或2步，有多少种走法？

**分析**：
- 最后一步可能是从n-1上来（走了1步）
- 最后一步可能是从n-2上来（走了2步）
- 所以：`f(n) = f(n-1) + f(n-2)`

**代码**：
```cpp
int climbStairs(int n) {
    if (n <= 2) return n;
    
    int dp[100];  // dp[i] = 到第i级的方法数
    dp[1] = 1;    // 1级：1种（走1步）
    dp[2] = 2;    // 2级：2种（1+1或2）
    
    for (int i = 3; i <= n; i++) {
        dp[i] = dp[i-1] + dp[i-2];
    }
    
    return dp[n];
}
```

**DP三要素**：
1. **状态定义**：`dp[i]`表示什么
2. **状态转移**：`dp[i]`怎么从`dp[i-1]`等推导
3. **初始条件**：`dp[0]`、`dp[1]`是多少

### 🟢 练习1：图的连通性（基础）

**要求**：给定一个图，判断从起点能否到达终点。

**思路**：用BFS或DFS遍历，看是否能访问到终点。

<details>
<summary>✅ 参考答案</summary>

```cpp
bool canReach(int start, int end) {
    bool visited[100] = {false};
    queue<int> q;
    
    q.push(start);
    visited[start] = true;
    
    while (!q.empty()) {
        int u = q.front();
        q.pop();
        
        if (u == end) return true;
        
        for (int v : g[u]) {
            if (!visited[v]) {
                visited[v] = true;
                q.push(v);
            }
        }
    }
    return false;
}
```

### 🟡 练习2：最长递增子序列（进阶）

**问题**：给定序列 [10, 9, 2, 5, 3, 7, 101, 18]，最长递增子序列是？

**答案**：[2, 3, 7, 101]，长度4

**DP思路**：
- `dp[i]` = 以第i个数结尾的最长递增子序列长度
- 对于每个i，看前面所有j，如果`nums[j] < nums[i]`，则`dp[i] = max(dp[i], dp[j]+1)`

<details>
<summary>✅ 参考答案</summary>

```cpp
int lengthOfLIS(vector<int>& nums) {
    int n = nums.size();
    if (n == 0) return 0;
    
    int dp[1000];
    int ans = 1;
    
    for (int i = 0; i < n; i++) {
        dp[i] = 1;  // 至少包含自己
        for (int j = 0; j < i; j++) {
            if (nums[j] < nums[i]) {
                dp[i] = max(dp[i], dp[j] + 1);
            }
        }
        ans = max(ans, dp[i]);
    }
    
    return ans;
}
```
</details>

### 🔴 练习3：01背包问题（挑战）

**问题**：有N件物品，每件有重量w和价值v。背包容量为C，怎么装价值最大？

**DP思路**：
- `dp[i][j]` = 前i件物品，容量为j时的最大价值
- 对于第i件：不选 → `dp[i-1][j]`，选 → `dp[i-1][j-w[i]] + v[i]`
- `dp[i][j] = max(不选, 选)`

**空间优化**：用一维数组，倒序更新

<details>
<summary>✅ 参考答案</summary>

```cpp
int knapsack(int w[], int v[], int n, int C) {
    int dp[1000] = {0};  // dp[j] = 容量j时的最大价值
    
    for (int i = 0; i < n; i++) {
        for (int j = C; j >= w[i]; j--) {  // 倒序！
            dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
        }
    }
    
    return dp[C];
}
```
</details>

## 📚 知识速查表

| 算法 | 时间复杂度 | 应用 |
|:---|:---:|:---|
| BFS | O(V+E) | 最短路（无权）、连通性 |
| DFS | O(V+E) | 连通块、拓扑排序 |
| Dijkstra | O(V²) 或 O(E log V) | 单源最短路（非负权）|
| DP | 因题而异 | 最优化问题 |

**DP经典模型**：
- 线性DP：LIS、LCS、背包
- 区间DP：石子合并、矩阵链乘
- 树形DP：树上最大独立集
- 状态压缩DP：旅行商问题
