## 【大题1】格式字符串的"类型匹配陷阱"

**考察点**：scanf/printf 格式符与实际变量类型必须匹配

```cpp
#include <cstdio>
int main() {
    int a;
    float b;
    double c;
    
    // 错误用法演示
    scanf("%f", &a);      // 语句1：用%f读int
    scanf("%d", &b);      // 语句2：用%d读float
    scanf("%f", &c);      // 语句3：用%f读double
    
    printf("%d %f %f\n", a, b, c);
    return 0;
}
```

**假设输入**：`3.14 100 2.71828`

### 一、选择题

**1. 语句1 `scanf("%f", &a)` 执行后（ ）**

A. a正确读取3  
B. a读取到垃圾值  
C. 编译错误  
D. 运行时崩溃

**2. 语句2 `scanf("%d", &b)` 执行后（ ）**

A. b正确读取100.0  
B. b读取到垃圾值  
C. 读取失败，b保持原值  
D. 程序崩溃

**3. 语句3应该用哪个格式符读取double（ ）**

A. %f  
B. %lf  
C. %d  
D. %c

### 二、判断题

**4. scanf中，%f对应float*，%lf对应double*，%d对应int*，必须严格匹配。**  
（ ）正确 （ ）错误

**5. 如果格式符与变量类型不匹配，会导致未定义行为，可能读取到垃圾值。**  
（ ）正确 （ ）错误

**6. printf中，float会自动提升为double，所以%f既可以输出float也可以输出double。**  
（ ）正确 （ ）错误

### 三、填空题

**7. 读取int变量a，应该用 `scanf("______", &a);`**

**8. 读取double变量x，应该用 `scanf("______", &x);`**

**9. printf输出double变量y，应该用 `printf("______", y);`**

---

**解析关键点**：
- scanf：%f→float*, %lf→double*, %d→int*，必须严格匹配
- printf：float会自动提升为double，所以%f可以输出两者
- 类型不匹配 = 未定义行为

## 【大题2】%c读取字符的"空白陷阱"

**考察点**：%c 会读取任何字符，包括空格、换行符

```cpp
#include <cstdio>
int main() {
    int age;
    char grade;
    char name[20];
    
    scanf("%d", &age);           // 输入：18\n
    scanf("%c", &grade);        // 语句1
    scanf("%s", name);          // 语句2
    
    printf("age=%d, grade=[%c], name=[%s]\n", age, grade, name);
    return 0;
}
```

**输入**：`18
A Zhang`

### 一、选择题

**1. 语句1执行后，grade的值是（ ）**

A. 'A'  
B. '\n'（换行符）  
C. ' '（空格）  
D. 未定义

**2. 语句2执行后，name的值是（ ）**

A. "A"  
B. " Zhang"  
C. "Zhang"  
D. 读取失败

**3. 要让scanf("%c", &grade)跳过空白读取'A'，应该写成（ ）**

A. scanf(" %c", &grade)  
B. scanf("\n%c", &grade)  
C. scanf("%c ", &grade)  
D. scanf("%c\n", &grade)

### 二、判断题

**4. %c 会读取下一个字符，包括空格、换行符等空白字符。**  
（ ）正确 （ ）错误

**5. %c 前面加空格 `" %c"` 可以跳过所有空白字符，读取第一个非空白字符。**  
（ ）正确 （ ）错误

**6. 读取数字后直接用%c读取字符，往往会读到输入数字时留下的换行符。**  
（ ）正确 （ ）错误

### 三、填空题

**7. 输入数字后想读取一个字符（跳过换行），格式字符串应写成 `"______%c"`**。

**8. scanf("%c", &ch) 读取时，如果缓冲区中有'\n'，ch的值将是 ______（用字符常量表示）。**

**9. 为了避免读到换行符，可以在%c前加 ______ 或先用 ______ 清空缓冲区。**

---

**解析关键点**：
- %c 读取任何字符，包括\n
- 数字输入后留下\n，%c直接读到\n
- " %c"（%c前有空格）跳过所有空白
- 或用getchar()消耗\n

## 【大题3】%s读取字符串的"溢出陷阱"

**考察点**：%s 不检查缓冲区大小，可能导致缓冲区溢出

```cpp
#include <cstdio>
int main() {
    char name[5];  // 只能装4个字符+1个'\0'
    
    printf("请输入名字：");
    scanf("%s", name);  // 危险！
    
    printf("你好，%s\n", name);
    return 0;
}
```

### 一、选择题

**1. 如果输入"Tom"（3个字符），程序（ ）**

A. 正常运行，输出"你好，Tom"  
B. 缓冲区溢出  
C. 编译错误  
D. 崩溃

**2. 如果输入"Alexander"（9个字符），程序可能（ ）**

A. 正常运行，只显示前4个字符  
B. 缓冲区溢出，可能覆盖其他数据或崩溃  
C. 自动截断到4个字符  
D. 提示输入太长

**3. 安全的写法是（ ）**

A. scanf("%4s", name)  
B. scanf("%5s", name)  
C. scanf("%s", &name)  
D. 不能用scanf读字符串

### 二、判断题

**4. %s 读取字符串时，会在末尾自动添加'\0'结束符。**  
（ ）正确 （ ）错误

**5. scanf("%s", name) 不限制长度，如果输入超过数组大小，会导致缓冲区溢出。**  
（ ）正确 （ ）错误

**6. scanf("%4s", name) 最多读取4个字符，自动添加'\0'，不会溢出。**  
（ ）正确 （ ）错误

### 三、填空题

**7. char str[10] 用scanf读取时，为了安全，应该用 `"%______s"`，最多读取9个字符。**

**8. %s 遇到______字符会停止读取。**

**9. 缓冲区溢出可能导致程序______或______被覆盖。**

---

**解析关键点**：
- %s 不检查边界，危险！
- 必须用"%ns"限制长度，n=数组大小-1
- char[5] 最多存4个字符+'\0'，用"%4s"

## 【大题4】printf的"格式化精度陷阱"

**考察点**：printf 格式控制符的精度设置

```cpp
#include <cstdio>
int main() {
    double pi = 3.1415926535;
    int num = 42;
    char str[] = "Hello";
    
    printf("[%f]\n", pi);           // 输出1
    printf("[%.2f]\n", pi);        // 输出2
    printf("[%10f]\n", pi);        // 输出3
    printf("[%10.2f]\n", pi);      // 输出4
    printf("[%04d]\n", num);       // 输出5
    printf("[%.3s]\n", str);       // 输出6
    
    return 0;
}
```

### 一、选择题

**1. 输出2 `[%.2f]` 的结果是（ ）**

A. [3.14]  
B. [3.1415926535]  
C. [3.1]  
D. [.2f]

**2. 输出4 `[%10.2f]` 的结果是（ ）**

A. [3.14]（前面6个空格）  
B. [3.14]（前面8个空格）  
C. [3.141592]  
D. [3.14]（总宽度10，小数2位）

**3. 输出6 `[%.3s]` 的结果是（ ）**

A. [Hello]  
B. [Hel]  
C. [lo]  
D. [Hello..]

### 二、判断题

**4. `%.2f` 表示保留2位小数，四舍五入。**  
（ ）正确 （ ）错误

**5. `%10f` 表示总宽度为10，右对齐，不足补空格。**  
（ ）正确 （ ）错误

**6. `%04d` 表示总宽度为4，不足4位前面补0。**  
（ ）正确 （ ）错误

### 三、填空题

**7. `printf("%.2f", 3.14159)` 输出 ______。**

**8. `printf("%08d", 123)` 输出 ______（前面补几个0）。**

**9. `printf("%6.2f", 3.1)` 中，6表示______，2表示______。**

---

**解析关键点**：
- %.nf：小数点后n位
- %mf：总宽度m，右对齐
- %m.nf：总宽度m，小数n位
- %04d：宽度4，前补0
- %.ns：字符串前n个字符

## 核心要点总结

### scanf 安全使用检查清单

- [ ] %d 对应 int*，%f 对应 float*，%lf 对应 double*
- [ ] %c 前加空格跳过空白：" %c"
- [ ] %s 必须限制长度："%99s"（数组大小-1）
- [ ] 检查返回值，确保读取成功
- [ ] 考虑使用 fgets + sscanf 更安全

### printf 安全使用检查清单

- [ ] 格式符数量与参数数量匹配
- [ ] %.nf 控制小数位，%mf 控制宽度
- [ ] %s 要确保字符串以'\0'结尾
- [ ] 注意整数溢出时不要使用%d打印大数

### 常见错误对比

| 错误写法 | 问题 | 正确写法 |
|:---|:---|:---|
| scanf("%f", &x) // x是double | 类型不匹配 | scanf("%lf", &x) |
| scanf("%d%c", &n, &ch) | %c读到换行 | scanf("%d %c", &n, &ch) |
| scanf("%s", str) // str[10] | 缓冲区溢出 | scanf("%9s", str) |
| printf("%d %d", a) | 参数缺失 | printf("%d %d", a, b) |
| gets(buffer) | 已废弃，不安全 | fgets(buffer, size, stdin) |
