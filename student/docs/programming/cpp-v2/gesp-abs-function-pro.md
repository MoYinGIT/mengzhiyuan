## 核心要点总结

### abs()使用检查清单

- [ ] abs()求整数的绝对值
- [ ] abs(正数) = 正数本身
- [ ] abs(负数) = 去掉负号
- [ ] abs(0) = 0
- [ ] abs()返回int类型
- [ ] 浮点数用fabs()，不是abs()
- [ ] abs()不修改原变量，只是返回值
- [ ] 要改变原变量，需要a = abs(a)
- [ ] abs(a-b)表示两数距离
- [ ] 头文件：cstdlib 或 cmath

### 常见错误对比

| 错误写法 | 问题 | 正确写法 |
|:---|:---|:---|
| abs(b) | b没变 | b = abs(b) |
| abs(-7.5) | 可能截断 | fabs(-7.5) |
| abs(a-b) > 0 | 总是真（除了a=b） | a != b |
| 忘记头文件 | 编译警告 | `#include <cstdlib>` |

### GESP一二级abs()考点

1. **基本功能**：求整数的绝对值
2. **返回值**：正数不变，负数变正，0不变
3. **返回值类型**：int
4. **头文件**：cstdlib 或 cmath
5. **应用场景**：计算距离、判断误差、比较大小
6. **常见错误**：不修改原变量、与fabs()混淆
7. **自定义实现**：用if或三目运算符实现
8. **边界问题**：abs(INT_MIN)的溢出（了解即可）
9. **数学性质**：|a|+|b| vs |a+b|，双重abs
10. **实际应用**：数轴距离、误差检查、数据校验
