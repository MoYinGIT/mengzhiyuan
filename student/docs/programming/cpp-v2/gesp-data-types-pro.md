## 核心要点总结

### 数据类型使用检查清单

- [ ] int用于整数，范围约-21亿~21亿
- [ ] char用于单字符，本质是1字节小整数
- [ ] double用于小数，注意精度问题
- [ ] bool用于真/假，输出1/0
- [ ] sizeof获取类型大小
- [ ] 整数除法截断小数，不是四舍五入
- [ ] int与double运算，结果提升为double
- [ ] 强制转换：(类型)表达式 或 类型(表达式)
- [ ] 浮点数避免用==比较
- [ ] 数字字符转整数：ch - '0'

### 常见错误对比

| 错误写法 | 问题 | 正确写法 |
|:---|:---|:---|
| 5/2 | 结果是2，不是2.5 | 5/2.0 或 (double)5/2 |
| (double)(5/2) | 先整数除得2，再转2.0 | (double)5/2 |
| int avg = sum/3 | 截断小数 | double avg = (double)sum/3 |
| if (c == 0.3) | 浮点数精度问题 | if (\|c-0.3\| < 0.000001) |
| char = 65 | 易混淆 | char = 'A'（更清晰） |
| bool(100) == 100 | bool转int是1 | bool(100) == true |
