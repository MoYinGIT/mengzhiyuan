    int w3 = 4, h3 = 4;

    int area2 = w2 * h2;

## 知识点总结

| 考点 | 说明 |
|:---|:---|
| 基本功能 | 返回两个值中的较小值 |
| 头文件 | `<algorithm>` 或 `<bits/stdc++.h>` |
| 参数类型 | 两个参数必须类型相同 |
| 返回值 | 较小值的原类型 |
| 嵌套使用 | 可以嵌套比较多个值 |
| 常见错误 | 参数个数错误、类型不匹配、返回值未使用 |
| 实际应用 | 找最小值、限制上限、确保非正、排序辅助 |
| 对比函数 | `max()` 返回较大值 |
| 负数处理 | `min(-5, -3)` 返回 -5（绝对值更大） |
