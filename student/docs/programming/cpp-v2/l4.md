## 📚 知识探索

### 🔍 探索1：从重复代码到函数

**场景**：计算多个数组的平均值

❌ **笨方法**：复制粘贴代码
```cpp
// 计算第1个数组的平均值
int sum1 = 0;
for (int i = 0; i < 5; i++) sum1 += arr1[i];
double avg1 = sum1 / 5.0;

// 计算第2个数组的平均值（代码几乎一样！）
int sum2 = 0;
for (int i = 0; i < 5; i++) sum2 += arr2[i];
double avg2 = sum2 / 5.0;

// 第3个、第4个... 要重复写多少遍？
```

**想一想**：这些代码有什么共同点？
- 都是计算平均值
- 逻辑完全一样
- 只有输入的数组不同

✅ **聪明方法**：把重复代码**打包**成一个「工具」！

这就是**函数**——代码的「工具箱」。

---

### 🔍 探索2：创建你的第一个函数

**生活类比**：
```
函数 = 一个工具
    ├── 输入：原材料（参数）
    ├── 处理：加工步骤（函数体）
    └── 输出：成品（返回值）

例如：榨汁机
    ├── 输入：水果
    ├── 处理：榨汁
    └── 输出：果汁
```

**代码示例**：计算两个数的和

```cpp
// 1. 定义函数（制造工具）
int add(int a, int b) {    // int = 返回类型, add = 函数名, (int a, int b) = 参数
    int result = a + b;    // 函数体：具体做什么
    return result;         // 返回值
}

// 2. 使用函数（使用工具）
int main() {
    int x = add(3, 5);     // 调用函数，x = 8
    cout << x << endl;
    return 0;
}
```

**函数三要素**：
| 要素 | 说明 | 示例 |
|:---|:---|:---|
| **返回值类型** | 输出什么 | `int` |
| **函数名** | 工具的名字 | `add` |
| **参数** | 输入什么 | `(int a, int b)` |

### 📝 填空练习

**定义一个求最大值的函数**：
```cpp
___ max(___ a, ___ b) {    // 填空：返回类型、参数类型
    if (a > b) {
        ______;             // 填空：返回什么？
    } else {
        ______;             // 填空：返回什么？
    }
}
```

<details>
<summary>✅ 答案对照</summary>

```cpp
int max(int a, int b) {
    if (a > b) {
        return a;
    } else {
        return b;
    }
}
```
</details>

### 🔍 探索3：void函数——不返回值的函数

**场景**：打印一行分隔线

```cpp
void printLine() {         // void = 不返回值
    cout << "============" << endl;
}

int main() {
    printLine();           // 调用，输出分隔线
    cout << "菜单" << endl;
    printLine();           // 再调用一次
    return 0;
}
```

**输出**：
```
============
菜单
============
```

**想一想**：为什么用 `void`？
- 这个函数只需要"做事"（打印）
- 不需要"给出结果"
- 就像「开机」这个操作，做了就行，不用返回什么

### 🔍 探索4：参数传递——复制 vs 修改

**重要概念**：函数参数默认是「复制」一份给函数

```cpp
void addTen(int x) {       // x是num的复制品
    x = x + 10;            // 修改的是复制品x
    cout << "函数里x=" << x << endl;  // 输出20
}

int main() {
    int num = 10;
    addTen(num);           // 把num的值复制给x
    cout << "main里num=" << num << endl;  // 还是10！
    return 0;
}
```

**发现了吗？**
- 函数里修改了 `x`
- 但 `main` 里的 `num` 没变！
- 因为 `x` 只是 `num` 的复制品

**类比**：
```
你把作业复印一份给同学
同学在复印件上涂鸦
你的原件还是干净的
```

### 🟢 练习1：判断素数（基础）

**要求**：编写函数 `isPrime(int n)`，判断n是否是素数，返回true/false。

**提示**：
- 素数：只能被1和自身整除的数
- 2是最小的素数
- 用循环检查2到n-1是否能整除n

**分解**：
1. 定义函数，返回类型是 `bool`
2. 如果n<2，返回false
3. 循环检查2到n-1
4. 如果发现能整除的，返回false
5. 都没发现，返回true

<details>
<summary>✅ 参考答案</summary>

```cpp
bool isPrime(int n) {
    if (n < 2) return false;
    for (int i = 2; i < n; i++) {
        if (n % i == 0) return false;  // 发现能整除的，不是素数
    }
    return true;  // 没找到，是素数
}

int main() {
    int x;
    cin >> x;
    if (isPrime(x)) {
        cout << "是素数" << endl;
    } else {
        cout << "不是素数" << endl;
    }
    return 0;
}
```
</details>

### 🟡 练习2：数组求和函数（进阶）

**要求**：编写函数 `int arraySum(int arr[], int n)`，计算数组前n个元素的和。

**为什么需要n？** 数组传给函数时，函数不知道数组有多大！

<details>
<summary>✅ 参考答案</summary>

```cpp
int arraySum(int arr[], int n) {
    int sum = 0;
    for (int i = 0; i < n; i++) {
        sum += arr[i];
    }
    return sum;
}

int main() {
    int scores[5] = {85, 90, 78, 92, 88};
    int total = arraySum(scores, 5);
    cout << "总分：" << total << endl;
    return 0;
}
```
</details>

### 🔴 练习3：递归入门（挑战）

**递归**：函数调用自己

**经典例子**：阶乘
- 5! = 5 × 4 × 3 × 2 × 1
- 5! = 5 × 4!
- 4! = 4 × 3!
- ...
- 1! = 1（结束条件）

**试一试**：补全阶乘函数
```cpp
int factorial(int n) {
    if (n == 1) return ____;      // 填空：结束条件
    return n * factorial(____);   // 填空：递归调用
}
```

<details>
<summary>✅ 答案对照</summary>

```cpp
int factorial(int n) {
    if (n == 1) return 1;         // 1的阶乘是1
    return n * factorial(n - 1);  // n! = n × (n-1)!
}
```
</details>

**递归思维要点**：
1. 找到**结束条件**（不然会无限循环）
2. 找到**递推关系**（大问题分解为小问题）
