## 🎯 DP学习路线

### 第一阶段：理解DP思想
1. 什么是最优子结构？
2. 什么是无后效性？
3. 如何定义状态？
4. 如何写出状态转移方程？

### 第二阶段：掌握经典模型
1. 线性DP：LIS、LCS
2. 背包问题：01、完全、多重
3. 区间DP：石子合并、编辑距离
4. 树形DP：树的最大独立集

### 第三阶段：综合运用
1. 根据问题特征选择DP模型
2. 状态压缩优化
3. 单调队列优化
4. 斜率优化
