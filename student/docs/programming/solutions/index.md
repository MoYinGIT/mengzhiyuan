## 🎯 如何使用

### 学习路径建议

```
第一阶段：基础篇 (2-4周)
    ↓ 掌握数组、字符串、基础数学
第二阶段：搜索篇 (3-4周)
    ↓ 理解BFS/DFS、Flood Fill
第三阶段：动态规划篇 (4-6周)
    ↓ 攻克背包、LIS、树形DP
```

### 阅读方法

1. **先读题** — 独立思考 10 分钟
2. **看思路** — 对比自己的思路差异
3. **写代码** — 不看答案，独立实现
4. **对答案** — 找出差距，记录要点
5. **做总结** — 归纳此类题的通解模板
