## 🎯 学习建议

### 第一阶段：理解BFS
1. 掌握队列（Queue）的基本操作
2. 理解「层序遍历」的思想
3. 学会记录访问状态（vis数组）

### 第二阶段：理解DFS
1. 掌握递归的基本原理
2. 理解回溯法（Backtracking）
3. 学会剪枝优化

### 第三阶段：综合应用
1. 根据问题特点选择BFS或DFS
2. BFS适合求最短路径
3. DFS适合枚举所有方案
