## 📝 代码实现

### C语言版本

```c
/*
 * 题目18：A1160 倒序数
 * 
 * 将一个整数的各位数字逆序输出
 */

#include &lt;stdio.h&gt;

int main() {
    int n;
    scanf("%d", &n);
    
    int rev = 0;
    while (n &gt; 0) {
        rev = rev * 10 + n % 10;
        n /= 10;
    }
    
    printf("%d\n", rev);
    
    return 0;
}

```

### C++版本
