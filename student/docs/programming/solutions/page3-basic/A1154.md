## 📝 代码实现

### C语言版本

```c
/*
 * 题目12：A1154 亲和数
 * 
 * 亲和数：如果a的所有真因子之和等于b，且b的所有真因子之和等于a，则a和b是亲和数
 * 
 * 真因子：除了自身以外的因子
 */

#include &lt;stdio.h&gt;

// 计算n的所有真因子之和
int sum_of_factors(int n) {
    int sum = 0;
    for (int i = 1; i &lt; n; i++) {
        if (n % i == 0) {
            sum += i;
        }
    }
    return sum;
}

int main() {
    int a, b;
    scanf("%d %d", &a, &b);
    
    if (sum_of_factors(a) == b && sum_of_factors(b) == a) {
        printf("yes\n");
    } else {
        printf("no\n");
    }
    
    return 0;
}

```
