## 📝 代码实现

### C++版本

```cpp
/*
 * 题目22：A1164 digit函数
 */

#include &lt;iostream&gt;
using namespace std;

int digit(int n, int k) {
    for (int i = 1; i &lt; k; i++) {
        n /= 10;
        if (n == 0) return -1;
    }
    return n % 10;
}

int main() {
    int n, k;
    cin &gt;&gt; n &gt;&gt; k;
    cout &lt;&lt; digit(n, k) &lt;&lt; endl;
    return 0;
}

```
