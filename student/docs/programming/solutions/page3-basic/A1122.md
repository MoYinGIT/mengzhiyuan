# A1122 计算鞍点

---

## 📖 题目描述

给定一个 $n \times m$ 的矩阵，找出其中的**鞍点**。

**鞍点定义**：矩阵中某个元素是其所在行的最大值，同时是其所在列的最小值。

**输入格式**：  
第一行两个整数 $n, m$ $(1 \leq n, m \leq 100)$  
接下来 $n$ 行，每行 $m$ 个整数

**输出格式**：  
如果存在鞍点，输出其行号、列号和值（行号和列号从0开始）  
如果不存在，输出 `none`

**数据保证**：矩阵中至多只有一个鞍点

---

## 💡 思路分析

### 算法步骤

1. **遍历每一行**，找出该行的最大值及其列下标
2. **验证该元素**是否是其所在列的最小值
3. **若满足条件**，即为鞍点，输出结果

### 为什么这样可行？

- 鞍点必须满足两个条件，缺一不可
- 先找行最大可以缩小搜索范围
- 再找列最小验证第二个条件

## 📝 代码实现

### C语言版本

```c
#include <stdio.h>

int main() {
    int n, m;
    int a[100][100];
    
    scanf("%d %d", &n, &m);
    
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < m; j++) {
            scanf("%d", &a[i][j]);
        }
    }
    
    int found = 0;
    int saddle_row, saddle_col, saddle_val;
    
    for (int i = 0; i < n; i++) {
        // 找第i行的最大值
        int max_val = a[i][0];
        int max_col = 0;
        for (int j = 1; j < m; j++) {
            if (a[i][j] > max_val) {
                max_val = a[i][j];
                max_col = j;
            }
        }
        
        // 检查是否是该列的最小值
        int is_min = 1;
        for (int k = 0; k < n; k++) {
            if (a[k][max_col] < max_val) {
                is_min = 0;
                break;
            }
        }
        
        if (is_min) {
            found = 1;
            saddle_row = i;
            saddle_col = max_col;
            saddle_val = max_val;
            break; // 题目保证只有一个鞍点
        }
    }
    
    if (found) {
        printf("%d %d %d\n", saddle_row, saddle_col, saddle_val);
    } else {
        printf("none\n");
    }
    
    return 0;
}
```

### C++版本

```cpp
#include <iostream>
using namespace std;

int main() {
    int n, m;
    int a[100][100];
    
    cin >> n >> m;
    
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            cin >> a[i][j];
    
    for (int i = 0; i < n; i++) {
        // 找行最大
        int max_col = 0;
        for (int j = 1; j < m; j++)
            if (a[i][j] > a[i][max_col])
                max_col = j;
        
        // 检查列最小
        bool is_min = true;
        for (int k = 0; k < n; k++)
            if (a[k][max_col] < a[i][max_col])
                is_min = false;
        
        if (is_min) {
            cout << i << " " << max_col << " " << a[i][max_col] << endl;
            return 0;
        }
    }
    
    cout << "none" << endl;
    return 0;
}
```

## ⏱️ 复杂度分析

| 指标 | 复杂度 | 说明 |
|:---|:---:|:---|
| 时间 | $O(n \times m)$ | 遍历矩阵两次 |
| 空间 | $O(n \times m)$ | 存储矩阵 |

---

## 🎯 关键点总结

1. **行最大、列最小** — 两个条件缺一不可
2. **先找行最大** — 缩小搜索范围
3. **注意数组下标** — 从0开始还是从1开始
4. **及时break** — 找到后可提前退出

---

## 🤔 扩展思考

1. **如果矩阵有多个鞍点怎么办？**  
   修改代码，不break，收集所有鞍点。

2. **如何优化到 $O(n \times m)$ 只遍历一次？**  
   预处理每行最大值和每列最小值，然后检查交点。

3. **鞍点一定存在吗？**  
   不一定，例如：
   ```
   1 2
   3 4
   ```
   此矩阵无鞍点。
