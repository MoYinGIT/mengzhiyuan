## 📝 代码实现

### C语言版本

```c
/*
 * 题目20：A1162 字符串逆序
 * 
 * 将输入的字符串逆序输出
 */

#include &lt;stdio.h&gt;
#include &lt;string.h&gt;

int main() {
    char s[1000];
    scanf("%s", s);
    
    int len = strlen(s);
    for (int i = len - 1; i &gt;= 0; i--) {
        printf("%c", s[i]);
    }
    printf("\n");
    
    return 0;
}

```

### C++版本

```cpp

 * 题目20：A1162 字符串逆序

}
```
