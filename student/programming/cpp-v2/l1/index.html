🎯 本级别目标

学完L1，你将能够：

  • ✅ 理解计算机基本组成和程序运行原理
  • ✅ 熟练使用Dev-C++编写、编译、运行程序
  • ✅ 掌握C++基本语法：变量、数据类型、输入输出
  • ✅ 使用三大结构（顺序、分支、循环）解决问题
  • ✅ 编写20-50行的简单程序
