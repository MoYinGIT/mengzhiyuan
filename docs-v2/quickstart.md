## 📚 知识探索

### 🔍 探索1：什么是IDE？

**生活场景**：
> 想象一下，你要写一篇作文。你可以用简单的记事本，也可以用功能丰富的Word。记事本虽然简单，但没有拼写检查、没有格式调整；Word则能帮你检查错别字、自动排版、插入图片。

**编程中的IDE**就像Word之于写作文——它是程序员的"智能编辑器"。

**IDE = Integrated Development Environment（集成开发环境）**

| 功能 | 作用 | 生活类比 |
|:---|:---|:---|
| 代码编辑 | 写程序的地方 | 写字的纸 |
| 语法高亮 | 用颜色区分代码 | 荧光笔标记 |
| 自动补全 | 帮你写完代码 | 输入法联想 |
| 编译运行 | 把代码变成程序 | 打印机输出 |
| 调试工具 | 找错误 | 橡皮擦改错 |

**想一想**：为什么初学者需要IDE，而不能直接用记事本编程？

<details>
<summary>💡 需要提示？</summary>

IDE就像是学骑自行车的辅助轮——它帮你减少犯错的机会，让你专注于学习编程思维本身，而不是被各种技术细节困扰。

</details>

### 🔍 探索2：如何选择IDE？

**试一试**：看看下面的对比表格，选择最适合你的IDE。

| IDE | 适用人群 | 平台 | 安装难度 | 功能丰富度 |
|:---|:---|:---:|:---:|:---:|
| **Dev-C++** | 🟢 纯新手 | Windows | ⭐ 简单 | ⭐⭐ 基础 |
| **VS Code** | 🟡 有一定基础 | 全平台 | ⭐⭐ 中等 | ⭐⭐⭐⭐⭐ 强大 |
| **CLion** | 🔴 专业选手 | 全平台 | ⭐⭐⭐ 复杂 | ⭐⭐⭐⭐⭐ 专业 |

**选择建议**：

```
你是Windows用户且第一次接触编程？
    ↓ 是 → 选择 Dev-C++
    ↓ 否 → 继续看
        
你需要跨平台（Mac/Linux）或想要更多功能？
    ↓ 是 → 选择 VS Code
    ↓ 否 → 选择 Dev-C++
```

**发现规律了吗？** 🤔 工具选择的核心原则是：**够用就好，循序渐进**

## ✍️ 动手实践

### 🟢 练习1：安装 Dev-C++（Windows新手推荐）

**目标**：在Windows电脑上安装Dev-C++并运行Hello World

**分解步骤**：

#### Step 1：下载安装包

1. 打开浏览器，访问：[https://sourceforge.net/projects/orwelldevcpp/](https://sourceforge.net/projects/orwelldevcpp/)
2. 点击绿色按钮 **Download**
3. 等待下载完成（约50MB）

<details>
<summary>💡 下载太慢？</summary>

- 如果SourceForge下载慢，可以搜索"Dev-C++ 百度网盘"找国内镜像
- 或请老师/同学分享安装包

</details>

#### Step 2：运行安装程序

1. 找到下载的文件（通常在"下载"文件夹）
2. 双击运行 `Dev-Cpp-5.x.x-TDM-GCC-x64-Setup.exe`
3. 在弹出的窗口中选择语言：**简体中文**

| 步骤 | 截图说明 | 操作 |
|:---:|:---|:---|
| 1 | 选择语言 | 选"简体中文"，点"OK" |
| 2 | 许可协议 | 点击"我接受" |
| 3 | 选择组件 | 保持默认，点"下一步" |
| 4 | 安装位置 | 建议保持默认，点"安装" |
| 5 | 等待安装 | 进度条走完，点"完成" |

#### Step 3：首次配置

1. 打开Dev-C++
2. 选择语言：**简体中文**
3. 选择配色方案：推荐选择**"经典"**（对眼睛友好）
4. 点击"确定"

#### Step 4：创建第一个程序

1. 点击菜单：**文件** → **新建** → **源代码**
2. 输入以下代码：

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, 默隐·蒙知苑!" << endl;
    return 0;
}
```

3. 按 **Ctrl+S** 保存，命名为 `hello.cpp`
4. 按 **F11** 编译并运行
5. 看到黑窗口输出文字？🎉 **恭喜你成功了！**

<details>
<summary>✅ 成功截图示例</summary>

```

--------------------------------
Process exited after 0.123 seconds with return value 0
请按任意键继续 . . .
```

</details>

### 🟡 练习2：安装 VS Code（全平台进阶选择）

**目标**：安装VS Code并配置C++开发环境

**分解步骤**：

#### Step 1：下载VS Code

1. 访问：[https://code.visualstudio.com/](https://code.visualstudio.com/)
2. 点击 **Download**
3. 系统会自动识别你的平台

| 平台 | 下载文件 | 特殊说明 |
|:---:|:---|:---|
| Windows | VSCodeSetup-x64-x.x.x.exe | 注意选择64位/32位 |
| Mac | VSCode-darwin-universal.zip | Intel和M1都可用 |
| Linux | .deb或.rpm包 | 根据发行版选择 |

#### Step 2：安装VS Code

**Windows用户**：
1. 运行安装程序
2. **重要！** 勾选"添加到PATH"选项
3. 一路"下一步"完成安装

**Mac用户**：
1. 解压zip文件
2. 将VS Code拖入"应用程序"文件夹
3. 首次打开可能需要右键选择"打开"

**Linux用户**：
```bash
# Ubuntu/Debian
sudo dpkg -i code_*.deb
sudo apt-get install -f
```

#### Step 3：安装C++插件

1. 打开VS Code
2. 点击左侧的 **扩展** 图标（四个方块）
3. 搜索 **C/C++**
4. 安装 Microsoft 官方的 **C/C++** 插件

| 必装插件 | 作用 |
|:---|:---|
| C/C++ | 代码高亮、调试支持 |
| Code Runner | 一键运行代码 |
| C++ Intellisense | 智能代码补全 |

#### Step 4：安装编译器

**Windows - 安装MinGW**：

1. 访问：[https://www.mingw-w64.org/downloads/](https://www.mingw-w64.org/downloads/)
2. 下载 **MinGW-W64 GCC-8.1.0**
3. 解压到 `C:\mingw64`
4. 添加到环境变量：
   - 右键"此电脑" → 属性 → 高级系统设置
   - 环境变量 → Path → 编辑
   - 添加 `C:\mingw64\bin`
5. 重启VS Code

<details>
<summary>💡 验证编译器安装</summary>

打开命令提示符（cmd），输入：
```bash
g++ --version
```

如果显示版本号，说明安装成功！

</details>

**Mac - 安装Xcode工具**：

1. 打开终端（Terminal）
2. 输入命令：
```bash
xcode-select --install
```
3. 按提示安装

**Linux**：
```bash
# Ubuntu/Debian
sudo apt-get install g++

# CentOS/RHEL
sudo yum install gcc-c++
```

#### Step 5：运行Hello World

1. 在VS Code中创建新文件
2. 输入Hello World代码（同上）
3. 保存为 `hello.cpp`
4. 按 **Ctrl+Alt+N** 运行（或点击右上角▶️）

### 🔴 练习3：环境配置常见问题排查

#### 问题1：Dev-C++编译报错"无法找到编译器"

| 症状 | 原因 | 解决 |
|:---|:---|:---|
| 编译时提示找不到gcc | 安装不完整或路径问题 | 重新安装Dev-C++ |
| 提示"系统找不到指定文件" | 杀毒软件误删 | 添加信任或关闭杀毒 |

**Step by Step解决**：
1. 关闭Dev-C++
2. 关闭杀毒软件
3. 重新安装Dev-C++（以管理员身份运行）
4. 安装时保持默认路径
5. 将安装目录加入杀毒软件白名单

#### 问题2：VS Code提示"无法找到g++"

**诊断流程**：

```
错误：无法找到g++
    ↓
打开cmd，输入g++ --version
    ↓
提示"不是内部或外部命令"？
    ↓ 是
MinGW没有正确添加到PATH
    ↓
检查环境变量设置
    ↓
重启VS Code
```

<details>
<summary>💡 详细排查步骤</summary>

1. **确认MinGW安装位置**：找到 `C:\mingw64\bin\g++.exe`
2. **检查环境变量**：
   - Win+R → 输入 `sysdm.cpl` → 回车
   - 高级 → 环境变量 → Path
   - 确认有 `C:\mingw64\bin`
3. **重启所有程序**：修改环境变量后需要重启VS Code和终端
4. **使用绝对路径测试**：在VS Code设置中指定完整路径

</details>

#### 问题3：Mac上提示"无法打开，因为无法验证开发者"

**解决**：
1. 右键点击VS Code图标
2. 选择"打开"
3. 在弹出的对话框中点击"打开"
4. 以后就可以正常双击打开了

#### 问题4：代码运行后窗口一闪而过

**原因**：程序运行太快，来不及看清结果

**解决**（Dev-C++）：
- 按 **F11** 而不是 **F10**（F11会暂停显示结果）

**解决**（VS Code）：
- 在代码最后添加：
```cpp
system("pause");  // Windows
// 或
getchar();        // 通用
```

### 🐛 常见错误自查

| 错误提示 | 可能原因 | 解决方法 |
|:---|:---|:---|
| "找不到编译器" | MinGW未安装或未配置PATH | 检查环境变量 |
| "无法打开源文件" | 文件名或路径有误 | 检查文件名，避免中文路径 |
| "expected ;" | 漏了分号 | 在行尾添加分号 |
| "cout was not declared" | 忘记using namespace std | 添加using声明 |
| 窗口一闪而过 | 程序结束太快 | 使用F11或添加pause |
