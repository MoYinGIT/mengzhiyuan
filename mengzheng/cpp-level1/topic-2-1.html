创建文件

⏱️ 预估学习时长: 20分钟 📋 验证题目: 180道 🎯 难度: 入门级

📚 知识详解

一、什么是C++源文件

C++源文件是存储C++程序代码的文本文件。程序员在源文件中编写指令，告诉计算机如何执行特定的任务。源文件是程序开发的基础，就像作家用笔和纸写作一样，程序员用源文件来「写」程序。

💡 类比理解
源文件就像是一张「乐谱」，程序员是「作曲家」，计算机是「演奏家」。乐谱上记录着音符（代码），演奏家按照乐谱演奏出美妙的音乐（程序运行）。

二、C++文件的扩展名

文件扩展名是文件名中点（.）后面的部分，用于标识文件的类型。C++编程中常见的文件扩展名有：

📄
.cpp — C++源文件（最常用）
包含C++程序的主要代码，如main.cpp、hello.cpp
📋
.h 或 .hpp — 头文件
包含函数声明和类定义，如iostream.h、math.h
⚙️
.o 或 .obj — 目标文件
编译器生成的中间文件，不是程序员直接编写的
🚀
.exe — 可执行文件（Windows）
编译链接后生成的可运行程序
⚠️ 重要提示
GESP考试中，C++源文件的标准扩展名是.cpp。在Dev-C++、VS Code等IDE中创建C++文件时，一定要使用正确的扩展名，否则可能导致编译错误。

三、文件命名规范

良好的文件命名习惯有助于代码管理和团队协作。以下是C++文件的命名建议：

  1. 使用有意义的名称：文件名应反映文件的功能，如calculator.cpp（计算器）、sort.cpp（排序）
  2. 使用英文字母、数字和下划线：避免使用中文、空格和特殊字符
  3. 区分大小写：C++是区分大小写的语言，Hello.cpp和hello.cpp是不同的文件
  4. 不要用数字开头：文件名应以字母或下划线开头
  5. 主函数文件习惯命名：包含main()函数的文件常命名为main.cpp
❌ 避免的文件名示例
• 我的程序.cpp（含中文）
• hello world.cpp（含空格）
• 123abc.cpp（数字开头）
• program$.cpp（特殊字符）
✅ 推荐的文件名示例
• main.cpp
• hello_world.cpp
• calculate_sum.cpp
• StudentInfo.cpp

四、在Dev-C++中创建文件

Dev-C++是一款轻量级的C++集成开发环境，适合初学者使用。以下是创建新文件的步骤：

1打开Dev-C++

启动Dev-C++应用程序，进入主界面。

2新建源文件

点击菜单栏「文件」→「新建」→「源代码」（快捷键：Ctrl+N）。

3保存文件

点击「文件」→「保存」（快捷键：Ctrl+S），选择保存位置，输入文件名（如hello.cpp），确保扩展名为.cpp。

4开始编写代码

在编辑器中输入C++代码。

五、在VS Code中创建文件

VS Code是一款功能强大的代码编辑器，配合C++扩展可以很好地支持C++开发：

1打开VS Code

启动VS Code，打开或创建一个文件夹作为工作区。

2新建文件

点击「文件」→「新建文件」（快捷键：Ctrl+N），或点击左侧文件资源管理器的「新建文件」按钮。

3命名并保存

输入文件名（如main.cpp），按回车确认。VS Code会根据扩展名.cpp自动识别为C++文件，并提供语法高亮。

六、你的第一个C++程序

下面是一个经典的「Hello, World!」程序，通常是程序员学习新语言时编写的第一个程序：

// 这是一个简单的C++程序 // 文件名: hello.cpp #include // 包含输入输出流库 using namespace std; // 使用标准命名空间 int main() { // 主函数，程序从这里开始执行 cout << "Hello, World!" << endl; // 输出文字到屏幕 return 0; // 程序正常结束，返回0 }

将上述代码输入到新建的文件中，保存为hello.cpp，你就完成了第一个C++文件的创建！接下来可以编译运行，看看程序是否成功输出「Hello, World!」。

七、常见错误与注意事项

⚠️ 常见错误
  1. 忘记加扩展名：保存时只写文件名hello，没有写hello.cpp
  2. 扩展名错误：使用.txt、.c或其他扩展名代替.cpp
  3. 文件名含中文或空格：导致编译器无法识别
  4. 保存位置找不到：文件保存后不知道在哪里
  5. 没有保存就编译：修改代码后忘记保存，编译的是旧版本
✅ 好习惯养成
  • 养成「写代码→保存→编译→运行」的工作流程
  • 定期保存文件（Ctrl+S），避免丢失工作成果
  • 为不同项目创建独立的文件夹，保持文件组织有序
  • 使用有意义的文件名，方便日后查找

八、常见考点总结

  1. C++源文件的标准扩展名是.cpp
  2. C++头文件的扩展名是.h或.hpp
  3. 文件名不应包含中文、空格和特殊字符
  4. 文件名区分大小写
  5. 包含main()函数的文件常命名为main.cpp
  6. 编译前的文件叫源文件（源代码），编译后的可执行文件在Windows下是.exe
  7. 在Dev-C++中使用Ctrl+N新建文件，Ctrl+S保存文件
