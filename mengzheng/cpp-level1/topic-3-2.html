关键字

⏱️ 预估学习时长: 45分钟 📋 验证题目: 100题 🎯 难度: 入门级

📚 知识详解

一、什么是关键字

关键字（Keyword）是C++语言预定义的保留字，具有特定的语法含义和功能。它们是C++语言的「基础词汇」，就像汉语中的「主谓宾」一样，是构成程序的基本元素。

💡 一句话理解
关键字是C++语言预定义的「保留字」，有特定含义，不能用作变量名或函数名。

二、关键字的特点

  1. 预定义性：由C++语言预先定义，用户不能创造新的关键字
  2. 特定含义：每个关键字都有固定的语法功能和含义
  3. 全部小写：C++关键字都是小写字母（如 int, not INT）
  4. 不能作为标识符：关键字不能用作变量名、函数名等标识符
// ❌ 错误：关键字不能作为标识符 int if = 10; // if 是关键字，不能用作变量名 int for = 5; // for 是关键字，不能用作变量名 // ✅ 正确：使用非关键字作为标识符 int num = 10; // num 不是关键字 int count = 5; // count 不是关键字

四、关键字 vs 标识符

🔵 关键字

  • C++预定义
  • 有特定含义
  • 全部小写
  • 不能作为标识符
  • 如：int, if, for

🟣 标识符

  • 用户自定义
  • 表示变量/函数名
  • 区分大小写
  • 不能是关键字
  • 如：num, count, sum
// 关键字 vs 标识符示例 // int 是关键字，表示整数类型 // age 是标识符（变量名） int age = 18; // if 是关键字，表示条件判断 // score 是标识符（变量名） if (score >= 60) { // ... } // for 是关键字，表示循环 // i 是标识符（循环变量名） for (int i = 0; i < 10; i++) { // ... }

六、常见错误

❌ 错误1：使用关键字作为变量名
int int = 10; // 错误！int是关键字
int return = 5; // 错误！return是关键字
❌ 错误2：关键字大小写错误
INT num = 10; // 错误！关键字必须小写
For (int i = 0; ...) // 错误！for必须小写
⚠️ 注意：区分大小写
Int、INT、iNt 都可以作为标识符，因为它们不是全小写的"int"。
但不推荐使用，容易造成混淆。

常见陷阱与技巧

  1. main不是关键字：虽然main是特殊的函数名，但它不是关键字，理论上可以用作其他标识符（但不推荐）。
  2. sizeof是关键字：虽然看起来像函数，但sizeof是关键字，不是函数。
  3. true/false是关键字：在C++中，true和false是关键字，不是宏定义。
  4. and/or/not也是关键字：C++支持and(&&)、or(||)、not(!)作为关键字，但很少使用。
