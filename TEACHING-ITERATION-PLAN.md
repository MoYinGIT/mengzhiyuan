## 二、重构标准

### 2.1 引导而不灌输

**原格式（灌输式）**:
```markdown
## 冒泡排序

冒泡排序是一种简单的排序算法。它重复地走访过要排序的数列，一次比较两个元素...

算法步骤：
1. 比较相邻的元素。如果第一个比第二个大，就交换它们两个
2. 对每一对相邻元素作同样的工作...

代码实现：
```cpp
void bubbleSort(int arr[], int n) {
    for (int i = 0; i < n-1; i++)
        for (int j = 0; j < n-i-1; j++)
            if (arr[j] > arr[j+1])
                swap(arr[j], arr[j+1]);
}
```
```

**新格式（引导式）**:
```markdown
## 冒泡排序探索

### 🎯 挑战：整理扑克牌

假设你有5张乱序的扑克牌：[3, 1, 4, 5, 2]。只能两张两张比较，怎么排成升序？

**先试试2张牌的情况**：
- [2, 1] → 怎么排？
- 如果第一个比第二个大，就______（交换位置）

<details>
<summary>💡 需要提示？</summary>

**提示1**：比较两张牌，大的放后面。
</details>

**再试试3张牌**：[3, 1, 2]
1. 比较前两张 [3, 1] → 3大，交换 → [1, 3, 2]
2. 比较 [3, 2] → 3大，交换 → [1, 2, 3]

**你发现规律了吗？** 🤔

---

### 🔍 规律总结

每轮"两两比较"后，**最大的数会像气泡一样"浮"到最后**！

这就是「冒泡」名字的由来。

---

### ✍️ 自己实现

用数组 `a[5] = {5, 4, 3, 2, 1}`，你能写出代码吗？

**填空练习**：
```cpp
for (int i = 0; i < ___; i++) {        // 需要几轮？
    for (int j = 0; j < ___; j++) {    // 每轮比较几次？
        if (a[j] ___ a[j+1])           // 什么条件交换？
            swap(a[j], a[j+1]);
    }
}
```

<details>
<summary>✅ 答案对照</summary>

```cpp
for (int i = 0; i < n-1; i++) {        // n个数需要n-1轮
    for (int j = 0; j < n-1-i; j++) {  // 每轮少比较一个（最后的已排好）
        if (a[j] > a[j+1])             // 前面的大于后面的，交换
            swap(a[j], a[j+1]);
    }
}
```
</details>

---

### 🧠 反思

1. 为什么内层循环是 `n-1-i` 而不是 `n-1`？
2. 如果数组已经有序，能不能提前结束？
3. 时间复杂度是多少？最坏情况和最好情况分别是什么？
```

### 2.3 分解而不畏惧

**复杂问题必须分解**：
```markdown
### 🧩 问题分解：奖学金排名

**原题**：输入n个学生的成绩，按总分从高到低排名。

**看起来复杂？分解它！**

**Step 1**：只考虑1个学生
- 输入：语文、数学、英语三科成绩
- 计算：总分 = 语文 + 数学 + 英语
- 输出：总分

**Step 2**：考虑2个学生
- 输入：两个学生的成绩
- 计算：各自总分
- 比较：谁的总分高？

**Step 3**：考虑n个学生
- 用什么存储n个学生的数据？→ **结构体数组**
- 怎么排序？→ **冒泡排序或sort函数**

**现在你能做了吗？** 从Step 1开始写代码。
```
