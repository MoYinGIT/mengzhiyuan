## 二、各级别AI整合教学方案

### 2.1 L1-L2：编程启蒙阶段

**学生特征**: 6-9岁，零基础，具象思维

**GESP大纲要点**:
- 顺序结构、变量、输入输出
- 分支结构（if-else）
- 循环结构（for/while基础）
- 简单数学运算

**AI整合策略**:

| 教学环节 | 传统方式 | AI增强方式 | 工具选择 |
|:---|:---|:---|:---:|
| **概念讲解** | 课本图示 | AI生成动画演示 | Claude/GPT-4 |
| **代码纠错** | 教师逐个查看 | AI即时诊断错误 | 智谱清言 |
| **作业批改** | 人工批改 | AI批量+个性化反馈 | Kimi |
| **学习报告** | 定期家长会 | AI生成周学习分析 | Kimi |

**智谱清言专用提示词（L1-L2）**:
```
【角色】你是GESP L1-L2编程启蒙老师，用生动形象的方式教小朋友编程。

【目标学生】8岁，第一次接触编程

1. 导入故事（2分钟）
```

**GESP大纲要点**:

|:---:|:---|:---|

| **CF 1033C** | L6博弈论、DP | 状态转移分析 |

**AI整合策略**:

#### 策略1：算法证明辅助

**智谱清言提示词（贪心正确性证明）**:
```
【角色】你是算法竞赛教练，擅长严格的算法正确性证明。

【算法】贪心算法
【题目类型】区间调度问题

【任务】请给出完整的贪心正确性证明：

1. **贪心策略描述**
   - 每次选择结束时间最早的区间

2. **最优子结构证明**
   - 证明：最优解包含贪心选择后的子问题的最优解

3. **贪心选择性质证明**
   - 证明：存在一个最优解包含贪心选择
   - 使用反证法或替换论证

4. **复杂度分析**
   - 时间：O(n log n)
   - 空间：O(n)

【输出要求】
- 使用数学归纳法或反证法
- 给出完整的证明步骤
- 适合GESP L5-L6学生理解
```

#### 策略2：动态规划入门引导

**智谱清言提示词（DP六步法）**:
```
【角色】你是DP教学专家，擅长将复杂问题分解为可理解的步骤。

【目标】教会GESP L6学生掌握DP基础
【例题】爬楼梯问题（经典DP入门）

【六步教学法】

Step 1: 问题抽象
- 将具体问题转化为数学模型
- n阶楼梯，每次爬1或2阶，问方法数

Step 2: 状态定义
- dp[i] = 爬到第i阶的方法数
- 明确定义的含义

Step 3: 状态转移
- dp[i] = dp[i-1] + dp[i-2]
- 解释：最后一步可能是1阶或2阶

Step 4: 边界条件
- dp[0] = 1, dp[1] = 1
- 解释基础情况

Step 5: 代码实现
- 给出记忆化搜索和递推两种写法
- 对比优缺点

Step 6: 变式拓展
- 每次可以爬1/2/3阶
- 某些台阶不能踩
- 带权重的爬楼梯

【输出要求】
- 每步配一个简单例子
- 用表格展示状态转移过程
- 最后给出3道GESP L6难度的DP练习题
```
