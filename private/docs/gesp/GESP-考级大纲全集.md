# 第一阶段：Scratch 图形化编程

---

## Scratch L1 一级

### 知识点描述（原文）

| 编号 | 知识块 | 知识点 |
|:---:|:---|:---|
| 1 | 计算机基础知识 | 认识计算机的软硬件组成，了解计算机历史，理解程序与编程语言的概念，鼠标与键盘的基本使用等 |
| 2 | 图形化编程平台 | 舞台区、角色区、模块区、脚本区、背景区，新建和保存作品，语言的选择，从本地打开软件等 |
| 3 | 编程数学 | 坐标系、坐标变化，小数、负数、随机数，顺逆时针、角度等 |
| 4 | 三大基本结构 | 顺序结构、分支结构、循环结构 |
| 5 | 事件触发 | 事件的基本概念，点击运行、当角色被点击、键盘被按下等 |
| 6 | 角色的操作 | 导入角色，角色说…，颜色，大小，显示，隐藏，移动，旋转，造型编辑/切换，声音编辑，碰到边缘反弹，图形特效等 |
| 7 | 背景的操作 | 导入背景，背景和角色的区别，背景编辑/切换，声音编辑，设置背景音乐等 |
| 8 | 侦测与控制 | 碰到鼠标指针/舞台边缘/颜色，按下键盘/鼠标，停止脚本 |

### 详细说明

#### 1. 计算机基础知识
- 计算机软硬件组成
- 计算机历史
- 程序与编程语言的概念
- 鼠标与键盘的基本使用

#### 2. 图形化编程平台
- **舞台区**: 程序运行显示区域
- **角色区**: 管理和选择角色
- **模块区**: 积木分类（运动、外观、声音、事件、控制、侦测、运算、变量）
- **脚本区**: 拖拽积木编写程序
- **背景区**: 管理和切换背景
- 新建和保存作品
- 语言选择
- 从本地打开软件

#### 3. 编程数学
- 坐标系: x轴（左右）、y轴（上下），中心点为(0,0)
- 坐标变化
- 小数、负数
- 随机数
- 顺时针、逆时针
- 角度（0-360度）

#### 4. 三大基本结构
- **顺序结构**: 从上到下依次执行
- **分支结构**: 如果...那么...否则...
- **循环结构**: 重复执行、重复执行直到...

#### 5. 事件触发
- 事件的基本概念
- 点击绿旗运行
- 当角色被点击
- 键盘被按下

#### 6. 角色的操作
- 导入角色
- 角色说话（"说..."）
- 颜色设置
- 大小设置
- 显示/隐藏
- 移动
- 旋转
- 造型编辑/切换
- 声音编辑
- 碰到边缘反弹
- 图形特效

#### 7. 背景的操作
- 导入背景
- 背景和角色的区别
- 背景编辑/切换
- 声音编辑
- 设置背景音乐

#### 8. 侦测与控制
- 碰到鼠标指针
- 碰到舞台边缘
- 碰到颜色
- 按下键盘
- 按下鼠标
- 停止脚本

### 详细说明

#### 1. 计数原理
| 原理 | 说明 | 公式 |
|:---|:---|:---|
| 加法原理 | 分类计数，要么...要么... | 总方法数 = 各类方法数之和 |
| 乘法原理 | 分步计数，先...再... | 总方法数 = 各步方法数之积 |

#### 2. 排列与组合
| 概念 | 定义 | 公式 |
|:---|:---|:---|
| 排列 | 从n个元素中取m个，考虑顺序 | A(n,m) = n!/(n-m)! |
| 组合 | 从n个元素中取m个，不考虑顺序 | C(n,m) = n!/(m!(n-m)!) |

#### 3. 杨辉三角
- 杨辉三角第n行对应二项式系数 C(n,0)~C(n,n)
- 递推关系：C(n,k) = C(n-1,k-1) + C(n-1,k)
- 应用：组合数计算、多项式展开

```
杨辉三角示例：
第0行:      1
第1行:     1 1
第2行:    1 2 1
第3行:   1 3 3 1
第4行:  1 4 6 4 1
```

#### 4. 倍增法
- 倍增思想：将问题规模按2的幂次分解
- 应用：快速幂、ST表（区间最值）、树上倍增（LCA）
- 时间复杂度优化：O(n) → O(log n)

#### 5. 代数与平面几何
| 几何图形 | 面积公式 |
|:---|:---|
| 三角形 | S = 底 × 高 / 2 |
| 圆形 | S = π × r² |
| 长方形 | S = 长 × 宽 |

- 一元一次方程
- 二元一次方程

#### 6. 图论算法及综合应用
**最小生成树**:
| 算法 | 时间复杂度 | 特点 | 适用场景 |
|:---|:---:|:---|:---|

| Prim | O(V²) / O(E log V) | 从点出发，优先队列优化 | 稠密图 |

**最短路径**:
| 算法 | 时间复杂度 | 特点 | 适用场景 |

# 学习路径建议

## 三阶段递进模型

### 第一阶段：图形化编程启蒙（Scratch 1-4级）
| 级别 | 核心能力 | 关键知识点 |
|:---:|:---|:---|
| L1 | 基础操作 | 界面认知、三大结构 |
| L2 | 流程思维 | 流程图、变量、克隆、广播 |
| L3 | 算法入门 | 列表、枚举、模拟 |
| L4 | 函数封装 | 函数、排序算法 |

### 第二阶段：代码编程入门（Python/C++ 1-2级）
| 级别 | 核心能力 | 关键知识点 |
|:---:|:---|:---|
| L1 | 语法基础 | 数据类型、输入输出、三大结构 |
| L2 | 程序思维 | 流程图、嵌套结构、数学函数 |

**Python 特色**：Turtle 绘图入门  
**C++ 特色**：基础语法扎实训练

### 第三阶段：算法竞赛进阶（C++ 3-8级）
| 级别 | 核心能力 | 关键知识点 |
|:---:|:---|:---|
| L3 | 底层基础 | 进制编码、位运算、一维数组/列表 |
| L4 | 数据结构 | 指针/结构体、二维数组、递推排序 |
| L5 | 算法设计 | 数论、链表、递归、分治、贪心、二分 |
| L6 | 图论DP | 树、DFS/BFS、动态规划、面向对象 |
| L7 | 进阶算法 | 复杂DP、图论入门、哈希表 |
| L8 | 综合应用 | 排列组合、MST、最短路径、算法优化 |

**进阶路径**：L3 → L4 → L5 → L6 → L7 → L8（逐级递进，不可跳跃）

## 推荐学习主线

**Scratch L1~L4 → Python L1~L2 → C++ L3~L8**

或

**Scratch L1~L4 → C++ L1~L8**（直接以C++为主线）
