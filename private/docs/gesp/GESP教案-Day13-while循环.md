### 第二阶段：新知讲授（35分钟）

**08:40-09:15**

#### 2.1 while循环语法（10分钟）

**板书**：
```cpp
while (循环条件) {
    // 循环体：要重复执行的代码
    // 循环变量更新（重要！）
}
```

**口诀**："外条件，内执行，别忘了更新变量"

**类比**：跑步机
- 循环条件 = 跑步机开关（开着就跑）
- 循环体 = 跑步的动作
- 循环变量 = 计时器（决定什么时候停）

#### 2.2 第一个示例：输出1-5（10分钟）

**老师演示**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int i = 1;          // 1. 初始化循环变量
    while (i <= 5) {    // 2. 循环条件
        cout << i << " ";  // 3. 循环体
        i = i + 1;      // 4. 更新循环变量（重要！）
    }
    return 0;
}
```

**手工推演**（用白板画出每一步）：
```
i=1, i<=5? 是 → 输出1, i变成2
i=2, i<=5? 是 → 输出2, i变成3
i=3, i<=5? 是 → 输出3, i变成4
i=4, i<=5? 是 → 输出4, i变成5
i=5, i<=5? 是 → 输出5, i变成6
i=6, i<=5? 否 → 结束
```

**输出结果**：`1 2 3 4 5`

**强调**：如果不写`i = i + 1`，会怎样？
- 演示死循环（提前准备，用简单程序展示，及时终止）
- 告诉学生：死循环要按Ctrl+C或关闭程序

#### 2.3 第二个示例：累加1+2+3+...+100（15分钟）

**引入**：
高斯的故事：1+2+3+...+100 = ?

**老师演示**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int i = 1;          // 循环变量
    int sum = 0;        // 累加器（关键！）
    
    while (i <= 100) {
        sum = sum + i;  // 累加
        i = i + 1;      // 更新
    }
    
    cout << sum << endl;
    return 0;
}
```

**实物演示**（5个盒子）：
- 盒子`i`：放当前数字（1→2→3→4→5）
- 盒子`sum`：放累加和（0→1→3→6→10→15）

**手工推演**（前5步）：
```
初始: i=1, sum=0
第1轮: sum=0+1=1, i=2
第2轮: sum=1+2=3, i=3
第3轮: sum=3+3=6, i=4
第4轮: sum=6+4=10, i=5
第5轮: sum=10+5=15, i=6
...
```

**关键要点**：
1. 累加器`sum`初始化为0
2. 累加器在循环体内更新
3. 循环变量和累加器都要更新

#### 3.2 练习2：独立编写（15分钟）

**题目**：计算1×2×3×...×10（阶乘）

**提示**：
- 循环变量`i`从1到10
- 累乘器`product`，初始化为1（为什么不能是0？）
- 循环体内：`product = product * i`

**参考答案**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int i = 1;
    int product = 1;        // 累乘器初始化为1
    
    while (i <= 10) {
        product = product * i;
        i = i + 1;
    }
    
    cout << product << endl;
    return 0;
}
```

**讲解**：
- 为什么累加器初始化为0，累乘器初始化为1？
- 数学原理：0是加法的单位元，1是乘法的单位元

```cpp

while (i <= n) {
```
```cpp
    sum = sum + i;
```
