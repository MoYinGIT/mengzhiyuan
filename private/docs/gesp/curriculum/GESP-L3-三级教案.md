## 三、详细教学设计

### 第1课时：计算机数据编码（上）- 原码、反码、补码

#### 教学目标
- 理解计算机如何表示正负数
- 掌握原码、反码、补码的定义
- 理解为什么计算机使用补码

#### 教学内容

**1. 有符号数的表示（15分钟）**
```
计算机中只有0和1，如何表示负数？

最高位作为符号位：
- 0 表示正数
- 1 表示负数

以8位二进制为例：
+5 = 00000101
-5 = 10000101（原码表示）
```

**2. 原码、反码、补码（30分钟）**

| 编码方式 | 正数 | 负数 |
|:---|:---|:---|
| 原码 | 符号位+绝对值 | 符号位+绝对值 |
| 反码 | 与原码相同 | 符号位不变，其他位取反 |
| 补码 | 与原码相同 | 反码+1 |

**示例（以-5为例）**：
```
原码：10000101
反码：11111010（符号位不变，其余取反）
补码：11111011（反码+1）
```

**为什么用补码？**
```
1. 统一了0的表示
   +0 = 00000000
   -0 = 10000000（原码）→ 00000000（补码）

2. 减法变加法，简化硬件设计
   5 - 3 = 5 + (-3)
   
3. 多表示一个最小负数
   8位补码范围：-128 ~ 127
```

#### 示例代码

**【例1】观察负数存储**
```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 5;
    int b = -5;
    
    // 查看内存中的表示（简化版）
    unsigned int *pa = (unsigned int*)&a;
    unsigned int *pb = (unsigned int*)&b;
    
    cout << "5的十六进制表示：" << hex << *pa << endl;
    cout << "-5的十六进制表示：" << hex << *pb << endl;
    
    // 补码取反加1得到原码的验证
    cout << "~(-5) + 1 = " << dec << (~b + 1) << endl;
    
    return 0;
}
```

### 第2课时：进制转换

#### 教学目标
- 掌握各进制之间的转换方法
- 理解进制转换的原理
- 熟练进行编程实现

#### 教学内容

**1. 进制基础（15分钟）**
```
十进制：0-9，逢10进1
二进制：0-1，逢2进1
八进制：0-7，逢8进1
十六进制：0-9,A-F，逢16进1

进制表示法：
- 二进制：0b1010 或 (1010)₂
- 八进制：012 或 (12)₈
- 十进制：10 或 (10)₁₀
- 十六进制：0xA 或 (A)₁₆
```

**2. 进制转换方法（30分钟）**

**十进制转其他进制**：除基取余，逆序排列
```
十进制 13 转二进制：
13 ÷ 2 = 6 余 1
6 ÷ 2 = 3 余 0
3 ÷ 2 = 1 余 1
1 ÷ 2 = 0 余 1

结果：1101（逆序读取余数）
```

**其他进制转十进制**：按权展开
```
二进制 1101 转十进制：
1×2³ + 1×2² + 0×2¹ + 1×2⁰ = 8 + 4 + 0 + 1 = 13
```

**二进制与八进制/十六进制的快速转换**：
```
二进制 → 八进制：三位一组
  1 101 110 → 156₈

二进制 → 十六进制：四位一组
  1101 0110 → D6₁₆
```

#### 示例代码

**【例2】十进制转二进制**
```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    
    int binary[32], count = 0;
    
    if (n == 0) {
        cout << 0 << endl;
        return 0;
    }
    
    while (n > 0) {
        binary[count++] = n % 2;
        n /= 2;
    }
    
    for (int i = count - 1; i >= 0; i--) {
        cout << binary[i];
    }
    cout << endl;
    
    return 0;
}
```

**【例3】任意进制转换**
```cpp
#include <iostream>
#include <string>
using namespace std;

// 十进制转任意进制（2-16）
void decToBase(int n, int base) {
    char digits[] = "0123456789ABCDEF";
    char result[32];
    int count = 0;
    
    while (n > 0) {
        result[count++] = digits[n % base];
        n /= base;
    }
    
    for (int i = count - 1; i >= 0; i--) {
        cout << result[i];
    }
}

int main() {
    int n;
    cin >> n;
    
    cout << "二进制：";
    decToBase(n, 2);
    cout << endl;
    
    cout << "八进制：";
    decToBase(n, 8);
    cout << endl;
    
    cout << "十六进制：";
    decToBase(n, 16);
    cout << endl;
    
    return 0;
}
```

### 第3-4课时：位运算

#### 教学目标
- 掌握六种位运算符的使用
- 理解位运算的应用场景
- 能够用位运算优化程序

#### 教学内容

**1. 位运算符详解（40分钟）**

| 运算符 | 名称 | 功能 | 示例 |
|:---:|:---:|:---|:---|
| & | 按位与 | 两位都为1，结果为1 | 5&3 = 1 |
| \| | 按位或 | 有一位为1，结果为1 | 5\|3 = 7 |
| ^ | 按位异或 | 两位不同，结果为1 | 5^3 = 6 |
| ~ | 按位取反 | 0变1，1变0 | ~5 = -6 |
| << | 左移 | 所有位左移，补0 | 5<<1 = 10 |
| >> | 右移 | 所有位右移 | 5>>1 = 2 |

**位运算图解**：
```
    5 = 0101
    3 = 0011
    -----------
&   1 = 0001  （都为1才为1）
|    7 = 0111  （有1就为1）
^    6 = 0110  （不同才为1）
```

**2. 位运算应用（30分钟）**

**应用1：判断奇偶**
```cpp
// 传统方法
if (n % 2 == 1)  // 奇数

// 位运算方法（更快）
if (n & 1)  // 看最后一位，1为奇数，0为偶数
```

**应用2：交换两个数**
```cpp
// 不用临时变量
a = a ^ b;
b = a ^ b;  // b = (a^b)^b = a
a = a ^ b;  // a = (a^b)^a = b
```

**应用3：快速乘除2**
```cpp
n << 1  // 相当于 n * 2
n >> 1  // 相当于 n / 2
n << k  // 相当于 n * 2^k
```

**应用4：判断一个数是否是2的幂**
```cpp
if ((n & (n - 1)) == 0 && n != 0) {
    // n是2的幂
}
// 原理：2的幂二进制只有一个1，如 8=1000, 7=0111, 8&7=0
```

#### 示例代码

**【例4】位运算综合应用**
```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 29;  // 11101
    int b = 15;  // 01111
    
    cout << "a = " << a << ", b = " << b << endl;
    cout << "a & b = " << (a & b) << endl;   // 13 (01101)
    cout << "a | b = " << (a | b) << endl;   // 31 (11111)
    cout << "a ^ b = " << (a ^ b) << endl;   // 18 (10010)
    cout << "~a = " << (~a) << endl;         // -30
    cout << "a << 2 = " << (a << 2) << endl; // 116 (1110100)
    cout << "a >> 2 = " << (a >> 2) << endl; // 7 (00111)
    
    // 统计二进制中1的个数
    int n = 29, count = 0;
    while (n) {
        count += n & 1;
        n >>= 1;
    }
    cout << "29的二进制中有" << count << "个1" << endl;
    
    return 0;
}
```

| 方式 | 优点 | 缺点 |
|:---|:---|:---|

**2. 伪代码规范（20分钟）**
```
伪代码约定：
- 用缩进表示代码块
- ← 表示赋值
- = 表示比较
- // 表示注释

示例：求最大公约数
函数 gcd(a, b):
    当 b ≠ 0 时:
        temp ← a 除以 b 的余数
        a ← b
        b ← temp
    返回 a
```

### 第6-7课时：一维数组

#### 教学目标
- 掌握一维数组的定义和初始化
- 熟练进行数组遍历和操作
- 能够用数组解决实际问题

#### 教学内容

**1. 数组概念（15分钟）**
```
数组：相同类型数据的集合，连续存储在内存中

特点：
- 元素类型相同
- 连续存储
- 通过下标访问（从0开始）

内存示意：
arr[0] arr[1] arr[2] arr[3] arr[4]
  10     20     30     40     50
  
地址连续，每个元素占相同字节
```

**2. 数组操作（35分钟）**

**定义和初始化**：
```cpp
int arr[5];                    // 定义，元素值不确定
int arr[5] = {1, 2, 3, 4, 5}; // 完全初始化
int arr[5] = {1, 2};          // 部分初始化，其余为0
int arr[] = {1, 2, 3};        // 自动计算大小
int arr[5] = {0};             // 全部初始化为0
```

**基本操作**：
```cpp
// 输入
for (int i = 0; i < n; i++) {
    cin >> arr[i];
}

// 输出
for (int i = 0; i < n; i++) {
    cout << arr[i] << " ";
}

// 求和
int sum = 0;
for (int i = 0; i < n; i++) {
    sum += arr[i];
}

// 找最大值
int maxVal = arr[0];
for (int i = 1; i < n; i++) {
    if (arr[i] > maxVal) maxVal = arr[i];
}

// 逆序
for (int i = 0; i < n / 2; i++) {
    swap(arr[i], arr[n-1-i]);
}
```

#### 示例代码

**【例5】数组基础操作**
```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int arr[100], n;
    cin >> n;
    
    // 输入
    for (int i = 0; i < n; i++) {
        cin >> arr[i];
    }
    
    // 找最大值和最小值及其位置
    int maxVal = arr[0], maxPos = 0;
    int minVal = arr[0], minPos = 0;
    
    for (int i = 1; i < n; i++) {
        if (arr[i] > maxVal) {
            maxVal = arr[i];
            maxPos = i;
        }
        if (arr[i] < minVal) {
            minVal = arr[i];
            minPos = i;
        }
    }
    
    cout << "最大值：" << maxVal << "，位置：" << maxPos << endl;
    cout << "最小值：" << minVal << "，位置：" << minPos << endl;
    
    // 排序
    sort(arr, arr + n);
    
    cout << "排序后：";
    for (int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
    
    return 0;
}
```

### 第8-9课时：枚举法

#### 教学目标
- 理解枚举法的思想
- 掌握枚举法的优化技巧
- 能够应用枚举法解决问题

#### 教学内容

**1. 枚举法思想（20分钟）**
```
枚举法（穷举法）：
- 思想：列举所有可能的情况，逐一验证
- 适用：问题规模较小，解空间有限
- 关键：确定枚举范围和判断条件

优化原则：
1. 减少枚举范围
2. 提前剪枝
3. 利用数学性质
```

**2. 枚举法应用（30分钟）**

**百钱买百鸡（经典问题）**：
```cpp
// 公鸡5钱，母鸡3钱，小鸡1钱3只
// 100钱买100只鸡

for (int x = 0; x <= 20; x++) {      // 公鸡
    for (int y = 0; y <= 33; y++) {  // 母鸡
        int z = 100 - x - y;         // 小鸡
        if (z % 3 == 0 && 5*x + 3*y + z/3 == 100) {
            cout << x << " " << y << " " << z << endl;
        }
    }
}
```

**质数判断（枚举优化）**：
```cpp
// 优化前：枚举2到n-1
// 优化后：枚举2到√n

bool isPrime(int n) {
    if (n <= 1) return false;
    for (int i = 2; i * i <= n; i++) {
        if (n % i == 0) return false;
    }
    return true;
}
```

#### 示例代码

**【例6】完美立方**
```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    
    // 找所有a³ = b³ + c³ + d³，其中a,b,c,d <= n
    for (int a = 2; a <= n; a++) {
        for (int b = 2; b < a; b++) {
            for (int c = b; c < a; c++) {
                for (int d = c; d < a; d++) {
                    if (a*a*a == b*b*b + c*c*c + d*d*d) {
                        cout << "Cube = " << a << ", Triple = (" 
                             << b << "," << c << "," << d << ")" << endl;
                    }
                }
            }
        }
    }
    
    return 0;
}
```

**1. 模拟法思想（20分钟）**
```
模拟法：
- 思想：按照问题的实际过程或规则，逐步执行
- 适用：过程明确、步骤清晰的问题
- 关键：准确理解规则，正确实现每一步

常见应用：
- 游戏模拟
- 物理过程模拟
- 时钟、日期计算
- 约瑟夫问题
```

**2. 模拟法应用（30分钟）**

**约瑟夫问题**：
```cpp
// n个人围成一圈，从第1个人开始报数，报到m的人出圈
// 求最后剩下的人的编号

int josephus(int n, int m) {
    bool alive[100] = {false};  // 标记是否存活
    for (int i = 0; i < n; i++) alive[i] = true;
    
    int count = n;  // 剩余人数
    int pos = 0;    // 当前位置
    int num = 0;    // 报数
    
    while (count > 1) {
        if (alive[pos]) {
            num++;
            if (num == m) {
                alive[pos] = false;
                count--;
                num = 0;
            }
        }
        pos = (pos + 1) % n;
    }
    
    for (int i = 0; i < n; i++) {
        if (alive[i]) return i + 1;
    }
    return -1;
}
```

**【例8】字符串展开**
```cpp
#include <iostream>
#include <cctype>
using namespace std;

int main() {
    string s;
    cin >> s;
    
    string result;
    for (int i = 0; i < s.length(); i++) {
        if (isalpha(s[i]) && i + 2 < s.length() && s[i+1] == '-') {
            char start = s[i];
            char end = s[i+2];
            
            if (start < end) {
                for (char c = start; c <= end; c++) {
                    result += c;
                }
            } else {
                for (char c = start; c >= end; c--) {
                    result += c;
                }
            }
            i += 2;
        } else {
            result += s[i];
        }
    }
    
    cout << result << endl;
    
    return 0;
}
```
