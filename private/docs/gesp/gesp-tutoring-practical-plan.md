## 三、智谱清言GESP辅导专用提示词

### 场景1：L3循环专题辅导（基于P2669）

**提示词模板**:
```
【角色】你是GESP C++三级辅导专家，擅长用严谨的推导讲解循环结构。

【背景】学生正在准备GESP三级考试，需要掌握循环嵌套和数学规律题。

【今日真题】P2669 [NOIP2015 普及组] 金币
题目：国王第一天给1个金币，之后两天每天给2个，再之后三天每天给3个...
给定k天，求总共收到多少金币。

【任务】请系统讲解这道题的完整解法。

【讲解要求】

1. **问题分析**
   - 将题目转化为数学模型
   - 明确：第n轮给n个金币，持续n天
   - 分析：如何确定k天包含多少完整轮，以及剩余天数

2. **算法设计**
   - 完整轮的处理：1×1 + 2×2 + 3×3 + ... + n×n
   - 剩余天数的处理：剩余天数 × (n+1)
   - 循环终止条件的严谨推导

3. **代码实现**（C++）
   ```cpp
   #include <iostream>
   using namespace std;
   
   int main() {
       int k;
       cin >> k;
       int sum = 0;
       int n = 1;  // 当前轮数
       while (k > 0) {
           if (k >= n) {
               sum += n * n;  // 完整一轮
               k -= n;
           } else {
               sum += k * n;  // 不足一轮
               k = 0;
           }
           n++;
       }
       cout << sum << endl;
       return 0;
   }
   ```

4. **复杂度分析**
   - 时间复杂度：O(√k)，因为1+2+...+n ≈ k，n ≈ √k
   - 空间复杂度：O(1)

5. **变式练习**（给出3道类似题目）
   - 变式1：如果第n轮给n²个金币，每天给n²个
   - 变式2：如果轮数是斐波那契数列
   - 变式3：反向计算：给定金币数，求最少天数

6. **常见错误警示**
   - 混淆轮数和天数
   - 循环终止条件写错导致死循环
   - 忘记处理最后不足一轮的情况

【输出要求】
- 使用学术化但易懂的语言
- 关键步骤给出数学推导
- 代码符合GESP规范（不用高级特性）
- 最后给出GESP三级模拟题链接
```

### 场景2：L4字符串专题辅导（基于P1055）

**提示词模板**:
```
【角色】你是GESP C++四级辅导专家，擅长字符串处理和细节调试。

【背景】学生在准备GESP四级考试，字符串处理是重点也是难点。

【今日真题】P1055 ISBN号码
题目：验证ISBN号码正确性，根据前9位计算识别码，与第10位对比。
计算规则：第i位数字 × i 的累加和，模11。

【任务】请详细讲解字符串处理技巧和常见陷阱。

【讲解要求】

1. **字符与数字转换**
   - char → int：c - '0'
   - 原理：ASCII码中'0'~'9'是连续的
   - 边界检查：确保字符在'0'~'9'范围内

2. **ISBN计算详解**
   - 权重设计：第1位×1，第2位×2，...，第9位×9
   - 累加和模11得到识别码
   - 特殊情况：识别码为10时用'X'表示

3. **代码实现**（含注释）
   ```cpp
   #include <iostream>
   using namespace std;
   
   int main() {
       char isbn[14];  // 预留'\0'位置
       cin >> isbn;
       
       int sum = 0;
       int weight = 1;
       for (int i = 0; i < 11; i++) {  // 前11位包含'-'符号
           if (isbn[i] != '-') {
               sum += (isbn[i] - '0') * weight;
               weight++;
           }
       }
       
       int check = sum % 11;
       char checkChar;
       if (check == 10) checkChar = 'X';
       else checkChar = check + '0';
       
       if (checkChar == isbn[12]) {  // 第13位是识别码（索引12）
           cout << "Right" << endl;
       } else {
           isbn[12] = checkChar;
           cout << isbn << endl;
       }
       return 0;
   }
   ```

4. **常见错误分析**（重点）
   - ❌ 错误1：字符数组大小设为13，忘记'\0'需要1字节
   - ❌ 错误2：未处理识别码为'X'的情况
   - ❌ 错误3：'-'符号的位置计算错误
   - ❌ 错误4：权重累加时机错误

5. **调试技巧**
   - 如何添加调试输出检查中间结果
   - 使用测试用例验证：
     * 输入：0-670-82162-4 → Right
     * 输入：0-670-82162-0 → 0-670-82162-4

6. **GESP四级相关考点**
   - 字符串长度计算strlen
   - 子串提取
   - 字符串比较strcmp
   - 字符分类函数（isdigit, isalpha等）

【输出要求】
- 详细标注每行代码的作用
- 强调易错点（用⚠️标记）
- 提供3道GESP四级字符串练习题
```

### 场景3：L4排序专题辅导（基于P1093）

**提示词模板**:
```
【角色】你是GESP C++四级辅导专家，擅长结构体和排序算法教学。

【背景】学生正在学习结构体和自定义排序，这是GESP四级的重点。

【今日真题】P1093 [NOIP2007 普及组] 奖学金
题目：根据语文、数学、英语成绩，选出总分最高的前5名学生。
排序规则：总分降序 → 语文降序 → 学号升序

【任务】请系统讲解结构体排序的完整方法。

【讲解要求】

1. **结构体设计**
   ```cpp
   struct Student {
       int id;        // 学号
       int chinese;   // 语文
       int math;      // 数学
       int english;   // 英语
       int sum;       // 总分（冗余设计，避免重复计算）
   };
   ```
   - 为什么要设计sum字段？（空间换时间）
   - 结构体的内存布局（可选讲解）

2. **多关键字排序原理**
   - 第一关键字：总分（降序）
   - 第二关键字：语文（降序）
   - 第三关键字：学号（升序）
   - 比较逻辑的嵌套设计

3. **cmp函数设计**（核心）
   ```cpp
   bool cmp(const Student& a, const Student& b) {
       // 第一关键字：总分降序
       if (a.sum != b.sum) 
           return a.sum > b.sum;
       
       // 第二关键字：语文降序
       if (a.chinese != b.chinese)
           return a.chinese > b.chinese;
       
       // 第三关键字：学号升序
       return a.id < b.id;
   }
   ```

4. **完整代码**（带详细注释）
   ```cpp
   #include <iostream>
   #include <algorithm>
   using namespace std;
   
   struct Student {
       int id, chinese, math, english, sum;
   };
   
   bool cmp(const Student& a, const Student& b) {
       if (a.sum != b.sum) return a.sum > b.sum;
       if (a.chinese != b.chinese) return a.chinese > b.chinese;
       return a.id < b.id;
   }
   
   int main() {
       int n;
       cin >> n;
       Student s[305];  // 根据题目范围开数组
       
       for (int i = 0; i < n; i++) {
           s[i].id = i + 1;
           cin >> s[i].chinese >> s[i].math >> s[i].english;
           s[i].sum = s[i].chinese + s[i].math + s[i].english;
       }
       
       sort(s, s + n, cmp);
       
       for (int i = 0; i < 5; i++) {
           cout << s[i].id << " " << s[i].sum << endl;
       }
       return 0;
   }
   ```

5. **常见错误警示**（⚠️重点）
   - 降序用 ">"，升序用 "<"，容易混淆
   - 忘记处理总分相同的情况
   - 数组越界（学号从1开始，数组从0开始）
   - 比较函数中写成 assignment (=) 而不是 equality (==)

6. **扩展：稳定排序**
   - stable_sort的使用场景
   - GESP考试中是否必要？

7. **练习题**（3道变式）
   - 变式1：选出每班第一名（分组排序）
   - 变式2：成绩相同则按数学排序
   - 变式3：使用冒泡排序手动实现（不调用sort）

【输出要求】
- 代码逐行注释
- 强调cmp函数的设计逻辑
- 给出稳定排序的对比说明
```
