## 1. 认知偏差与决策

### 1.1 常见认知偏差 (LessWrong)

| 偏差名称 | 定义 | 应对策略 |
|:---|:---|:---|
| **确认偏误** | 倾向于寻找支持自己观点的证据 | 主动寻找反面证据 |
| **可得性启发** | 根据容易回忆的例子判断频率 | 使用统计数据而非印象 |
| **锚定效应** | 过度依赖最先获得的信息 | 独立评估，重新估算 |
| **沉没成本谬误** | 因已投入而继续不理想的选择 | 关注边际收益而非已投入 |
| **后见之明** | 事后认为事件可预测 | 记录预测，检验准确性 |

### 1.2 理性思维工具

**贝叶斯思维**:
- 将信念视为概率而非二元真假
- 根据新证据更新信念
- 公式: P(H|E) = P(E|H) × P(H) / P(E)

**预测校准**:
- 区分不同置信度的预测
- 80% 置信度的预测应该有 80% 准确率
- 记录并评估预测以改进校准

## 2. 学习与记忆

### 2.1 间隔重复 (Gwern.net)

**原理**: 艾宾浩斯遗忘曲线
- 记忆随时间衰减
- 在即将遗忘时复习最有效
- 最佳间隔: 1天 → 3天 → 7天 → 14天 → 30天

**工具推荐**:
- Anki (免费开源)
- RemNote
- SuperMemo

**卡片设计原则**:
1. 一次只测试一个概念
2. 使用主动回忆而非再认
3. 包含上下文信息
4. 定期清理低效卡片

### 2.2 主动学习策略

**测试效应**:
- 主动回忆比重读更有效
- 自我测试是强大的学习工具
- 错误在学习过程中是有价值的

**交错学习 (Interleaving)**:
- 不同主题交替练习
- 提高知识迁移能力
- 增强辨别能力

**深度学习**:
- 与已有知识建立连接
- 教授他人 (费曼技巧)
- 应用到实际问题

## 3. 动机与意志力

### 3.1 拖延症 (Wait But Why)

**拖延者的心理**:
- 即时满足 vs 长期目标
- 恐惧失败导致逃避
- 任务看起来过于庞大

**应对策略**:
1. **分解任务**: 将大任务分解为小步骤
2. **番茄工作法**: 25分钟专注 + 5分钟休息
3. **环境设计**: 减少干扰，增加启动便利
4. **自我同情**: 接受不完美，关注进步

### 3.2 心流状态

**心流的条件** (Flow State):
- 挑战与技能平衡
- 明确的目标
- 即时反馈
- 深度专注

**进入心流的方法**:
- 消除干扰
- 设定明确目标
- 选择适当难度的任务
- 创造仪式感

## 5. 人格与发展

### 5.1 成长型思维 (Growth Mindset)

**固定型 vs 成长型**:
- 固定型: 能力是天生的、固定的
- 成长型: 能力可以通过努力发展

**培养成长型思维**:
- 表扬努力而非天赋
- 将失败视为学习机会
- 关注过程而非结果

### 5.2 人格特质

**大五人格**:
1. **开放性**: 好奇心、创造力
2. **尽责性**: 自律、组织能力
3. **外向性**: 社交性、活力
4. **宜人性**: 合作、同理心
5. **神经质**: 情绪稳定性

**应用**:
- 了解自己的优势和劣势
- 选择适合的职业和环境
- 改善人际关系

## 核心洞察

> **心理学不是读心术，而是理解人类思维的系统方法。**

### 三大原则

1. **大脑有局限**: 工作记忆有限，注意力有限，意志力有限
2. **环境塑造行为**: 改变环境比改变意志更容易
3. **成长是可能的**: 大脑具有可塑性，能力可以通过练习提升

### 学习建议

1. 从了解认知偏差开始，提高元认知能力
2. 使用间隔重复系统学习核心概念
3. 将理论应用到实际生活中
4. 保持怀疑和开放的心态
