## 1. 概率论与统计

### 1.1 贝叶斯思维

**核心公式**:
```
P(H|E) = P(E|H) × P(H) / P(E)
```

**应用**:
- 更新信念
- 预测和推断
- 决策分析

**与频率派的区别**:
- 贝叶斯: 概率是信念程度
- 频率派: 概率是长期频率

### 1.2 相关不等于因果

**常见陷阱**:
- 混淆相关性与因果性
- 遗漏变量偏差
- 反向因果

**建立因果**:
- 随机对照试验 (RCT)
- 自然实验
- 工具变量
- 断点回归

### 1.3 回归均值

**现象**:
- 极端值倾向于向平均值回归
- 被误认为是干预效果

**例子**:
- 体育中的"二年级低谷"
- 医疗中的"安慰剂效应"
- 教育干预评估

### 1.4 顺序统计量

**概念**:
- 最大值、最小值的分布
- 选择效应

**应用**:
- 极端事件分析
- 选拔与评估
- 投资组合优化

**计算**:
