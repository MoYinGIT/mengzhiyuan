## 引言

**《易经·系辞》**云：*「易有太极，是生两仪，两仪生四象，四象生八卦。」*

六十四卦不仅是古代哲学的精髓，更蕴含着深刻的**算法思想**。阴阳二元对应计算机的0和1，卦变之道对应状态转移，象数之理对应数据结构。

本图谱将六十四卦与算法概念一一映射，助你在编程中领悟东方智慧。

### 算法详解：复卦栈结构

```
复：亨。出入无疾，朋来无咎。

算法诠释：
- 出入无疾：push/pop操作O(1)
- 朋来无咎：栈帧之间无干扰
- 反复其道：递归调用，层层深入，原路返回
```

**递归 = 复卦之道**
```cpp
void dfs(int u) {
    if (visited[u]) return;  // 复：回归条件
    visited[u] = true;
    for (int v : adj[u]) {
        dfs(v);  // 反复其道
    }
}
```

### 算法详解：豫卦记忆化

```
豫：利建侯行师。

算法诠释：
- 建侯：建立状态缓存（dp数组）
- 行师：利用缓存进行计算

核心思想：
无记忆 → 重复计算 → 指数复杂度
有记忆 → 缓存复用 → 多项式复杂度
```

**记忆化搜索示例**
```cpp
// Fibonacci with memoization
map<int, ll> memo;

ll fib(int n) {
    if (n <= 1) return n;
    if (memo.count(n)) return memo[n];  // 豫：利用缓存
    return memo[n] = fib(n-1) + fib(n-2);  // 建侯：建立缓存
}
```

### 算法详解：观卦排序贪心

```
观：盥而不荐，有孚颙若。

算法诠释：
- 盥：清洗数据（排序）
- 不荐：不盲目选择
- 有孚：有信念地按序选择
- 颙若：仰望全局，确保正确
```

**活动选择问题**
```cpp
// 按结束时间排序后贪心
sort(activities.begin(), activities.end(), 
    [](auto& a, auto& b) { return a.end < b.end; });  // 观：排序

int cnt = 1, last = activities[0].end;
for (int i = 1; i < n; i++) {
    if (activities[i].start >= last) {  // 有孚：有序选择
        cnt++;
        last = activities[i].end;
    }
}
```

### 算法详解：既济卦最短路

```
既济：亨，小利贞，初吉终乱。

算法诠释（Dijkstra）：
- 亨：算法正确，路径通达
- 小利贞：贪心选择局部最优
- 初吉：起点距离为0，吉祥如意
- 终乱：需处理负权边（用SPFA）
```

**Dijkstra算法**
```cpp
void dijkstra(int s) {
    memset(dist, 0x3f, sizeof(dist));
    dist[s] = 0;  // 初吉
    priority_queue<pii, vector<pii>, greater<pii>> pq;
    pq.push({0, s});
    
    while (!pq.empty()) {
        auto [d, u] = pq.top(); pq.pop();
        if (d > dist[u]) continue;  // 小利贞：已最优
        for (auto [v, w] : adj[u]) {
            if (dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;  // 亨：更新最短距离
                pq.push({dist[v], v});
            }
        }
    }
}
```

**归并排序**
```cpp
void mergeSort(int l, int r) {
    if (l >= r) return;
    int mid = (l + r) >> 1;
    mergeSort(l, mid);      // 左足
    mergeSort(mid + 1, r);  // 右足
    // 合并入鼎
    int i = l, j = mid + 1, k = 0;
    while (i <= mid && j <= r) {
        if (a[i] <= a[j]) temp[k++] = a[i++];
        else temp[k++] = a[j++];
    }
    while (i <= mid) temp[k++] = a[i++];
    while (j <= r) temp[k++] = a[j++];
    for (int p = 0; p < k; p++) a[l + p] = temp[p];
}
```

### 算法详解：剥卦二分查找

```
剥：不利有攸往。

算法诠释：
- 剥：层层剥离
- 不利有攸往：不需要遍历全部
- 核心：每次剥去一半，快速定位

时间复杂度：O(log n) —— 剥去之快
```

**二分查找**
```cpp
int binarySearch(vector<int>& a, int target) {
    int l = 0, r = a.size() - 1;
    while (l <= r) {
        int mid = (l + r) >> 1;  // 剥：取中点
        if (a[mid] == target) return mid;
        else if (a[mid] < target) l = mid + 1;  // 剥去左半
        else r = mid - 1;  // 剥去右半
    }
    return -1;  // 不利有攸往：未找到
}
```

### 算法详解：夬卦KMP算法

```
夬：扬于王庭，孚号有厉。

算法诠释：
- 扬于王庭：公开宣告（预处理next数组）
- 孚号：信任前缀函数
- 有厉：失配时果断跳转，不回头

核心：next数组如决断之剑，失配时当机立断
```

**KMP算法**
```cpp
void buildNext(string& p, vector<int>& next) {
    int m = p.size();
    next[0] = -1;
    int i = 0, j = -1;
    while (i < m - 1) {
        if (j == -1 || p[i] == p[j]) {
            i++, j++;
            next[i] = j;  // 扬于王庭：记录最大相等前后缀
        } else {
            j = next[j];  // 孚号有厉：果断回跳
        }
    }
}
```

## 编程启示录

### 1. 阴阳平衡 = 时空权衡

- **阳（时间）**：预处理、缓存、优化算法
- **阴（空间）**：节省内存、压缩存储
- **中道**：根据场景选择最优平衡点

### 2. 卦变 = 状态转移

六十四卦相错相综，如算法中的状态转移：
```
乾 → 姤 → 遁 → 否 → 观 → 剥 → 晋 → 大有 → 乾
（循环往复，生生不息）
```

### 3. 象数思维 = 抽象建模

- **象**：观察问题现象（建模）
- **数**：量化分析（复杂度）
- **理**：找出规律（算法设计）
- **占**：预测结果（正确性证明）
