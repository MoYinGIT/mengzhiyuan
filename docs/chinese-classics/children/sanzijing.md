## 📖 原文与解读

### 第一部分：教育的价值

> **人之初，性本善。性相近，习相远。**

**意思**：人生下来的时候，本性都是善良的。天性都差不多，但后天的学习和习惯使人差别很大。

**编程连接**：
```
每个孩子都有一颗好奇的心（性本善）
通过学习编程（习）
有的孩子成为了创造者
有的孩子只是消费者
差别在于学习和实践
```

**给小朋友的话**：
> 每个人天生都很棒！但想要变得更厉害，就要不断学习哦。

---

> **苟不教，性乃迁。教之道，贵以专。**

**意思**：如果不教育，善良的本性就会改变。教育的方法，贵在专心。

**编程连接**：
```
学习编程不能三天打鱼两天晒网
要专注，要持续
就像打游戏通关一样
坚持才能胜利
```

**给小朋友的话**：
> 学编程要专心，不要学一会儿就玩别的。坚持下去，你会很厉害的！

---

### 第二部分：学习的榜样

> **昔孟母，择邻处。子不学，断机杼。**

**故事**：孟子的妈妈为了让孟子有好的学习环境，搬了三次家。孟子逃学，孟母剪断织布，教育他学习不能半途而废。

**编程连接**：
```
孟母三迁 = 选择好的学习环境
子不学断机杼 = 学习不能半途而废

写程序也一样：
- 找个安静的地方写代码
- 遇到问题不要放弃，要解决问题
```

**Scratch 小游戏：孟母三迁**
```scratch
当 绿旗被点击
    移到 x:-200 y:0
    说 "我要找个好地方学习！" 持续 2 秒
    
    重复执行 3 次
        移动 100 步
        说 "这里可以吗？" 持续 1 秒
        如果 环境喧闹 那么
            说 "太吵了，我要搬家！" 持续 2 秒
        否则
            说 "这里很好！" 持续 2 秒
            停止 全部
        结束如果
    结束重复
结束
```

---

> **窦燕山，有义方。教五子，名俱扬。**

**故事**：窦燕山有五个儿子，他用正确的方法教育，五个儿子都很有成就。

**编程连接**：
```
好方法很重要！
学编程也有好方法：
1. 先看例子
2. 再动手做
3. 遇到问题查资料
4. 做完后分享给别人
```

---

### 第三部分：学习的内容

> **养不教，父之过。教不严，师之惰。**

**意思**：生养了孩子不教育是父亲的过错，教育不严格是老师的懒惰。

**现代理解**：
```
家长、老师、孩子，三方都要努力
- 家长要支持孩子学习
- 老师要认真教
- 孩子要用心学
```

---

> **子不学，非所宜。幼不学，老何为。**

**意思**：孩子不学习是不应该的。小时候不学习，老了能做什么呢？

**编程连接**：
```
小时候学编程：
- 大脑像海绵，吸收快
- 时间多，可以慢慢学
- 为未来打基础

现在开始学 Scratch
以后可以学 Python、C++
将来可以做游戏、做 APP、做人工智能！
```

---

### 第四部分：学习的方法

> **玉不琢，不成器。人不学，不知义。**

**意思**：玉石不雕琢，就不能成为有用的器物。人不学习，就不懂得道理。

**编程连接**：
```
每个人都是一块玉
学习就是雕琢的过程

写程序也是：
第一遍：能运行就行（毛坯）
第二遍：优化一下（粗加工）
第三遍：漂亮又高效（精品）
```

---

> **为人子，方少时。亲师友，习礼仪。**

**意思**：做儿女的，年纪还小的时候，应该亲近老师和朋友，学习礼仪。

**给小朋友的话**：
> 学编程也要"亲师友"：
- 尊重教你编程的老师
- 和一起学编程的朋友交流
- 在网上分享自己的作品
- 学习别人的优秀作品

## 🎮 编程与三字经

### 项目一：三字经接龙游戏

**规则**：
- 电脑出上句，你接下句
- 接对了得分

```scratch
定义 出题
    将 当前题号 设为 随机数从 1 到 5
    如果 当前题号 = 1 那么
        说 "人之初，___" 并等待
    结束如果
    // ... 更多题目
结束定义

定义 检查答案 接收 用户答案
    如果 当前题号 = 1 且 用户答案 包含 "性本善" 那么
        说 "答对了！+10分" 持续 2 秒
        将 分数 增加 10
    否则
        说 "再想想，答案是：性本善" 持续 2 秒
    结束如果
结束定义

当 绿旗被点击
    将 分数 设为 0
    重复执行 5 次
        出题
        检查答案 回答
    结束重复
    说 连接 "游戏结束，总分：" 分数 持续 3 秒
结束
```

### 项目二：三字经朗诵动画

**创意**：
- 每句三字经对应一个动画场景
- 点击句子播放动画

```scratch
当角色被点击
    如果 当前造型 = "人之初" 那么
        说 "人之初，性本善" 持续 3 秒
        切换造型到 小孩笑脸
        播放声音 笑声
    结束如果
结束
```

## 🌟 三字经的智慧

| 原文 | 智慧 | 编程应用 |
|:---|:---|:---|
| 性本善 | 每个人都有潜力 | 每个孩子都能学会编程 |
| 贵以专 | 要专心 | 写代码时不要分心 |
| 断机杼 | 不能半途而废 | Bug 要解决到底 |
| 玉不琢 | 需要磨练 | 代码要不断优化 |
| 亲师友 | 要交流 | 分享代码，学习他人 |
