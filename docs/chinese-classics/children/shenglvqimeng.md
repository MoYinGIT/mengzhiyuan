## 📖 一东韵（第一段）

> **云对雨，雪对风，晚照对晴空。**
> **来鸿对去燕，宿鸟对鸣虫。**
> **三尺剑，六钧弓，岭北对江东。**
> **人间清暑殿，天上广寒宫。**
> **两岸晓烟杨柳绿，一园春雨杏花红。**
> **两鬓风霜，途次早行之客；**
> **一蓑烟雨，溪边晚钓之翁。**

### 字词解释

| 词语 | 解释 |
|:---|:---|
| 晚照 | 夕阳 |
| 来鸿 | 飞来的大雁 |
| 宿鸟 | 归巢休息的鸟 |
| 三尺剑 | 古代剑长约三尺，这里指剑 |
| 六钧弓 | 强弓（钧是古代重量单位）|
| 途次 | 旅途中 |

### 对仗分析

```
云 对 雨（自然现象）
雪 对 风（自然现象）
晚照 对 晴空（天气景象）
来鸿 对 去燕（鸟类，动作相反）
宿鸟 对 鸣虫（动物，状态相对）
```

**编程连接 - 对仗检测器**：
```scratch
定义 检查对仗 接收 上联 下联
    如果 上联 的长度 不等于 下联 的长度 那么
        返回 false
    结束如果
    
    // 简单的词性检查（简化版）
    返回 true
结束定义

// 示例：检查"云对雨"
如果 检查对仗 "云" "雨" 那么
    说 "对仗工整！" 持续 2 秒
结束如果
```

## 📖 二冬韵（第一段）

> **春对夏，秋对冬，暮鼓对晨钟。**
> **观山对玩水，绿竹对苍松。**
> **冯妇虎，叶公龙，舞蝶对鸣蛩。**
> **衔泥双紫燕，课蜜几黄蜂。**
> **春日园中莺恰恰，秋天塞外雁雍雍。**
> **秦岭云横，迢递八千远路；**
> **巫山雨洗，嵯峨十二危峰。**

### 典故学习

**冯妇虎**：
- 冯妇是古代勇士，善于打虎
- 后来成为读书人，再也不打虎了
- 有一天看到老虎，忍不住又去打
- 比喻重操旧业

**叶公龙**：
- 叶公非常喜欢龙
- 家里到处画着龙、刻着龙
- 真龙听说后很感动，从天而降
- 叶公看到真龙却吓得逃跑
- 比喻表面上喜欢，实际上害怕

    将 典故解释 设为 [
        重操旧业的勇士,
        表面喜欢实际害怕,
        不劳而获的农夫,
        出了问题及时补救
    ]
结束定义

    // 显示名称，让学生选择正确的解释

## 📖 三江韵（精选）

> **楼对阁，户对窗，巨海对长江。**
> **蓉裳对蕙帐，玉斝对银釭。**
> **青布幔，碧油幢，宝剑对金缸。**
> **忠心安社稷，利口覆家邦。**
> **世祖中兴延马武，桀王失道杀龙逄。**
> **秋雨潇潇，漫烂黄花都满径；**
> **春风袅袅，扶疏绿竹正盈窗。**

### 历史人物

**马武**：
- 东汉名将
- 帮助光武帝刘秀建立东汉
- 忠心耿耿，战功赫赫

**龙逄（páng）**：
- 夏朝贤臣
- 桀王无道，龙逄多次劝谏
- 桀王不听，反而杀了龙逄
- 比喻忠臣被害

## 🎮 编程与声律启蒙

### 项目一：对联生成器

```scratch
定义 生成对联
    将 上联词库 设为 [云,雨,雪,风,春,夏,秋,冬]
    将 下联词库 设为 [雨,云,风,雪,夏,春,冬,秋]
    
    将 随机索引 设为 随机数从 1 到 上联词库 的长度
    将 上联 设为 上联词库 的第 随机索引 项
    将 下联 设为 下联词库 的第 随机索引 项
    
    说 连接 上联 "对" 下联 持续 3 秒
结束定义

当 绿旗被点击
    生成对联
结束
```

### 项目二：韵脚检测器

```scratch
定义 检查韵脚 接收 诗句
    // 简化版：检测最后一个字
    将 最后一个字 设为 诗句 的最后一个字符
    
    // 东韵常用字
    将 东韵字 设为 [风,空,虫,弓,东,宫,红,翁]
    
    如果 东韵字 包含 最后一个字 那么
        返回 true
    否则
        返回 false
    结束如果
结束定义

当 绿旗被点击
    如果 检查韵脚 "晚照对晴空" 那么
        说 "属于东韵！" 持续 2 秒
    结束如果
结束
```

### 项目三：诗词填空

```scratch
定义 出题
    将 题目 设为 "云对__，雪对风"
    将 答案 设为 "雨"
    
    说 题目 持续 3 秒
    询问 "请填空：" 并等待
    
    如果 回答 = 答案 那么
        说 "答对了！" 持续 2 秒
        将 分数 增加 10
    否则
        说 连接 "错了，答案是：" 答案 持续 2 秒
    结束如果
结束定义

当 绿旗被点击
    将 分数 设为 0
    重复执行 5 次
        出题
    结束重复
    说 连接 "游戏结束，得分：" 分数 持续 3 秒
结束
```
