# 《易经》帛书甲本

> 马王堆汉墓出土帛书 · 西汉初期抄本

---

## 版本说明

帛书《易经》1973年出土于湖南长沙马王堆汉墓三号墓，是现存最早的《易经》版本之一。与通行本相比，帛书本有以下显著特点：

### 主要差异

1. **卦序不同**：通行本以乾、坤为首，帛书本有独特的卦序排列
2. **卦名古字**：多用古字、假借字，如"键"代"乾"、"川"代"坤"
3. **古易传**：附有与通行本不同的古易传
4. **无《彖传》《象传》**：仅有《系辞》等部分内容

## 帛书本易传

帛书本附有古易传，包括：

### 现存篇目

| 篇名 | 说明 |
|:---|:---|
| **《系辞》** | 与通行本相似，但顺序不同 |
| **《易之义》** | 解释卦义，通行本无 |
| **《要》** | 论易之要旨，通行本无 |
| **《缪和》** | 问答体易传，通行本无 |
| **《昭力》** | 问答体易传，通行本无 |

### 与通行本差异

- **无《彖传》**：通行本每卦有《彖》解释卦辞，帛书本无
- **无《象传》**：通行本每卦有《大象》《小象》，帛书本无
- **有独特篇章**：如《易之义》《要》《缪和》《昭力》等

## 卦名对照详解

### 八纯卦

| 帛书名 | 通行本 | 字义 | 象征 |
|:---:|:---:|:---:|:---|
| **键** | 乾 | 刚健 | 天 |
| **川** | 坤 | 大地 | 地 |
| **辰** | 震 | 震动 | 雷 |
| **筭** | 巽 | 进入 | 风 |
| **罗** | 离 | 附丽 | 火 |
| **赣** | 坎 | 水之意 | 水 |
| **根** | 艮 | 止之意 | 山 |
| **夺** | 兑 | 喜悦 | 泽 |

### 重要卦名演变

- **键 → 乾**："键"强调刚健之性，"乾"为天之象
- **川 → 坤**："川"为大地之形，"坤"为柔顺之德
- **辰 → 震**："辰"与"震"音近假借
- **罗 → 离**："罗"为网罗附丽之意

## 研究价值

### 学术价值

1. **校勘价值**：可用于校正通行本的讹误
2. **文字学价值**：保留大量先秦古字
3. **易学史价值**：反映汉初易学面貌
4. **思想史价值**：早期易传思想的珍贵资料

### 与通行本的关系

帛书本与通行本同源而异流，两者关系：
- **同源**：均源自先秦《易经》
- **异流**：抄写传承中形成不同版本
- **互补**：相互参照，更全面了解《易经》
