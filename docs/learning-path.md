> 从Scratch到C++竞赛的完整进阶路线

---

```

  │                        │                        │
  ▼                        ▼                        ▼
┌─────────┐            ┌─────────┐            ┌─────────┐
│ Scratch │ ────────▶  │ Python  │ ────────▶  │  C++    │
```

## 阶段一：Scratch 图形化编程（6-8岁）

### 学习目标
- 培养编程思维和逻辑能力
- 理解顺序、分支、循环
- 激发创造力和兴趣

### 学习时长
- **总时长**：6-12个月
- **每周**：2-3小时
- **项目**：完成5-8个创意项目

### 进阶路径

```
Scratch L1（2个月）
    ├── 基础概念：舞台、角色、指令
    ├── 事件驱动：点击绿旗
    └── 项目：动画故事
            │
            ▼
Scratch L2（2个月）
    ├── 循环结构：重复执行
    ├── 条件判断：如果...那么
    └── 项目：简单游戏
            │
            ▼
Scratch L3（2个月）
    ├── 变量：计数器、分数
    ├── 消息广播：角色协作
    └── 项目：完整游戏
            │
            ▼
Scratch L4（2个月）
    ├── 列表/数组
    ├── 自定义积木（函数）
    └── 项目：复杂应用
```

### 里程碑检查清单

- [ ] 能独立完成动画故事
- [ ] 理解循环和条件的区别
- [ ] 会使用变量记录状态
- [ ] 能制作简单的互动游戏
- [ ] 对编程保持兴趣和热情

---

## 阶段二：Python 代码编程（9-11岁）

### 学习目标
- 从图形化过渡到代码
- 掌握基础语法和逻辑思维
- 培养代码编写规范

### 学习时长
- **总时长**：4-6个月
- **每周**：3-4小时
- **项目**：完成10+个小程序

### 进阶路径

```
Python L1（2-3个月）
    ├── 变量和数据类型
    ├── 输入输出
    ├── 运算符
    └── 字符串基础
            │
            ▼
Python L2（2-3个月）
    ├── 条件语句（if-elif-else）
    ├── 循环（for、while）
    ├── 列表基础
    └── 简单函数
```

### 里程碑检查清单

- [ ] 能独立编写20-30行程序
- [ ] 理解代码的执行顺序
- [ ] 掌握基础语法（缩进、冒号等）
- [ ] 能使用循环处理重复问题
- [ ] 能调试简单的语法错误

---

## 阶段三：C++ 竞赛编程（12岁+）

### 学习目标
- 掌握C++语法和STL
- 学习算法和数据结构
- 准备GESP考级和竞赛

### 学习时长
- **总时长**：24-36个月
- **每周**：5-10小时（含刷题）
- **目标**：通过GESP L8，参加CSP-J/S

### 进阶路径

```
C++ L1-L2（6个月）
    ├── 变量、数据类型
    ├── 输入输出（cin/cout）
    ├── 分支结构（if/switch）
    ├── 循环结构（for/while）
    └── 基础练习50题
            │
            ▼
C++ L3-L4（8个月）
    ├── 数组与字符串
    ├── 函数与递归
    ├── 结构体
    ├── 基础算法（排序、查找）
    └── 进阶练习80题
            │
            ▼
C++ L5-L6（10个月）
    ├── 指针与内存
    ├── 链表、栈、队列
    ├── 数论基础
    ├── 递归与分治
    ├── 动态规划入门
    └── 算法练习150题
            │
            ▼
C++ L7-L8（12个月）
    ├── 高级数据结构
    │   ├── 并查集、树状数组
    │   ├── 线段树、Trie
    │   └── 二叉堆
    ├── 图论算法
    │   ├── 最短路
    │   ├── 最小生成树
    │   └── 拓扑排序
    ├── 高级DP
    ├── 竞赛数学
    └── 竞赛练习200题+
```

### 各阶段里程碑

#### L1-L2 里程碑
- [ ] 能编写50行以内的程序
- [ ] 掌握三种程序结构
- [ ] 能独立调试语法错误
- [ ] 通过GESP L2

#### L3-L4 里程碑
- [ ] 熟练使用数组处理批量数据
- [ ] 理解函数和递归
- [ ] 掌握结构体和排序
- [ ] 通过GESP L4

#### L5-L6 里程碑
- [ ] 理解指针和内存管理
- [ ] 掌握链表和基础数据结构
- [ ] 能独立解决DP入门题
- [ ] 通过GESP L6

#### L7-L8 里程碑
- [ ] 掌握高级数据结构
- [ ] 能独立解决图论问题
- [ ] 能参加CSP-J并获奖
- [ ] 通过GESP L8

## 学习时间规划

### 日常学习计划

| 时间段 | 活动 | 时长 | 说明 |
|:---:|:---|:---:|:---|
| **理论学习** | 学习新知识点 | 30-60分钟 | 看书/视频/教程 |
| **代码练习** | 动手写代码 | 30-60分钟 | 做例题、练习题 |
| **刷题巩固** | 在线评测 | 30-60分钟 | OJ平台做题 |
| **复盘总结** | 整理错题 | 15-30分钟 | 记录、复习 |

### 周计划示例

```
周一：学习新知识点（L3数组）
周二：完成教材例题 + 练习
周三：刷题（数组相关10题）
周四：学习新知识点（L3字符串）
周五：完成教材例题 + 练习
周六：刷题（字符串相关10题）
周日：复盘本周错题 + 休息
```

### 假期强化计划

**寒暑假**（每天3-4小时）：
- 上午：学习新知识点
- 下午：刷题练习
- 晚上：复盘总结

**目标**：假期完成一个级别的学习

## 给家长的建议

### 各阶段家长角色

| 阶段 | 家长任务 | 注意事项 |
|:---:|:---|:---|
| **Scratch** | 陪伴创作、欣赏作品 | 多鼓励，不评判 |
| **Python** | 协助调试、讨论思路 | 不代写代码 |
| **C++初级** | 监督练习、情绪支持 | 关注过程非结果 |
| **C++进阶** | 创造环境、后勤保障 | 尊重孩子节奏 |
| **竞赛期** | 心理支持、压力疏导 | 不施加额外压力 |

### 关键决策点

**6-8岁**：是否开始编程？
- ✅ 可以开始Scratch，培养兴趣为主
- ❌ 不要急于学习代码

**9-11岁**：是否转学C++？
- ✅ Python基础扎实后可转
- ❌ 不要跳过Python直接C++

**12岁+**：是否走竞赛路线？
- ✅ 有兴趣、有天赋、有时间
- ❌ 不要为了升学强行竞赛
