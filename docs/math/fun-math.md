## 🎮 数学游戏

### 1. 24点游戏

**规则**：
- 给定4个数字
- 使用 +、-、×、÷ 和括号
- 每个数字只能用一次
- 使结果等于24

**示例**：
```
数字：3、3、8、8

解法：(8 ÷ (3 - 8 ÷ 3)) = 24

步骤：
8 ÷ 3 = 8/3
3 - 8/3 = 1/3
8 ÷ (1/3) = 24
```

**Scratch 项目：24点计算器**
```scratch

        返回 true
    否则 如果 a * b + c + d = 24 那么
        返回 true
    // ... 更多组合
    结束如果
    返回 false
结束定义

    将 数字 设为 [3, 3, 8, 8]

    否则
        说 "让我想想..." 持续 2 秒
    结束如果
```

### 3. 汉诺塔

**规则**：
- 三根柱子，n个大小不同的盘子
- 大盘子不能压在小盘子上
- 把所有盘子从A柱移到C柱

**最少步数**：2ⁿ - 1
- 3个盘子：7步
- 4个盘子：15步
- 5个盘子：31步

**Scratch 项目：汉诺塔可视化**
```scratch
定义 汉诺塔 接收 n 从柱 到柱 辅助柱
    如果 n = 1 那么
        说 连接 "把盘子从" 从柱 "移到" 到柱 持续 1 秒
    否则
        汉诺塔 n-1 从柱 辅助柱 到柱
        汉诺塔 1 从柱 到柱 辅助柱
        汉诺塔 n-1 辅助柱 到柱 从柱
    结束如果
结束定义

当 绿旗被点击
    汉诺塔 3 "A" "C" "B"
结束
```

## 🧩 数学谜题

### 1. 鸡兔同笼

**问题**：笼子里有鸡和兔，头共35个，脚共94只。鸡兔各几只？

**解法一：假设法**
```
假设全是鸡：
35 × 2 = 70只脚
实际94只，少了：94 - 70 = 24只
每把一只鸡换成兔，多2只脚
兔：24 ÷ 2 = 12只
鸡：35 - 12 = 23只
```

**解法二：编程法**
```cpp
#include <iostream>
using namespace std;

int main() {
    int heads = 35, legs = 94;
    
    for (int chicken = 0; chicken <= heads; chicken++) {
        int rabbit = heads - chicken;
        if (chicken * 2 + rabbit * 4 == legs) {
            cout << "鸡：" << chicken << "，兔：" << rabbit << endl;
        }
    }
    return 0;
}
```

### 2. 年龄问题

**问题**：父亲今年35岁，儿子5岁。几年后父亲年龄是儿子的3倍？

**解法**：
```
设x年后：
35 + x = 3 × (5 + x)
35 + x = 15 + 3x
20 = 2x
x = 10

答：10年后，父亲45岁，儿子15岁。
```

### 3. 渡河问题

**问题**：农夫带着狼、羊、白菜过河。船只能载农夫和一件东西。狼吃羊，羊吃白菜，农夫不在时会发生危险。如何安全过河？

**解法**：
```
1. 带羊过河，农夫返回
2. 带狼（或白菜）过河，带羊返回
3. 带白菜（或狼）过河，农夫返回
4. 带羊过河
```

## 🔢 逻辑推理

### 1. 真假话问题

**问题**：甲说"乙在说谎"，乙说"丙在说谎"，丙说"甲和乙都在说谎"。谁在说谎？

**分析**：
```
假设甲说真话：
- 乙在说谎
- 乙说"丙在说谎"是假话，所以丙说真话
- 丙说"甲和乙都在说谎"是真话，与甲说真话矛盾

假设甲说谎：
- 乙说真话
- 丙在说谎（因为乙说真话）
- 丙说"甲和乙都在说谎"是假话，符合甲说谎、乙说真话

结论：甲说谎，乙说真话，丙说谎
```

### 2. 爱因斯坦谜题（简化版）

**问题**：三人分别来自北京、上海、广州。已知：
- A不是来自北京
- B不是来自上海
- 来自北京的人不是最年轻的
- B比来自广州的人年龄大

问：三人分别来自哪里？

**解答**：
```
从"B比来自广州的人年龄大"：B不是广州的
从"B不是来自上海"：B只能是北京的
从"A不是来自北京"：A只能是上海或广州
从"B比来自广州的人年龄大"：A是广州的
所以：C是上海的

答案：A-广州，B-北京，C-上海
```
