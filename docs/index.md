## 🎯 我们的使命

> **让所有孩子有处可学。**

在信息时代，编程和数学已成为基础素养。然而，优质教育资源往往分散、昂贵、难以获取。

「默隐·蒙知苑」致力于：
- 🌱 提供**系统化**的学习路径
- 📖 整理**高质量**的教学内容
- 👨‍👩‍👧 建立**亲子共学**的桥梁
- 💡 培养**面向未来**的核心能力

## 🚀 快速开始

| 阶段 | 推荐年龄 | 学习内容 | 目标 |
|:---:|:---:|:---|:---|
| **启蒙** | 6-10岁 | Scratch 图形化编程 | 培养逻辑思维，激发兴趣 |
| **入门** | 10-12岁 | Python 代码编程 | 掌握编程基础，解决问题 |
| **进阶** | 12岁+ | C++ 竞赛编程 | GESP/NOIP 考级，算法竞赛 |
| **拓展** | 全年龄 | 数学思维、心理学 | 全面发展，健康成长 |

## 🏛️ 三经典根基

我们的教育理念根植于中华国学三经典：

| 经典 | 核心 | 在教育中的体现 |
|:---|:---|:---|
| **《易经》** | 穷则变，变则通 | 因材施教，灵活调整教学方法 |
| **《道德经》** | 道法自然，无为而治 | 顺应孩子天性，不强行灌输 |
| **《论语》** | 有教无类，因材施教 | 免费开放，惠及每一个孩子 |
