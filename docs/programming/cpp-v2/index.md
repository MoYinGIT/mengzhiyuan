## 🎯 关于 v2.0 教学体系

本套教程采用**五大思维模型**重构：

| 原则 | 说明 |
|:---|:---|
| **引导而不灌输** | 不给答案，只给提示，让你自己发现知识 |
| **分层而不跳跃** | 每课都有前置检查，确保基础扎实再进阶 |
| **分解而不畏惧** | 复杂问题拆成小步骤，一步一步解决 |
| **练习而不盲目** | 每个知识点都有针对性练习，即学即练 |
| **反思而不停滞** | 课后三省吾身，建立错题本，持续进步 |

## 🎓 学习路径

```
L1 (编程入门)
    ↓ 完成练习 + 通过自检
L2 (分支循环)
    ↓ 完成练习 + 通过自检
L3 (数组基础)
    ↓ 完成练习 + 通过自检
L4 (函数结构体)
    ↓ 完成练习 + 通过自检
L5-L8 (算法进阶)
```

**重要**：不要跳跃！每课都有**课前自检**，请确认掌握前置知识后再学习。

---

## 📝 学习方法

### 1. 课前
- 完成**课前自检**，确认前置知识已掌握
- 如果没有掌握，返回前一课复习

### 2. 课中
- 按照**知识探索**的顺序学习
- 先自己思考，需要时再点开**提示**
- 完成所有**填空练习**和**思考题**

### 3. 课后
- 完成**动手实践**的所有练习
- 填写**错题本**
- 完成**三省吾身**自检
