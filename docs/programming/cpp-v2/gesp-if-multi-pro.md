    int month = 3;
    
    // 正确的比较

## 核心要点总结

### if多分支使用检查清单

- [ ] 基本格式：`if (条件1) {...} else if (条件2) {...} ... else {...}`
- [ ] 从上到下依次判断，满足一个条件就执行对应分支并结束
- [ ] 只执行一个分支，不会执行多个
- [ ] 条件必须从严格到宽松排列（如`>=90` → `>=80` → ...）
- [ ] else if可以有多个，else可以省略（建议保留）
- [ ] 使用`==`进行比较，不要用`=`赋值
- [ ] 各分支中定义的变量只在各自分支有效
- [ ] 对于互斥条件，多分支比多个if更合适

### 常见错误对比

| 错误写法 | 问题 | 正确写法 |
|:---|:---|:---|
| 条件顺序颠倒 | 逻辑错误，如`>=60`放最前 | 从严格到宽松排列 |
| `if (a = 5)` | 赋值而非比较 | `if (a == 5)` |
| 省略else后需要兜底 | 可能什么都不执行 | 建议保留else |
| 分支内定义变量后外部使用 | 作用域错误 | 在外部定义变量 |

### GESP一级if多分支考点

1. **基本语法**：if-else if-else的格式
2. **执行流程**：从上到下，满足即停止
3. **条件顺序**：从严格到宽松
4. **与多个if区别**：多分支只执行一个
5. **else作用**：兜底处理，可省略但建议保留
6. **实际应用**：成绩等级、BMI、税费、星期判断等
