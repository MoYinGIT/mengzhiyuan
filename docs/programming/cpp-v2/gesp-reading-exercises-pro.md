## 【大题1】数据类型的"隐形转换"

**考察点**：整型与浮点型混合运算时的类型转换规则

```cpp
#include <iostream>
using namespace std;
int main() {
    int a = 5;
    double b = 2.0;
    int c = a / b * 2;      // 语句1
    double d = a / 2 * b;   // 语句2
    int e = a / 2 * 2;      // 语句3
    
    cout << c << endl;
    cout << d << endl;
    cout << e << endl;
    return 0;
}
```

### 一、选择题（考察理解深度）

**1. 语句1 `int c = a / b * 2` 的执行过程是（ ）**

A. 5/2=2，然后2*2=4，最后赋值给c  
B. 5/2.0=2.5，然后2.5*2=5.0，最后转为整数5赋值给c  
C. 5*2=10，然后10/2.0=5，最后转为整数5赋值给c  
D. 直接计算5/2.0*2=5，然后赋值给c

**2. 语句2 `double d = a / 2 * b` 的执行过程是（ ）**

A. 5/2=2，然后2*2.0=4.0，d=4.0  
B. 5*2.0=10.0，然后10.0/2=5.0，d=5.0  
C. 5/2.0=2.5，然后2.5*2.0=5.0，d=5.0  
D. 编译错误，类型不匹配

**3. 如果将语句3改为 `int e = a / 2.0 * 2`，结果是（ ）**

A. 4  
B. 5  
C. 5.0  
D. 编译错误

### 二、判断题（考察概念辨析）

**4. `int c = a / b` 中，因为c是int类型，所以a/b会按整数除法计算。**  
（ ）正确 （ ）错误

**5. 表达式 `5 / 2 * 2.0` 和 `5 * 2.0 / 2` 的结果相同。**  
（ ）正确 （ ）错误

**6. 在混合运算中，只要有一个操作数是double，整个表达式就会按double计算。**  
（ ）正确 （ ）错误

### 三、填空题（考察推理能力）

**7. 语句1的最终结果c = ______（填写整数）**

**8. 语句2的最终结果d = ______（保留1位小数）**

**9. 如果要使 `a / 2 * b` 的结果为5.0，应该将表达式改为 ______**

---

**解析关键点**：
- 语句1：5/2.0=2.5（double），2.5*2=5.0（double），转int得5
- 语句2：5/2=2（int），2*2.0=4.0（double）
- 语句3：5/2=2（int），2*2=4（int）
- 陷阱：除法从左到右计算，类型转换发生在运算时，不是赋值时

## 【大题2】自增运算符的"坑"

**考察点**：前置++与后置++的区别，以及连续自增的求值顺序

```cpp

using namespace std;
int main() {

    cout << y << " " << b << endl;
```

**1. 语句1执行后，x的值是（ ）**

A. a=7, b=7  
B. a=6, b=7  
C. a=7, b=6  
D. 不确定

### 二、判断题

**4. `a++`先用a的原值，再自增；`++a`先自增，再用新值。**  
（ ）正确 （ ）错误

**5. 在同一个表达式中多次修改同一个变量（如`a++ + a++`）是安全的，结果是确定的。**  
（ ）正确 （ ）错误

**6. 语句`int x = ++a;`和`int x = a++; x = x + 1;`的效果完全相同。**

**8. 为了避免歧义，语句2应该拆分为：`++b; int y = b + ______;`（填写表达式）

**9. 后置++表达式的值是变量______的值，前置++表达式的值是变量______的值。**

---

**解析关键点**：

## 【大题3】逻辑运算的"短路"

**考察点**：逻辑运算符的短路特性，以及0/非0值的判断

```cpp
#include <iostream>
using namespace std;
int main() {
    int a = 0, b = 5, c = 0;
    bool r1 = (a++ && ++b);     // 语句1
    bool r2 = (++c || --b);     // 语句2
    bool r3 = (a && !b || c);   // 语句3
    
    cout << r1 << " " << a << " " << b << endl;
    cout << r2 << " " << c << " " << b << endl;
    cout << r3 << endl;
    return 0;
}
```

### 一、选择题

**1. 语句1执行后，a和b的值分别是（ ）**

A. a=1, b=5  
B. a=1, b=6  
C. a=0, b=5  
D. a=0, b=6

**2. 语句1中r1的值是（ ）**

A. true  
B. false  
C. 0  
D. 1

**3. 语句3 `a && !b || c` 的运算顺序是（ ）**

A. 从左到右：(a && !b) || c  
B. 先算||：a && (!b || c)  
C. 先算!：(a && b) || c  
D. 先算&&和||，优先级相同

### 二、判断题

**4. `&&`运算符具有短路特性：如果左边为false，右边不会执行。**  
（ ）正确 （ ）错误

**5. `||`运算符具有短路特性：如果左边为true，右边不会执行。**  
（ ）正确 （ ）错误

**6. 在C++中，非0值都视为true，0视为false。**  
（ ）正确 （ ）错误

### 三、填空题

**7. 语句1执行后，r1 = ______（填0或1），因为左边a++的值为______，短路导致右边不执行。**

**8. 语句2执行后，b的值仍然是______，因为短路特性使得`--b`不会执行。**

**9. 运算符优先级从高到低：`!` > ______ > ______。**

---

**解析关键点**：
- &&短路：左边为false，右边不执行
- ||短路：左边为true，右边不执行
- 优先级：! > && > ||
- a++先用0，再自增为1

## 【大题6】数字处理的"数学逻辑"

**考察点**：数字分解算法的理解，特别是"去尾"和"取尾"

```cpp
#include <iostream>
using namespace std;
int main() {
    int n = 12345;
    int sum = 0, count = 0, reversed = 0;
    
    while (n > 0) {
        int digit = n % 10;     // 取个位
        sum += digit;
        count++;
        reversed = reversed * 10 + digit;
        n /= 10;                // 去个位
    }
    
    cout << sum << endl;
    cout << count << endl;
    cout << reversed << endl;
    return 0;
}
```

### 一、选择题

**1. 程序运行后，sum的值是（ ）**

A. 15  
B. 12345

D. 54321

**2. 第一次循环后，n的值是（ ）**

A. 12345  
B. 1234  
C. 1235  
D. 4

**3. `reversed = reversed * 10 + digit`的作用是（ ）**

A. 累加各位数字  
B. 构造反转后的数字  
C. 去掉最后一位  
D. 计算位数

### 二、判断题

**4. `n % 10`得到n的个位数字（最后一位）。**  
（ ）正确 （ ）错误

**5. `n /= 10`去掉n的个位数字（相当于整除10）。**  
（ ）正确 （ ）错误

**6. 如果n=100，反转后的数字是1（前导零丢失）。**  
（ ）正确 （ ）错误

### 三、填空题

**7. 如果n=12345，循环执行______次，因为n有______位。**

**8. 如果n=100，sum=______，reversed=______。**

**9. 如果要计算各位数字之积（而非和），应该将`sum += digit`改为`product ______ digit`**。

---

**解析关键点**：
- 12345 → 1+2+3+4+5=15

## 【大题8】switch的"穿透"

```cpp
#include <iostream>
using namespace std;
int main() {
    int score = 85;
    int grade = score / 10;
    
    switch (grade) {
        case 10:
        case 9:
            cout << "A" << endl;
            break;
        case 8:
            cout << "B" << endl;
        case 7:
            cout << "C" << endl;
            break;
        case 6:
            cout << "D" << endl;
        default:
            cout << "F" << endl;
    }
    return 0;
}
```

**1. score=85时，程序的输出是（ ）**

A. B

**3. score=75时，程序的输出是（ ）**

A. C  
B. D  
C. CD  
D. CDF

### 二、判断题

**4. case 10后面没有break，会"穿透"到case 9执行。**  
（ ）正确 （ ）错误

**5. case 8后面没有break，所以执行完case 8会继续执行case 7。**  
（ ）正确 （ ）错误

**6. default分支在所有case都不匹配时执行，不需要break。**  
（ ）正确 （ ）错误

### 三、填空题

**7. 如果要让85分只输出B，需要在case 8的cout后添加______语句。**

**8. case 10和case 9共用输出"A"，这种写法称为case______。**

**9. 如果score=100，grade=______，会执行case______。**

---

**解析关键点**：
- 85/10=8 → 进入case 8 → 输出B → 穿透到case 7 → 输出C → break
- 结果：BC

## 【大题10】类型转换的"精度损失"

**考察点**：强制类型转换的截断效果，以及整数除法的陷阱

```cpp
#include <iostream>
using namespace std;
int main() {
    double a = 7.9;
    double b = 7.1;
    
    int x = (int)a;           // 截断
    int y = (int)(a + 0.5);   // 四舍五入
    int z = (int)b;           // 截断
    int w = (int)(b + 0.5);   // 四舍五入
    
    double result1 = 5 / 2;   // 整数除法
    double result2 = 5.0 / 2; // 浮点除法
    
    cout << x << " " << y << " " << z << " " << w << endl;
    cout << result1 << " " << result2 << endl;
    
    return 0;
}
```

### 一、选择题

**1. 第一行输出是（ ）**

A. 7 8 7 7  
B. 7 8 7 8
