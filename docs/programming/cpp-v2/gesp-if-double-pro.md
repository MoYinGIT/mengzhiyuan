## 【大题4】if双分支中的赋值陷阱

**考察点**：区分`=`（赋值）和`==`（比较）在if-else中的使用

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 5;
    
    // 正确使用比较
    if (a == 5) {
        cout << "a等于5" << endl;
    } else {
        cout << "a不等于5" << endl;
    }
    
    // 错误：使用了赋值
    int b = 3;
    if (b = 0) {  // 注意这里是赋值！
        cout << "b为真" << endl;
    } else {
        cout << "b为假" << endl;
    }
    cout << "b=" << b << endl;
    
    return 0;
}
```

### 一、选择题

**1. 第一个if-else的输出是（ ）**

A. a等于5  
B. a不等于5  
C. b为真  
D. b为假

**2. 第二个if-else的输出是（ ）**

A. b为真  
B. b为假  
C. b为假，换行，b=0  
D. b=0

**3. 为什么第二个if-else会执行else块（ ）**

A. 因为b不等于0  
B. 因为`b=0`将0赋给b，表达式的值为0（假）  
C. 因为b原来是3  
D. 编译错误

### 二、判断题

**4. `if (a == 5)` 是比较a是否等于5，`if (a = 5)` 是将5赋给a。**  
（ ）正确 （ ）错误

**5. 在if条件中使用`=`而不是`==`是常见的逻辑错误。**  
（ ）正确 （ ）错误

**6. `if (b = 0)` 中，b被赋值为0，表达式的值为0，条件为假，执行else块。**  
（ ）正确 （ ）错误

### 三、填空题

**7. 判断相等用`a ______ 5`（一个/两个等号）。**

**8. 赋值用`a ______ 5`（一个/两个等号）。**

**9. `if (a = 0)` 会将a赋值为______，表达式的值为______（真/假）。**

---

**解析关键点**：
- a==5为真，输出"a等于5"
- b=0是赋值，表达式值为0（假），执行else块，输出"b为假"
- 最后b的值为0

## 核心要点总结

### if双分支使用检查清单

- [ ] 基本格式：`if (条件) { 语句块1; } else { 语句块2; }`
- [ ] 条件为真执行if块，为假执行else块
- [ ] if块和else块必定有一个执行，不会都不执行，也不会都执行
- [ ] 可以使用嵌套if-else实现多级判断
- [ ] 结合逻辑运算符`&&`、`||`、`!`实现复杂条件
- [ ] 注意区分`==`（比较）和`=`（赋值）
- [ ] if和else后面不要加分号
- [ ] 建议使用花括号，即使只有一条语句
- [ ] 注意块内变量的作用域

### 常见错误对比

| 错误写法 | 问题 | 正确写法 |
|:---|:---|:---|
| `if (a = 5)` | 赋值而非比较 | `if (a == 5)` |
| `if (条件); { } else { }` | 分号导致语法错误 | `if (条件) { } else { }` |
| 省略花括号后添加代码 | 逻辑错误 | 始终使用花括号 |
| 在if-else内定义变量后外部使用 | 作用域错误 | 在外部定义变量 |
