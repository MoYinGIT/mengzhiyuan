## 【大题1】if单分支的基础语法

```cpp
    }
    
    cout << "程序结束" << endl;
    return 0;
}
```

### 一、选择题

**1. 上述程序中，语句1会执行吗（ ）**

A. 会执行  
B. 不会执行

D. 不确定

**3. 程序的输出结果是（ ）**

A. a大于5

D. 程序结束

### 二、判断题

（ ）正确 （ ）错误

（ ）正确 （ ）错误

**6. if语句的条件为假时，if块内的语句不会执行，程序会继续执行if块后面的语句。**

**8. 在C++中，0表示______（真/假），非0表示______（真/假）。**

## 【大题4】if语句中的赋值陷阱

**考察点**：区分`=`（赋值）和`==`（比较），以及赋值表达式的值

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 5;
    
    // 代码段1：比较运算符
    if (a == 5) {
        cout << "a等于5" << endl;
    }
    
    // 代码段2：赋值运算符（常见错误）
    if (a = 0) {
        cout << "a赋值为0" << endl;
    }
    cout << "a=" << a << endl;
    
    // 代码段3：赋值表达式的值
    int b = 10;
    if (b = 3) {
        cout << "条件为真" << endl;
    }
    cout << "b=" << b << endl;
    
    return 0;
}
```

### 一、选择题

**1. 代码段1的输出是（ ）**

A. a等于5  
B. 无输出  
C. a=5  
D. 编译错误

**2. 代码段2的输出是（ ）**

A. a赋值为0，换行，a=0  
B. a=0  
C. 无输出  
D. 编译错误

**3. 代码段3的输出是（ ）**

A. 条件为真，换行，b=3  
B. b=3  
C. 无输出  
D. 编译错误

### 二、判断题

**4. `=`是赋值运算符，`==`是比较运算符，两者完全不同。**  
（ ）正确 （ ）错误

**5. `if (a = 0)` 会将0赋值给a，然后判断表达式的值（0为假），所以if块不会执行。**  
（ ）正确 （ ）错误

**6. 赋值表达式的值就是被赋的值，所以`if (b = 3)`中3为非0，条件为真，if块会执行。**  
（ ）正确 （ ）错误

### 三、填空题

**7. 判断相等应该用______（一个/两个）等号，赋值用______（一个/两个）等号。**

**8. `if (a = 5)` 会将______赋值给______，然后表达式的值为______（0/非0）。**

**9. 常见的逻辑错误：`if (a = 0)` 本意是判断a是否等于0，但实际效果是______。**

---

**解析关键点**：
- `==`是比较，`=`是赋值
- `if (a = 0)` 将0赋给a，表达式值为0（假），不执行if块
- `if (b = 3)` 将3赋给b，表达式值为3（真），执行if块

## 核心要点总结

### if单分支使用检查清单

- [ ] 基本格式：`if (条件) { 语句块; }`
- [ ] 条件为真（非0）执行if块，为假（0）跳过
- [ ] 无论if块是否执行，程序都继续向下执行
- [ ] 建议使用花括号，即使只有一条语句
- [ ] 比较用`==`，赋值用`=`，不要混淆
- [ ] if块内可以访问和修改外部变量
- [ ] if块内定义的变量只在if块内有效
- [ ] if条件后不要加分号

### 常见错误对比

| 错误写法 | 问题 | 正确写法 |
|:---|:---|:---|
| `if (a = 5)` | 赋值而非比较 | `if (a == 5)` |
| `if (a > 5); { ... }` | 分号导致if控制空语句 | `if (a > 5) { ... }` |
| 省略花括号添加代码 | 逻辑错误 | 始终使用花括号 |
| `if (a > 5) int x = 10;` | 声明语句需要花括号 | `if (...) { int x = 10; }` |

### GESP一级if单分支考点

1. **基本语法**：if (条件) { 语句块 }
2. **真假判断**：非0为真，0为假
3. **花括号使用**：建议使用，避免错误
4. **比较与赋值**：`==` vs `=`
5. **变量作用域**：if块内定义的变量作用域
6. **实际应用**：累加筛选、绝对值、最大值、交换等
