## 【大题1】定义 vs 初始化 vs 赋值的"本质区别"

**考察点**：理解声明、初始化、赋值是三个不同的概念

```cpp
#include <iostream>
using namespace std;

int main() {
    int a;          // 语句1：定义
    int b = 10;     // 语句2：定义+初始化
    a = 20;         // 语句3：赋值
    
    int c = a = 30; // 语句4：连续操作
    
    cout << "a=" << a << ", b=" << b << ", c=" << c << endl;
    return 0;
}
```

### 一、选择题

**1. 语句1 `int a;` 执行后，a的值是（ ）**

A. 0  
B. 随机值（垃圾值）  
C. 未定义，不能使用  
D. 编译错误

**2. 语句4 `int c = a = 30;` 的执行顺序是（ ）**

A. 先执行int c = a，再执行a = 30  
B. 先执行a = 30，再执行int c = a  
C. 同时执行  
D. 编译错误

**3. 程序最终输出是（ ）**

A. a=20, b=10, c=20  
B. a=30, b=10, c=30  
C. a=30, b=10, c=20  
D. a=20, b=10, c=30

### 二、判断题

**4. 变量定义时如果没有初始化，其值是不确定的（垃圾值），直接使用是危险的。**  
（ ）正确 （ ）错误

**5. `int b = 10;` 是初始化，在定义时赋予初值；`a = 20;` 是赋值，给已存在的变量新值。**  
（ ）正确 （ ）错误

**6. `int c = a = 30;` 中，a=30的结果是30，然后将30赋值给c。**  
（ ）正确 （ ）错误

### 三、填空题

**7. 变量定义时第一次赋值称为______，之后改变值称为______。**

**8. `int x;` 后直接使用x的值是______（确定/不确定）的，这种错误称为使用______变量。**

**9. 良好的编程习惯：定义变量时应该立即______，避免______值。**

---

**解析关键点**：
- 只定义不初始化 = 垃圾值
- 初始化：定义时给初值
- 赋值：给已有变量新值
- a=30返回30，再赋给c

## 【大题2】连续赋值的"右结合性陷阱"

**考察点**：赋值运算符是右结合的，从右向左计算

```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b, c;
    
    a = b = c = 10;     // 语句1：连续赋值
    
    a = 5 + (b = 3);    // 语句2：赋值在表达式中
    
    int x, y, z;
    x = (y = (z = 1));  // 语句3：显式括号
    
    cout << "a=" << a << ", b=" << b << ", c=" << c << endl;
    cout << "x=" << x << ", y=" << y << ", z=" << z << endl;
    return 0;
}
```

### 一、选择题

**1. 语句1 `a = b = c = 10;` 的执行顺序是（ ）**

A. 从左到右：先a=b，再b=c，再c=10  
B. 从右到左：先c=10，再b=c，再a=b  
C. 同时执行  
D. 编译错误

**2. 语句2执行后，a和b的值分别是（ ）**

A. a=5, b=3  
B. a=8, b=3  
C. a=3, b=5  
D. a=5, b未变

**3. 语句2中 `(b = 3)` 的值是（ ）**

A. 0  
B. 1  
C. 3  
D. b

### 二、判断题

**4. 赋值运算符`=`是右结合的，所以`a = b = c`等价于`a = (b = c)`。**  
（ ）正确 （ ）错误

**5. 赋值表达式的值就是赋值后左边变量的值。**  
（ ）正确 （ ）错误

**6. `a = b = c = 10` 执行后，a、b、c的值都是10。**  
（ ）正确 （ ）错误

### 三、填空题

**7. 赋值运算符的结合性是______（左/右）结合。**

**8. `a = b = c = 5` 等价于 `a = ______(b = ______(c = 5))`（填写括号位置）。**

**9. 表达式 `a = (b = 2) + 3` 中，先执行______=2，结果是______，然后a=______。**

---

**解析关键点**：
- =是右结合：a=b=c 等价于 a=(b=(c=5))
- 赋值表达式返回赋值后左值
- (b=3)的值是3，所以a=5+3=8

## 【大题3】复合赋值运算符的"副作用陷阱"

**考察点**：+=、-=等复合赋值运算符的使用，以及与简单赋值的区别

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 10;
    
    a = a + 5;      // 语句1：简单赋值
    b += 5;         // 语句2：复合赋值
    
    int x = 3, y = 3;
    x = x * 2 + 1;  // 语句3
    y *= 2 + 1;     // 语句4：注意！不是y = y * 2 + 1
    
    cout << "a=" << a << ", b=" << b << endl;
    cout << "x=" << x << ", y=" << y << endl;
    return 0;
}
```

### 一、选择题

**1. 语句1和语句2的关系是（ ）**

A. 完全不同  
B. 效果相同，但复合赋值更高效  
C. 语句2是语句1的简写，效果相同  
D. 语句2有副作用

**2. 语句4执行后，y的值是（ ）**

A. 7  
B. 9  
C. 10  
D. 6

**3. `y *= 2 + 1` 等价于（ ）**

A. `y = y * 2 + 1`  
B. `y = y * (2 + 1)`  
C. `y = (y *= 2) + 1`  
D. `y = y + 2 + 1`

### 二、判断题

**4. `a += b` 等价于 `a = a + b`，但前者只计算一次a的地址，可能更高效。**  
（ ）正确 （ ）错误

**5. 复合赋值运算符的优先级比普通赋值运算符低。**  
（ ）正确 （ ）错误

**6. `y *= 2 + 1` 中，由于+=优先级低，实际等价于`y = y * (2 + 1)`。**  
（ ）正确 （ ）错误

### 三、填空题

**7. 复合赋值运算符`+=`、`-=`、`*=`、`/=`、`%=`的优先级比算术运算符______（高/低）。**

**8. `a += 3` 完全等价于 `a = ______ + 3`。**

**9. 如果a=5，执行`a *= a += 2`后，a的值是______（先算a+=2得7，再算a*=7）。**

---

**解析关键点**：
- a += b 等价于 a = a + b
- 但 += 优先级低于 +，所以 y *= 2+1 = y * (2+1) = 9
- 复合赋值只计算一次左值地址

## 【大题4】自增/自减与赋值的"混合陷阱"

**考察点**：++/--与赋值混合使用时的求值顺序陷阱

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 5, b;
    
    b = a++;    // 语句1：后置++

    a = 5;
    b = ++a;    // 语句2：前置++
    cout << "b=" << b << ", a=" << a << endl;
    
    a = 5;
    b = a++ + ++a;  // 语句3：危险！
    cout << "b=" << b << ", a=" << a << endl;
    
    return 0;
}
```

### 一、选择题

**1. 语句1执行后，b和a的值分别是（ ）**

A. b=5, a=5  
B. b=5, a=6  
C. b=6, a=6  
D. b=6, a=5

**2. 语句2执行后，b和a的值分别是（ ）**

A. b=5, a=5  
B. b=5, a=6  
C. b=6, a=6  
D. b=6, a=5

**3. 语句3的执行结果（ ）**

A. b=11, a=7  
B. b=12, a=7  
C. 不确定（未定义行为）  
D. 编译错误

### 二、判断题

**4. `b = a++` 中，先将a的值赋给b，然后a自增1。**  
（ ）正确 （ ）错误

**5. `b = ++a` 中，先将a自增1，然后将新值赋给b。**  
（ ）正确 （ ）错误

**6. 在同一个表达式中多次修改同一个变量（如`a++ + ++a`）是安全的。**  
（ ）正确 （ ）错误

### 三、填空题

**7. 后置++的优先级______（高于/低于）赋值运算符。**

**8. 表达式`b = a++`中，赋值给b的是a的______（原值/新值）。**

**9. 为了避免未定义行为，应该将`b = a++ + ++a`拆分成多条语句，每条语句只修改变量______次。**

---

**解析关键点**：
- a++：先用后增，b=5, a=6
- ++a：先增后用，b=6, a=6
- 同一表达式多次修改同一变量 = 未定义行为

## 【大题9】位运算与复合赋值的"低级陷阱"

**考察点**：位运算符的复合赋值形式，以及位运算的特殊用途

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 5;      // 二进制：101
    
    a <<= 1;        // 语句1：左移1位
    cout << "a=" << a << endl;
    
    a >>= 1;        // 语句2：右移1位
    cout << "a=" << a << endl;
    
    int flags = 0;
    flags |= 1;     // 语句3：设置第0位
    flags |= 4;     // 语句4：设置第2位
    cout << "flags=" << flags << endl;
    
    flags &= ~2;    // 语句5：清除第1位（即使它已经是0）
    cout << "flags=" << flags << endl;
    
    return 0;
}
```

### 一、选择题

**1. 语句1执行后，a的值是（ ）**

A. 5  
B. 10  
C. 6  
D. 2

**2. 语句3和4执行后，flags的值是（ ）**

A. 1  
B. 4  
C. 5  
D. 3

**3. `a <<= 1` 等价于（ ）**

A. `a = a * 2`  
B. `a = a + 1`  
C. `a = a << 1`  
D. A和C都对

### 二、判断题

**4. 左移1位（<<= 1）等价于乘以2，右移1位（>>= 1）等价于除以2（对正数）。**  
（ ）正确 （ ）错误

**5. `flags |= (1 << n)` 用于设置flags的第n位为1。**  
（ ）正确 （ ）错误

## 【大题10】变量生命周期的"作用域陷阱"

**考察点**：理解变量的作用域和生命周期

```cpp
#include <iostream>
using namespace std;

int global = 10;        // 全局变量

int main() {
    int local = 20;     // 局部变量
    
    {
        int block = 30;     // 块级变量
        static int s = 0;   // 静态局部变量

    }   // block在这里销毁
    
    // cout << block;     // 错误！block不可见
    
    {
        static int s = 100; // 另一个s，不同的变量

    cout << "global=" << global << ", local=" << local << endl;
    return 0;
}
```

**1. block变量的作用域是（ ）**

B. 1  
C. 随机值

A. 是同一个变量  
B. 是不同的变量，互不影响

D. 第二个s覆盖了第一个

**8. static局部变量的生命周期是______（块结束/程序结束），但作用域仍是块内部。**

## 核心要点总结

### 变量使用检查清单

- [ ] 定义变量时立即初始化，避免垃圾值
- [ ] 理解初始化（定义时）vs 赋值（定义后）的区别
- [ ] 赋值运算符=是右结合的
- [ ] 复合赋值运算符（+=等）优先级低于算术运算符
- [ ] 同一表达式中不要多次修改同一变量
- [ ] 浮点转整数会截断小数，不是四舍五入
- [ ] 大范围转小范围可能溢出
- [ ] const常量必须初始化且不可修改
- [ ] 区分=（赋值）和==（比较）
- [ ] 理解变量的作用域和生命周期

### 常见错误对比

| 错误写法 | 问题 | 正确写法 |
|:---|:---|:---|
| int a; cout << a; | 使用未初始化变量 | int a = 0; |
| a = b = c = 5 | 右结合，注意类型转换 | 保持类型一致 |
| y *= 2 + 1 | 等价于y = y * (2+1) | 明确优先级 |
| a++ + ++a | 未定义行为 | 拆成多条语句 |
| if (x = 0) | 赋值而非比较 | if (x == 0) |
| int i = 3.7 | 隐式截断 | int i = (int)3.7; |
