## 第一部分 GESP考试大纲概述

GESP（Grade Examination of Software Programming）是由中国计算机学会（CCF）组织的编程能力等级认证考试，分为1-8级。对于4-5年级的初学者，建议从一级开始学习，逐步掌握二级内容。

### 一、GESP一级考试大纲

**1. 计算机基础知识** - 计算机基本构成、操作系统、网络基础
**2. 编程环境** - Dev-C++ IDE、编译与运行
**3. 基本数据类型** - 整型、实型、字符型、布尔型
**4. 变量与常量** - 定义、初始化、const关键字
**5. 基本运算** - 算术、关系、逻辑、赋值运算符
**6. 输入输出** - cin/cout、scanf/printf
**7. 程序结构** - 顺序、分支(if-else)、循环(for/while)

### 二、GESP二级考试大纲

**1. 计算机存储与网络** - RAM/ROM/缓存、IP/域名/协议
**2. 程序设计语言** - 机器/汇编/高级语言、编译/解释型
**3. 流程图** - 基本符号、读懂流程图
**4. ASCII编码** - 常用字符ASCII码、大小写转换
**5. 数据类型转换** - 隐式转换、强制类型转换
**6. 多层分支结构** - if-else if-else、switch
**7. 多层循环结构** - 嵌套循环、二维图形
**8. 数学函数** - abs()、sqrt()、pow()等
