## 📚 知识探索

### 🔍 探索1：从变量到地址

**回顾L1**：变量就像一个「盒子」，里面存着值。

```cpp
int age = 15;  // 盒子里装着15
```

**新问题**：这个盒子在房间的哪个位置？

**答案**：每个变量在内存中都有一个**地址**（门牌号）！

```cpp
cout << &age << endl;  // & = 取地址，输出类似 0x7ffee...
```

**类比**：
```
变量 = 快递柜里的包裹
值 = 包裹里的东西（15）
地址 = 快递柜编号（A-103）
& = 看柜子的编号
```

---

### 🔍 探索2：指针——存储地址的变量

**指针**：专门用来存「地址」的变量。

```cpp
int age = 15;        // 普通变量，存整数
int *p = &age;       // 指针变量，存age的地址

// 指针的"两面性"
cout << p << endl;    // 输出地址（0x7ffee...）
cout << *p << endl;   // *p = 解引用，输出15
```

**符号理解**：
| 符号 | 读作 | 作用 |
|:---|:---|:---|
| `int *p` | 整型指针 | 声明p是指针 |
| `&age` | age的地址 | 取地址 |
| `*p` | p指向的值 | 解引用 |

**生活类比**：
```
age = 家里的保险柜，装着15万
p = 一张纸条，上面写着保险柜的位置
*p = 根据纸条找到保险柜，取出15万
```

---

### 📝 填空练习

**指针的基本操作**：
```cpp
int x = 10;
int *p = ____;      // 填空：让p指向x

// 通过指针修改值
____ = 20;          // 填空：把x改成20（用指针）

cout << x << endl;   // 输出？
```

<details>
<summary>✅ 答案对照</summary>

```cpp
int x = 10;
int *p = &x;        // p存x的地址
*p = 20;            // *p就是x，x变成20
cout << x << endl;   // 输出20
```
</details>

### 🔍 探索3：指针与函数——真正修改原变量

**回顾L4**：函数参数是"复制"，修改不影响原变量。

**新问题**：如果就想在函数里修改外面的变量呢？

✅ **答案**：传指针！

```cpp
void addTen(int *p) {    // 参数是指针
    *p = *p + 10;        // 修改指针指向的值
}

int main() {
    int num = 10;
    addTen(&num);         // 把num的地址传过去
    cout << num << endl;   // 输出20！真的被修改了！
    return 0;
}
```

**为什么这次成功了？**
- 之前：传的是"复制品"（值传递）
- 现在：传的是"地址"（指针传递）
- 通过地址，函数能找到"原件"并修改

### 🔍 探索4：结构体——打包相关数据

**场景**：存储一个学生的信息

❌ **笨方法**：分散的变量
```cpp
string name;    // 名字放这里
int age;        // 年龄放这里
int score;      // 成绩放那里
// 乱！而且容易丢！
```

✅ **聪明方法**：结构体——把相关数据**打包**！

```cpp
struct Student {        // 定义一个"学生"结构体
    string name;        // 成员：名字
    int age;            // 成员：年龄
    int score;          // 成员：成绩
};                      // 注意分号！

// 使用结构体
Student stu;            // 创建一个"学生"变量
stu.name = "小明";       // 用.访问成员
stu.age = 15;
stu.score = 90;

cout << stu.name << "考了" << stu.score << "分" << endl;
```

**类比**：
```
结构体 = 一个文件夹
成员 = 文件夹里的文件
. = 打开文件夹取文件
```

---

### 🔍 探索5：结构体数组——多个学生的信息

```cpp
Student stu[50];        // 50个学生的信息

// 输入第1个学生的信息
cin >> stu[0].name >> stu[0].age >> stu[0].score;

// 遍历所有学生
for (int i = 0; i < 50; i++) {
    cout << stu[i].name << " " << stu[i].score << endl;
}
```

### 🟢 练习1：指针基础（基础）

**要求**：定义变量 `int a = 5, b = 10`，用指针交换a和b的值。

**提示**：需要临时变量

<details>
<summary>✅ 参考答案</summary>

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 5, b = 10;
    int *pa = &a, *pb = &b;
    int temp;
    
    // 通过指针交换
    temp = *pa;
    *pa = *pb;
    *pb = temp;
    
    cout << "a=" << a << " b=" << b << endl;  // a=10 b=5
    return 0;
}
```
</details>

### 🟡 练习2：结构体与函数（进阶）

**要求**：定义`Student`结构体，编写函数`printStudent`打印学生信息。

**分解**：
1. 定义结构体
2. 编写void函数，参数是Student
3. 在函数里打印信息
4. main里创建学生，调用函数

<details>
<summary>✅ 参考答案</summary>

```cpp
#include <iostream>
#include <string>
using namespace std;

struct Student {
    string name;
    int age;
    int score;
};

void printStudent(Student s) {
    cout << "姓名：" << s.name << endl;
    cout << "年龄：" << s.age << endl;
    cout << "成绩：" << s.score << endl;
}

int main() {
    Student stu = {"小明", 15, 90};
    printStudent(stu);
    return 0;
}
```
</details>

### 🔴 练习3：链表初步（挑战）

**链表**：用指针连接的结构体

```cpp
struct Node {
    int data;       // 存数据
    Node *next;     // 指向下一个节点
};
```

**要求**：创建3个节点，连成链表 1 → 2 → 3，然后遍历打印。

**图示**：
```
[1|*]-->[2|*]-->[3|null]
```

<details>
<summary>✅ 参考答案</summary>

```cpp
#include <iostream>
using namespace std;

struct Node {
    int data;
    Node *next;
};

int main() {
    // 创建3个节点
    Node n1 = {1, nullptr};
    Node n2 = {2, nullptr};
    Node n3 = {3, nullptr};
    
    // 连接节点
    n1.next = &n2;
    n2.next = &n3;
    
    // 遍历链表
    Node *p = &n1;
    while (p != nullptr) {
        cout << p->data << " ";
        p = p->next;
    }
    
    return 0;
}
```
</details>

## 📚 知识速查表

| 概念 | 语法 | 示例 |
|:---|:---|:---|
| 取地址 | `&变量` | `int *p = &x` |
| 解引用 | `*指针` | `*p = 20` |
| 指针声明 | `类型 *名字` | `int *p` |
| 结构体定义 | `struct 名字 { };` | `struct Student { };` |
| 访问成员 | `变量.成员` | `stu.name` |
| 指针访问 | `指针->成员` | `p->name` |
| 空指针 | `nullptr` | `p = nullptr` |

**指针口诀**：
- `&`是取地址，`*`是取值
- `int *p`声明指针，`*p`使用指针
- 指针就是门牌号，`*p`就是房间里的东西
