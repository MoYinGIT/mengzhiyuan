## 📚 知识探索

### 🔍 探索1：栈——后进先出

**生活场景**：一叠盘子
- 放盘子：只能放在最上面
- 拿盘子：只能拿最上面的
- **后放的先拿**！

**栈(Stack)**：后进先出（LIFO - Last In First Out）

**基本操作**：
| 操作 | 作用 | 类比 |
|:---|:---|:---|
| `push(x)` | 把x放到栈顶 | 放盘子 |
| `pop()` | 移除栈顶元素 | 拿盘子 |
| `top()` | 看栈顶元素（不移除）| 看最上面的盘子 |
| `empty()` | 检查栈是否为空 | 检查还有没有盘子 |

**代码实现**：
```cpp
struct Stack {
    int data[100];  // 存数据
    int top;        // 栈顶位置
    
    void init() {   // 初始化
        top = -1;   // -1表示空栈
    }
    
    void push(int x) {
        top++;
        data[top] = x;
    }
    
    void pop() {
        top--;  // 只是移动指针，数据还在（但不访问）
    }
    
    int getTop() {
        return data[top];
    }
    
    bool empty() {
        return top == -1;
    }
};
```

### 🔍 探索2：队列——先进先出

**生活场景**：排队买票
- 新来的人：排到队尾
- 服务完成的人：从队首离开
- **先来的先走**！

**队列(Queue)**：先进先出（FIFO - First In First Out）

**基本操作**：
| 操作 | 作用 | 类比 |
|:---|:---|:---|
| `push(x)` | 把x放到队尾 | 排队 |
| `pop()` | 移除队首元素 | 服务完成离开 |
| `front()` | 看队首元素 | 看下一个要服务的人 |
| `empty()` | 检查队列是否为空 | 检查还有没有人 |

**代码实现**：
```cpp
struct Queue {
    int data[100];
    int front;  // 队首
    int rear;   // 队尾
    
    void init() {
        front = 0;
        rear = 0;
    }
    
    void push(int x) {
        data[rear] = x;
        rear++;
    }
    
    void pop() {
        front++;  // 队首后移，相当于移除
    }
    
    int getFront() {
        return data[front];
    }
    
    bool empty() {
        return front == rear;
    }
};
```

### 📝 填空练习

**用栈判断回文**：

回文：正着读反着读都一样，如"abcba"

**思路**：
1. 前半部分入栈
2. 后半部分和栈顶比较

```cpp
bool isPalindrome(string s) {
    Stack st;
    st.init();
    int n = s.length();
    
    // 前半部分入栈
    for (int i = 0; i < n/2; i++) {
        st.push(s[____]);  // 填空
    }
    
    // 后半部分比较
    for (int i = (n+1)/2; i < n; i++) {
        if (s[i] != st.____()) return false;  // 填空
        st.____();  // 填空：弹出
    }
    return true;
}
```

<details>
<summary>✅ 答案对照</summary>

```cpp
// 前半部分入栈
for (int i = 0; i < n/2; i++) {
    st.push(s[i]);
}

// 后半部分比较
for (int i = (n+1)/2; i < n; i++) {
    if (s[i] != st.getTop()) return false;
    st.pop();
}
```
</details>

### 🔍 探索3：二叉树——层次结构

**生活场景**：家族谱系
```
        爷爷
       /    \
     爸爸    叔叔
    /  \
  我   妹妹
```

**二叉树节点**：
```cpp
struct TreeNode {
    int data;           // 存的数据
    TreeNode *left;     // 左孩子
    TreeNode *right;    // 右孩子
};
```

**遍历方式**：

| 遍历 | 顺序 | 示例（对上面树）|
|:---|:---|:---|
| 前序 | 根-左-右 | 爷爷-爸爸-我-妹妹-叔叔 |
| 中序 | 左-根-右 | 我-爸爸-妹妹-爷爷-叔叔 |
| 后序 | 左-右-根 | 我-妹妹-爸爸-叔叔-爷爷 |

**前序遍历代码**：
```cpp
void preOrder(TreeNode *root) {
    if (root == nullptr) return;  // 空树返回
    
    cout << root->data << " ";     // 访问根
    preOrder(root->left);         // 遍历左子树
    preOrder(root->right);        // 遍历右子树
}
```

### 🔍 探索4：二叉搜索树——快速查找

**性质**：
- 左子树所有值 < 根
- 右子树所有值 > 根
- 左右子树也是二叉搜索树

**示例**：
```
      8
     / \
    3   10
   / \    \
  1   6    14
     / \   /
    4   7 13
```

**查找过程**（找6）：
1. 从8开始，6<8，去左子树
2. 到3，6>3，去右子树
3. 到6，找到了！

**查找效率**：O(树的高度)，最坏O(n)，最好O(log n)

### 🟢 练习1：括号匹配（基础）

**要求**：判断一个字符串的括号是否匹配。

**示例**：
- "()" → 匹配 ✓
- "({[]})" → 匹配 ✓
- "([)]" → 不匹配 ✗
- "((()" → 不匹配 ✗

**思路**：
- 左括号入栈
- 右括号与栈顶匹配，匹配则弹出，不匹配则错误

<details>
<summary>✅ 参考答案</summary>

```cpp
bool isValid(string s) {
    Stack st;
    st.init();
    
    for (int i = 0; i < s.length(); i++) {
        if (s[i] == '(' || s[i] == '[' || s[i] == '{') {
            st.push(s[i]);
        } else {
            if (st.empty()) return false;
            char top = st.getTop();
            if ((s[i] == ')' && top == '(') ||
                (s[i] == ']' && top == '[') ||
                (s[i] == '}' && top == '{')) {
                st.pop();
            } else {
                return false;
            }
        }
    }
    return st.empty();
}
```
</details>

### 🟡 练习2：层序遍历（进阶）

**要求**：按层次打印二叉树（从上到下，从左到右）。

**示例**：
```
    1
   / \
  2   3
 / \
4   5
```
输出：1 2 3 4 5

**思路**：用队列！
1. 根入队
2. 出队一个，访问，左右孩子入队
3. 重复直到队列为空

<details>
<summary>✅ 参考答案</summary>

```cpp
void levelOrder(TreeNode *root) {
    Queue q;
    q.init();
    q.push(root);
    
    while (!q.empty()) {
        TreeNode *node = q.getFront();
        q.pop();
        cout << node->data << " ";
        
        if (node->left) q.push(node->left);
        if (node->right) q.push(node->right);
    }
}
```
</details>

### 🔴 练习3：二叉搜索树插入（挑战）

**要求**：实现BST的插入操作。

**思路**：
- 如果树为空，新节点作为根
- 如果值 < 根，插入左子树
- 如果值 > 根，插入右子树
- 如果值 = 根，不插入（或更新）

<details>
<summary>✅ 参考答案</summary>

```cpp
TreeNode* insert(TreeNode *root, int val) {
    if (root == nullptr) {
        TreeNode *newNode = new TreeNode();
        newNode->data = val;
        newNode->left = newNode->right = nullptr;
        return newNode;
    }
    
    if (val < root->data) {
        root->left = insert(root->left, val);
    } else if (val > root->data) {
        root->right = insert(root->right, val);
    }
    
    return root;
}
```
</details>
