## 📚 知识探索

### 🔍 探索1：从1个盒子到100个盒子

**回顾L1**：我们学过变量是"盒子"
```cpp
int age = 15;  // 一个盒子，装着年龄
```

**新问题**：如果要存储全班50个同学的年龄，怎么办？

❌ **笨方法**：创建50个变量
```cpp
int age1, age2, age3, age4, age5... // 写到手酸！
```

**想一想**：这些盒子有什么共同点？
- 都是存"年龄"
- 都是整数
- 名字有规律（age1, age2...）

✅ **聪明方法**：用**数组**——一排整齐的盒子！

---

### 🔍 探索2：创建你的第一个数组

**语法**：
```cpp
int ages[5];  // 创建5个整数盒子，名字叫ages
```

**生活类比**：
```
普通变量：一个快递柜格子
数组：一整排快递柜（编号0, 1, 2, 3, 4）
```

**给盒子里放东西**：
```cpp
ages[0] = 15;  // 0号盒子放15
ages[1] = 16;  // 1号盒子放16
ages[2] = 14;  // 2号盒子放14
ages[3] = 15;  // 3号盒子放15
ages[4] = 16;  // 4号盒子放16
```

**⚠️ 重要提醒**：数组从 **0** 开始计数！
- 第1个元素是 `ages[0]`
- 第5个元素是 `ages[4]`
- **没有** `ages[5]`！（这是常见错误）

### 📝 填空练习

**创建并初始化数组**：
```cpp
int scores[___] = {85, 90, 78, 92, 88};  // 填空：有几个数？

cout << "第一个成绩：" << scores[___] << endl;  // 填空：第1个的下标？

int scores[5] = {85, 90, 78, 92, 88};  // 5个数
cout << "第一个成绩：" << scores[0] << endl;  // 下标0
cout << "最后一个成绩：" << scores[4] << endl;  // 下标4
```
</details>

### 🔍 探索3：遍历数组——用循环访问每个盒子

**问题**：打印数组中所有成绩

❌ **笨方法**：
```cpp
cout << scores[0] << endl;
cout << scores[1] << endl;
cout << scores[2] << endl;  // 要写5行！
```

✅ **聪明方法**：用 `for` 循环！

```cpp
for (int i = 0; i < 5; i++) {
    cout << "第" << i+1 << "个成绩：" << scores[i] << endl;
}
```

**发现规律了吗？**
- `i` 从 0 到 4（不是5！）
- `scores[i]` 就是第 i 个元素
- 循环5次，访问5个元素

### 📝 填空练习

**计算数组元素的和**：
```cpp
int scores[5] = {85, 90, 78, 92, 88};
int sum = ____;  // 填空：初始值应该是？

for (int i = 0; i < 5; i++) {
    sum = sum ____ scores[i];  // 填空：怎么累加？
}

cout << "总分：" << sum << endl;
cout << "平均分：" << sum / ____ << endl;  // 填空：除以几？
```

<details>
<summary>✅ 答案对照</summary>

```cpp
int sum = 0;  // 初始为0，空盒子等着加
for (int i = 0; i < 5; i++) {
    sum = sum + scores[i];  // 累加
}
cout << "平均分：" << sum / 5 << endl;  // 除以5
```
</details>

### 🔍 探索4：找最大最小值

**问题**：找出最高分

**算法思路**：
1. 先假设第1个成绩是最高
2. 依次和第2、3、4、5个比较
3. 如果发现更大的，就更新最高分

```cpp
int maxScore = scores[0];  // 先假设第1个最大

for (int i = 1; i < 5; i++) {
    if (scores[i] > maxScore) {
        maxScore = scores[i];  // 发现更大的，更新！
    }
}

cout << "最高分：" << maxScore << endl;
```

**想一想**：找最小值怎么改？

<details>
<summary>💡 提示</summary>

把 `>` 改成 `<`，把 `maxScore` 改成 `minScore`。
</details>

### 🟡 练习2：成绩统计（进阶）

**要求**：输入5个成绩，计算：
1. 总分和平均分
2. 最高分和最低分
3. 及格人数（≥60分）

**分解**：
- Step 1：输入5个成绩到数组
- Step 2：遍历数组，计算总分、最大最小值、及格人数
- Step 3：输出结果

<details>
<summary>✅ 参考答案</summary>

```cpp
#include <iostream>
using namespace std;

int main() {
    int scores[5];
    int sum = 0, maxScore, minScore, passCount = 0;
    
    // 输入
    for (int i = 0; i < 5; i++) {
        cin >> scores[i];
    }
    
    // 初始化最大最小值
    maxScore = minScore = scores[0];
    
    // 统计
    for (int i = 0; i < 5; i++) {
        sum += scores[i];
        if (scores[i] > maxScore) maxScore = scores[i];
        if (scores[i] < minScore) minScore = scores[i];
        if (scores[i] >= 60) passCount++;
    }
    
    cout << "总分：" << sum << endl;
    cout << "平均分：" << sum / 5.0 << endl;
    cout << "最高分：" << maxScore << endl;
    cout << "最低分：" << minScore << endl;
    cout << "及格人数：" << passCount << endl;
    
    return 0;
}
```
</details>

### 🔴 练习3：数组逆序（挑战）

**要求**：输入5个数，倒过来输出。

**示例**：
- 输入：`1 2 3 4 5`
- 输出：`5 4 3 2 1`

**提示**：
- 第1个和最后1个交换
- 第2个和倒数第2个交换
- ...

**想一想**：交换两个变量需要几步？

<details>
<summary>💡 交换提示</summary>

交换a和b需要临时变量：
```cpp
int temp = a;
a = b;
b = temp;
```
</details>

<details>
<summary>✅ 参考答案</summary>

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[5];
    
    // 输入
    for (int i = 0; i < 5; i++) {
        cin >> arr[i];
    }
    
    // 逆序：交换第i个和第(4-i)个
    for (int i = 0; i < 2; i++) {
        int temp = arr[i];
        arr[i] = arr[4 - i];
        arr[4 - i] = temp;
    }
    
    // 输出
    for (int i = 0; i < 5; i++) {
        cout << arr[i] << " ";
    }
    
    return 0;
}
```
</details>

| 错误 | 现象 | 解决 |
|:---|:---|:---|
| 数组越界 | `ages[5] = 10`（数组只有0-4）| 记住最大下标是`数组大小-1` |
| 未初始化就用 | 输出乱码 | 先赋值再使用 |
| 循环边界错误 | `i <= 5` 导致越界 | 用 `i < 5` |

## 📚 知识速查表

| 概念 | 语法 | 示例 |
|:---|:---|:---|
| 定义数组 | `类型 名字[大小]` | `int arr[10]` |
| 初始化 | `{值1, 值2...}` | `int arr[3] = {1,2,3}` |
| 访问元素 | `名字[下标]` | `arr[0]` |
| 遍历数组 | `for(i=0; i<n; i++)` | `arr[i]` |
| 数组大小 | 编译时确定 | 不能用变量定义大小 |

**重要提醒**：
- 下标从 **0** 开始！
- 最大下标 = **数组大小 - 1**
- 数组越界会导致程序崩溃！
