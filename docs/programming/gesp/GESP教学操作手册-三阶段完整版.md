# GESP 1-3级 三阶段教学操作手册

## 教学目标分层

| 阶段 | 时间节点 | 目标学员 | 目标级别 | 分数要求 |
|:---:|:---:|:---|:---:|:---:|
| **暑假班** | 9月考级 | 集训学员 | GESP L1 | 90分+ |
| **秋季班** | 12月考级 | 常规学员 | GESP L1 | 60分+及格 |
| **寒假班** | 3月考级 | 暑假/秋季学员 | L1全员+L3部分 | L1:60+ L3:60+ |

## 第一阶段：暑假集训（9月考级冲刺）

### 时间规划

```
7月第1周 ~ 8月第4周 = 8周
每周6天（周一至周六），每天3小时
总时长：8周 × 6天 × 3h = 144小时

特殊安排：
- 第4周末：第一次全真模拟
- 第6周末：第二次全真模拟  
- 第8周：考前冲刺（每天4小时）
```

### 每日3小时详细安排

```
08:30-08:45  签到+昨日作业讲评（15分钟）
08:45-09:45  新知授课（60分钟）
09:45-10:00  课间休息
10:00-11:00  例题精讲+学生跟练（60分钟）
11:00-12:00  课堂练习+老师巡视（60分钟）
12:00-12:15  当堂作业布置+答疑

作业要求：
- 每天3-5道编程题
- 22:00前提交在线评测系统
- 次日课前讲评共性错误
```

### 暑假班90分+保障措施

| 措施 | 具体做法 |
|:---|:---|
| **入学测试** | 数学基础测试，低于80分建议先补数学 |
| **每日通关** | 当天内容当天掌握，未完成留堂30分钟 |
| **分层教学** | 前20%学员增加拓展题，后20%单独辅导 |
| **双师配置** | 主讲+助教，确保一对一答疑时间 |
| **家校联动** | 每日作业打卡，缺交当天通知家长 |
| **真题驱动** | 第4周起每天接触真题，熟悉套路 |
| **错题本** | 强制整理，每周检查，考前重点复习 |

### 暑假班结营标准

```
✅ 通过标准：
   - 出勤率 ≥ 90%（缺勤≤5天）
   - 作业完成率 ≥ 90%
   - 模拟考平均分 ≥ 90分
   - 真题完成量 ≥ 20套

⚠️ 预警标准：
   - 模拟考 < 80分 → 一对一补弱
   - 作业缺交 ≥ 3次 → 家长约谈
   - 学习态度问题 → 劝退或转班
```

## 第二阶段：秋季学期（12月考级）

### 时间规划

```
9月第1周 ~ 12月第4周 = 16周
每周2次（周六+周日），每次2小时
总时长：16周 × 2次 × 2h = 64小时

特殊节点：
- 9月第1-2周：暑假结课学员参加9月考级
- 9月第3周：秋季班正式开始
- 11月第3-4周：12月考级报名
- 12月第2-3周：GESP考试
```

### 学员分层

| 层次 | 来源 | 基础 | 目标 |
|:---:|:---|:---:|:---|
| **A层（提高）** | 暑假班学员 | 已掌握L1 | L2冲刺或L1高分 |
| **B层（标准）** | 新生/基础弱 | 零基础 | L1通过（60分+） |
| **C层（补差）** | 暑假未达标 | 需巩固 | L1通过 |

### 分层教学安排

#### A层（暑假班延续）：L2冲刺

| 时间段 | 内容 | 产出 |
|:---:|:---|:---|
| 9月 | L2前置知识强化 | 存储/网络/进制/ASCII |
| 10月 | 多层结构+数学函数 | 嵌套结构熟练 |
| 11月 | 数组+字符串入门 | 一维数组操作 |
| 12月 | 考前冲刺 | L2模拟考 |

#### B层（新生标准班）：L1系统学习

| 周次 | 内容 | 说明 |
|:---:|:---|:---|
| 1-2 | 环境+变量+顺序 | 比暑假慢，稳扎稳打 |
| 3-4 | 分支结构 | 多练习，确保理解 |
| 5-7 | 循环结构 | 重点突破 |
| 8-9 | 期中复习+模拟 | 检测学习效果 |
| 10-12 | L1真题训练 | 每周2套真题 |
| 13-14 | 易错点专项 | 个性化补弱 |
| 15-16 | 考前冲刺 | 全真模拟 |

#### C层（补差班）：L1巩固

- 小班制（≤6人）
- 重复暑假内容但更细致
- 更多一对一时间
- 降低进度，确保质量

### 秋季班关键控制点

| 时间点 | 检查项 | 标准 |
|:---:|:---|:---:|
| 第4周末 | 期中测试 | 平均分70+，低于60单独辅导 |
| 第8周末 | 知识点全覆盖检查 | 每个学生薄弱环节清单 |
| 第12周 | 真题模拟考 | 平均分75+，低于60预警 |
| 第14周 | 报名确认 | 确认所有学生完成报名 |
| 第15-16周 | 冲刺集训 | 每天1套真题 |

## 第三阶段：寒假分层（3月考级）

### 时间规划

```
1月第2周 ~ 2月第3周 = 5-6周
每天3小时（延续暑假模式）
总时长：30-35天 × 3h = 90-105小时

分层开班：
- L1保过班：未通过L1的学员
- L1高分班：已通L1，冲高分
- L3冲刺班：已通L1，冲L3
```

### 寒假班型设置

#### 班型A：L1保过班（目标：3月考级全员通过）

| 周次 | 内容 | 策略 |
|:---:|:---|:---|
| Week1 | 薄弱点诊断+专项补弱 | 个性化方案 |
| Week2 | 分支循环强化 | 重点突破 |
| Week3 | 真题套卷×8套 | 考试化训练 |
| Week4 | 易错点+陷阱题 | 防失分 |
| Week5 | 考前冲刺 | 全真模拟 |
| Week6 | 冲刺+考试 | 保持状态 |

**入班标准**：
- 暑假/秋季学员但未通过L1
- 或其他机构转来的补考生

**保过承诺**：
- 未通过3月考级，免费重读至通过

#### 班型B：L1高分班（目标：95分+）

| 周次 | 内容 |
|:---|:---|
| Week1 | L2前置知识（存储/网络/进制）|
| Week2 | L2前置知识（ASCII/类型转换）|
| Week3 | 多层结构+数学函数 |
| Week4 | 数组入门 |
| Week5 | 综合练习 |
| Week6 | L2考级（同步冲L2）|

**入班标准**：
- 暑假班学员且L1已通过或模拟90分+
- 秋季A层学员

#### 班型C：L3冲刺班（目标：3月考级通过L3）

| 周次 | 内容 | 说明 |
|:---:|:---|:---|
| Week1 | 数组+字符串 | C++一维数组、Python列表字典 |
| Week2 | 位运算+进制 | 位运算应用 |
| Week3 | 枚举+模拟 | 算法思想 |
| Week4 | 真题训练×6套 | L3真题 |
| Week5 | 易错点+综合 | 难点突破 |
| Week6 | 考前冲刺 | 保持手感 |

**入班标准**：
- L1高分通过（90分+）
- 数学基础好（奥数经历优先）
- 学习能力强

### 寒假后目标达成路径

```
暑假班学员（假设40人）
    │
    ├─→ 9月L1通过35人（90分+）────┐
    │                              │
    └─→ 9月未通过5人 ──────────────┤
                                  ↓
                          寒假分班：
                          ├─ L1保过班：5人（9月未过）+ 秋季未过学员
                          ├─ L1高分班：20人（冲L2）
                          └─ L3冲刺班：15人（冲L3）
                                  ↓
                          3月考级目标：
                          ├─ L1保过班：全员通过L1
                          ├─ L1高分班：全员通过L2
                          └─ L3冲刺班：80%通过L3
```

## 六、老师操作Checklist

### 暑假班每日Checklist

- [ ] 08:30 签到，确认出勤
- [ ] 08:45 开始授课，点名
- [ ] 09:45 课间休息，答疑
- [ ] 10:00 继续授课/练习
- [ ] 12:00 布置作业，确认理解
- [ ] 22:00 检查作业提交，记录缺交
- [ ] 次日08:30 课前讲评昨日作业

### 周次Checklist

- [ ] 每周六：周测（真题模拟）
- [ ] 每周日：试卷讲评+错题整理
- [ ] 每周日：更新每个学生的学习档案
- [ ] 每周日晚：发送周报给家长

### 阶段Checklist

- [ ] 第4周末：阶段评估，分层调整
- [ ] 第6周末：模拟考成绩分析
- [ ] 第8周末：结营评估，秋季分班建议
- [ ] 9月考前：报名确认，设备检查
- [ ] 12月考前：同上
- [ ] 3月考前：同上

**核心原则**：
> **暑假抓速度，秋季抓巩固，寒假抓分层**

> **数据驱动，每周评估，及时调整**
