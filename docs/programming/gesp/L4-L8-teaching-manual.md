## L4 算法基础（40小时）

### 第1单元：字符串处理（8小时）

#### 课时1-2：字符串基础操作
- **知识点**：
  - char数组与string类
  - 字符串输入输出（cin/getline）
  - 长度计算、字符访问
- **示例代码**：
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s;
    getline(cin, s);
    cout << "长度: " << s.length() << endl;
    cout << "首字符: " << s[0] << endl;
    cout << "末字符: " << s[s.length()-1] << endl;
    return 0;
}
```
- **练习题**：P5733 文字处理软件

#### 课时3-4：字符串遍历与查找
- **知识点**：
  - 遍历每个字符
  - find() / substr() 用法
  - 字符统计（字母、数字、空格）
- **示例代码**：
```cpp
// 统计各类字符数量
int alpha = 0, digit = 0, space = 0;
for (int i = 0; i < s.length(); i++) {
    if (isalpha(s[i])) alpha++;
    else if (isdigit(s[i])) digit++;
    else if (s[i] == ' ') space++;
}
```

#### 课时5-6：字符串转换与处理
- **知识点**：
  - char与int互转
  - stoi / to_string
  - 大小写转换
- **经典题型**：ISBN校验码计算

#### 课时7-8：字符串综合练习
- **综合题**：
  - P1055 ISBN号码
  - P5734 文字处理软件
  - P5735 名片整理

### 第3单元：函数进阶（8小时）

#### 课时17-18：函数参数传递
- **知识点**：
  - 值传递 vs 引用传递
  - 数组作为参数
  - 默认参数值
- **示例代码**：
```cpp
// 值传递（不改变原值）
void addOne(int x) { x++; }

// 引用传递（改变原值）
void addOneRef(int &x) { x++; }

// 数组参数

    sort(arr, arr + n);
}
```

#### 课时19-20：递归函数入门
- **知识点**：
  - 递归三要素：终止条件、递归关系、返回值
  - 阶乘、斐波那契数列
  - 递归与迭代的对比
- **示例代码**：
```cpp
// 阶乘
int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}

// 斐波那契
int fib(int n) {
    if (n <= 1) return n;
    return fib(n-1) + fib(n-2);
}
```

## L5 算法进阶（50小时）

### 第5单元：数论基础（15小时）

#### 课时41-44：质数与筛法
- **知识点**：
  - 质数判定（试除法）
  - 埃氏筛、欧拉筛
  - 质因数分解
- **示例代码**：
```cpp
// 埃氏筛
bool isPrime[1000005];
void sieve(int n) {
    memset(isPrime, true, sizeof(isPrime));
    isPrime[0] = isPrime[1] = false;
    for (int i = 2; i * i <= n; i++) {
        if (isPrime[i]) {
            for (int j = i * i; j <= n; j += i) {
                isPrime[j] = false;
            }
        }
    }
}
```

#### 课时45-48：GCD与LCM
- **知识点**：
  - 辗转相除法
  - 更相减损术
  - 扩展欧几里得
- **示例代码**：
```cpp
int gcd(int a, int b) {
    return b == 0 ? a : gcd(b, a % b);
}

int lcm(int a, int b) {
    return a / gcd(a, b) * b;
}
```

#### 课时49-52：同余与模运算

## L6 动态规划入门（60小时）

### 第8单元：DP基础（20小时）

#### 课时91-95：DP概念
- **知识点**：
  - 最优子结构
  - 重叠子问题
  - 状态与状态转移
- **核心公式**：
```
dp[i] = max/min(dp[i-1], dp[i-2] + value)
```

#### 课时96-100：线性DP
- **经典题型**：
  - 爬楼梯（最小代价）
  - 打家劫舍
  - 最大子段和
- **状态设计**：
```cpp
// 爬楼梯最小代价
dp[i] = min(dp[i-1] + cost[i], dp[i-2] + cost[i]);

// 最大子段和
dp[i] = max(dp[i-1] + a[i], a[i]);
```

#### 课时101-105：背包问题入门
- **知识点**：
  - 01背包
  - 完全背包
  - 状态转移方程
- **示例代码**：
```cpp
// 01背包
dp[j] = max(dp[j], dp[j-w[i]] + v[i]);

// 完全背包（顺序遍历）
for (int j = w[i]; j <= W; j++) {
    dp[j] = max(dp[j], dp[j-w[i]] + v[i]);
}
```

- **示例代码**：
```cpp
// Dijkstra + priority_queue
priority_queue<pair<int,int>, vector<pair<int,int>>, greater<>> pq;
dist[1] = 0;
pq.push({0, 1});
while (!pq.empty()) {
    auto [d, u] = pq.top(); pq.pop();
    if (d > dist[u]) continue;
    for (auto [v, w] : adj[u]) {
        if (dist[u] + w < dist[v]) {
            dist[v] = dist[u] + w;
            pq.push({dist[v], v});
        }
    }
}
```
