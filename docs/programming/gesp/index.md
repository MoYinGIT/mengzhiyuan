## 🎯 级别速查

### L1-L2：编程启蒙
- **目标**：掌握 C++ 基础语法
- **核心内容**：变量、分支、循环、数组
- **时长**：约 60 小时

### L3-L4：基础夯实
- **目标**：掌握函数、结构体、基础算法
- **核心内容**：函数、字符串、结构体、排序
- **时长**：约 80 小时

### L5-L6：算法进阶
- **目标**：掌握数论、递归、动态规划
- **核心内容**：数论、链表、递归、DP入门
- **时长**：约 100 小时

### L7-L8：拔尖提升
- **目标**：掌握高级算法，冲击竞赛
- **核心内容**：复杂DP、图论、组合数学
- **时长**：约 120 小时

### 编程教学五步法

1. **观** - 理解问题（读题画图）
2. **思** - 设计方案（多种解法对比）
3. **行** - 编写代码（模块化设计）
4. **省** - 调试优化（从错误中学习）
5. **传** - 教授他人（能讲清楚才是真懂）
