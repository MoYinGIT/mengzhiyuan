# 编程学院

> **让所有孩子都能学会编程。**

编程学院遵循 [GESP（CCF编程能力等级认证）](https://gesp.ccf.org.cn/) 官方大纲，从 Scratch 图形化编程入门，到 C++ 竞赛进阶，循序渐进，稳扎稳打。

---

## 🎯 学习路径总览

### 主线路径（推荐）

```
Scratch L1-L3 (6-9岁) → L4过渡 → C++ L1-L8 (9岁+) → 竞赛
```

**主线特点**：
- 9岁（四年级）开始 C++，为竞赛预留充足时间
- 六年级可参加 CSP-J 入门组
- 初中冲刺 CSP-S 提高组

### 可选支线

```
Scratch L3 (8-9岁) → Python L1-L2 (兴趣导向)
```

**支线特点**：
- 适合对人工智能、数据分析感兴趣
- 不打算参加信息学竞赛
- 作为兴趣培养

## 🎯 路线选择指南

### 情况一：目标信息学竞赛

**推荐路线**：Scratch L3 → C++ L1-L8

| 年级 | 内容 | 目标 |
|:---:|:---|:---|
| 3年级 | Scratch L3 | 打好基础 |
| 4年级 | C++ L1-L2 | 语法入门 |
| 5年级 | C++ L3-L4 | 数据结构+基础算法 |
| 6年级 | C++ L5-L6 | **CSP-J 一等奖** |
| 初中 | C++ L7-L8 | **CSP-S 省一/省队** |

### 情况二：兴趣导向，不竞赛

**推荐路线**：Scratch L3 → Python L1-L2

- 学习数据处理、自动化脚本
- 人工智能入门（机器学习基础）
- 做有趣的小项目

### 情况三：不确定

**建议**：先按主线走，保留选择权

- 四年级开始 C++，学得会就继续竞赛路径
- 如果发现不适应，可随时转 Python
- C++ 语法严格，基础打牢后转其他语言容易

## 🏛️ 三经典与编程教育

### 《易经》· 穷则变，变则通

> **教育启示**: 当孩子遇到学习困难时，不强行推进，而是变换方法。

- Scratch 图形化 → 降低入门门槛
- **L4 过渡** → **关键转折点，从图形化到代码**
- C++ 性能卓越 → 满足竞赛需求

**因材施教**：
- 走主线：目标竞赛，9岁开始 C++
- 走支线：兴趣导向，10岁开始 Python

### 《道德经》· 道法自然

> **教育启示**: 顺应孩子认知发展规律，不揠苗助长。

- **6-9岁**: 形象思维 → Scratch 图形化（L1-L3）
- **9岁+**: 抽象思维发展 → **C++ 代码编程（L4过渡后直接C++）**
- **10-12岁**: Python 可选，适合对AI感兴趣的孩子

### 《论语》· 工欲善其事，必先利其器

> **教育启示**: 工具选择决定效率。

**C++ 是竞赛的利器**：
- CSP-J/S、NOIP 唯一指定语言
- 四年级开始打磨，六年级亮剑
- 基础不牢，地动山摇

## 💡 常见问题

**Q1: 为什么四年级开始 C++？**  
A: 9-10岁抽象思维快速发展，是代码编程的最佳起点。四年级开始，六年级有足够时间准备 CSP-J。

**Q2: 必须学 Python 吗？**  
A: **不是必须**。Python 是支线，适合对AI感兴趣但不竞赛的孩子。竞赛路径直接 C++。

**Q3: 学完 Scratch L3 可以跳级吗？**  
A: 基础好、数学强的孩子可以**跳过 L4**，直接 C++ L1。L4 是给需要缓冲的孩子准备的。

**Q4: Python 和 C++ 哪个更难？**  
A: **C++ 更难**，但竞赛必须用 C++。Python 语法简单，但无法参加信奥竞赛。

**Q5: 学多久可以参加竞赛？**  
A: 四年级开始 C++，六年级（约2年）可以参加 CSP-J 入门组。
