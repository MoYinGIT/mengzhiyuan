## 🏛️ 三经典与编程教育

### 《易经》· 穷则变，变则通

> **教育启示**: 当孩子遇到学习困难时，不强行推进，而是变换方法。

- Scratch 图形化 → 降低入门门槛
- Python 简洁语法 → 平滑过渡到代码
- C++ 性能卓越 → 满足竞赛需求

**因材施教**，根据孩子特点选择适合的学习路径。

### 《道德经》· 道法自然

> **教育启示**: 顺应孩子认知发展规律，不揠苗助长。

- **6-10岁**: 形象思维 → Scratch 图形化
- **10-12岁**: 抽象思维萌芽 → Python 代码
- **12岁+**: 逻辑思维成熟 → C++ 算法

### 《论语》· 有教无类

> **教育启示**: 编程不应是少数人的特权，每个孩子都应有机会学习。

- 本学院所有内容**永久免费**
- 从启蒙到竞赛，**完整路径**
- 所有资料**开源共享**

## 🚀 开始你的编程之旅

### 第一步：选择起点

根据年龄和基础选择适合的阶段：

- **零基础 6-10岁** → 从 [Scratch L1](./scratch/l1.md) 开始
- **零基础 10-12岁** → 可从 [Python L1](./python/l1.md) 开始
- **有Scratch基础** → 跳过L1，直接评估后进入合适级别
- **目标竞赛** → 尽快过渡到 C++，准备 GESP L3-L8

### 第二步：学习模式

**推荐学习节奏**:
- 每周 2-3 次课
- 每次 1-1.5 小时
- 课后完成练习
- 每级完成后参加模拟测试

### 第三步：持续进阶

完成每一级后：
1. 完成课后练习
2. 参加模拟测试
3. 通过后可报名正式 GESP 考试
4. 考取证书后继续下一级

## 💡 常见问题

**Q1: 孩子年龄小，应该从Scratch开始吗？**  
A: 6-10岁强烈建议从Scratch开始，图形化编程更符合该年龄段认知特点。

**Q2: 学完Scratch还要学Python吗？**  
A: 建议学习。Python是代码编程的桥梁，能帮助孩子理解真正编程语言的概念。

**Q3: 多久可以学完一级？**  
A: 一般 2-3 个月可以学完一级，具体取决于学习时间和基础。

**Q4: 必须参加考试吗？**  
A: 不强制，但建议参加。考试是检验学习成果的好方式，也能获得权威认证。

**Q5: 学多久可以参加竞赛？**  
A: 完成 C++ L2 后可尝试 CSP-J 入门，完成 GESP L6 后可系统准备竞赛。
