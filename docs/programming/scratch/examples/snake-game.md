# 贪吃蛇游戏

> 经典街机游戏 · 吃食物变长 · 不要撞墙

---

## 🎯 游戏目标

- 控制蛇吃食物
- 每吃一个食物，蛇变长一节
- 不要撞到墙壁或自己的身体
- 获得尽可能高的分数

---

## 🎮 游戏玩法

- **方向控制**：↑ ↓ ← → 控制蛇的移动方向
- **吃食物**：蛇头碰到食物，身体变长，得分+1
- **游戏结束**：撞到墙壁或自己的身体

        // 移动
        移动 20 步
        
        // 检查撞墙
        如果 x 坐标 < -230 或 x 坐标 > 230 或 y 坐标 < -170 或 y 坐标 > 170 那么
            说 "游戏结束！撞到墙了！" 持续 3 秒

结束

## 🧠 编程思维

### 数据结构：队列

```
贪吃蛇的身体就是一个队列：
- 头部 = 队列前端（插入）
- 尾部 = 队列后端（删除）

移动 = 在头部插入新位置，在尾部删除旧位置
吃食物 = 只插入不删除（队列变长）

这是数据结构的真实应用！
```

### 碰撞检测

```
贪吃蛇的碰撞检测：
1. 撞墙检测：检查坐标是否超出边界
2. 撞自己检测：检查头部位置是否在身体列表中

这用到了：
- 坐标系统
- 列表操作
- 条件判断
```
