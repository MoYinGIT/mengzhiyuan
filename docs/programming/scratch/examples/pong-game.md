# 示例：弹球游戏

> 经典的街机游戏复刻

## 🎮 游戏效果

- 球在舞台上来回反弹
- 玩家控制底部挡板接球
- 接到球得分，漏球减生命
- 生命为0时游戏结束

## 📝 完整代码

### 球代码

```scratch
// 变量：得分、生命（适用于所有角色）

// 初始化
当绿旗被点击
移到 x: (0) y: (0)
面向 (45) 方向
重复执行
    移动 (5) 步
    碰到边缘就反弹
    
    // 碰到挡板反弹
    如果 <碰到 [挡板 v]？> 那么
        面向 (180 - 方向) 方向
        将得分增加 (1)
        播放声音 [Beep v]
    结束
    
    // 漏球检测
    如果 <(y坐标) < (-180)> 那么
        将生命增加 (-1)
        播放声音 [Boop v]
        移到 x: (0) y: (0)
        等待 (1) 秒
        如果 <(生命) = (0)> 那么
            说游戏结束！2秒
            停止 [全部 v]
        结束
    结束
结束
```

### 挡板代码

```scratch
// 初始化
当绿旗被点击
移到 x: (0) y: (-150)
显示

// 鼠标控制
当绿旗被点击
重复执行
    移到 x: (鼠标的x坐标) y: (-150)
结束

// 键盘控制（可选）
当按下 [向左 v] 箭头
将x坐标增加 (-20)

当按下 [向右 v] 箭头
将x坐标增加 (20)
```

### 背景代码（初始化）

```scratch
// 游戏初始化
当绿旗被点击
将得分设为 (0)
将生命设为 (3)
```

## 💡 要点解析

| 知识点 | 说明 |
|:---|:---|
| 反弹角度 | 使用 `180 - 方向` 实现反弹 |
| 碰撞检测 | 判断球是否碰到挡板 |
| 边界判断 | 检测球是否漏到舞台底部 |
| 生命系统 | 用变量控制游戏结束条件 |

## 🎯 扩展挑战

1. **增加难度**：球速随得分增加
2. **道具系统**：随机出现道具，接到有特殊效果
3. **关卡设计**：得分达到目标后进入下一关
4. **双人模式**：两个挡板，一人一边
