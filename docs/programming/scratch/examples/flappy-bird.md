# Flappy Bird 游戏

> 现象级手游 · 点击飞行 · 躲避管道

---

## 🎯 游戏目标

- 控制小鸟飞行
- 点击屏幕/按键让小鸟上升
- 穿过管道之间的缝隙
- 不要撞到管道或地面
- 尽可能飞得更远，获得更高分数

---

## 🎮 游戏玩法

- **点击/按键**：小鸟上升
- **不点击**：小鸟受重力下落
- **穿过管道**：每穿过一对管道，得分+1
- **游戏结束**：撞到管道或地面

        结束如果
        如果 方向 < -60 那么

        结束如果

        结束如果

## 🧠 编程思维

### 物理模拟

```
Flappy Bird 的核心是简单的物理模拟：

速度 = 速度 + 加速度（重力）
位置 = 位置 + 速度

这就是：
v = v + a
y = y + v

游戏开发的基础物理！
```

### 无限地图生成

```
管道无限生成的原理：
1. 每隔一段时间生成一对管道
2. 管道从右向左移动
3. 移出屏幕后删除
4. 循环往复，形成无限地图

这是游戏中常用的技术！
```

### 碰撞检测简化

```
Flappy Bird 的碰撞检测可以简化：
- 小鸟是点或小圆
- 管道是矩形
- 检查圆和矩形是否相交

这比像素级碰撞检测高效得多！
```
