## 📚 知识模块

### 模块一：变量入门

#### 1.1 什么是变量？

**变量就像是一个盒子**，可以装东西，里面的内容可以改变。

**生活中的变量**：
- 你的年龄：每年增加1
- 游戏得分：打中敌人就加分
- 温度：随着天气变化

**Scratch 中的变量**

创建变量：
1. 点击「变量」模块
2. 点击「建立一个变量」
3. 输入变量名称，如「分数」
4. 选择「适用于所有角色」或「仅适用于当前角色」
变量类型：

| 类型 | 适用范围 | 使用场景 |
|:---|:---|:---|
| **公有变量** | 适用于所有角色 | 所有角色都能看到和修改，如游戏总分数 |
| **私有变量** | 仅适用于当前角色 | 只有这个角色能使用，如角色的生命值 |
#### 1.2 变量的基本操作

**四个基本积木**：

| 积木 | 功能 | 示例 |
|:---|:---|:---|
| `将分数设为 0` | 设置变量的值 | 游戏开始时重置分数 |
| `将分数增加 1` | 变量加N | 打中敌人加10分 |
| `将分数减少 1` | 变量减N | 被敌人打中减5分 |
| `分数` | 获取变量的值 | 显示当前分数 |

**第一个变量程序：计数器**

```scratch
当绿旗被点击
将计数设为 0
重复执行
    等待 1 秒
    将计数增加 1
    说 计数 2秒
结束
```

#### 1.3 实用案例

**案例一：计分板**

```scratch
// 变量：分数（适用于所有角色）

当绿旗被点击
将分数设为 0
重复执行
    说 分数  // 在舞台上显示分数
结束

// 当打中敌人时
当接收到打中敌人
将分数增加 10
播放声音得分
```

**案例二：计时器**

```scratch
// 变量：时间（适用于所有角色）

当绿旗被点击
将时间设为 30
重复执行直到 时间 = 0
    等待 1 秒
    将时间减少 1
结束
广播游戏结束
```

**案例三：生命值**

```scratch
// 变量：生命（仅适用于当前角色）

当绿旗被点击
将生命设为 3
重复执行
    说生命  // 显示生命
    如果生命 = 0 那么
        说游戏结束！
        停止全部
    结束
结束

// 当碰到敌人
当碰到颜色 #FF0000
将生命减少 1
说哎哟！
等待 0.5 秒
```

### 模块二：克隆技术

#### 2.1 什么是克隆？

**克隆就是复制一个一模一样的角色。**

就像孙悟空拔根毫毛变出很多小猴子！

**使用场景**：
- 游戏里有多个相同的敌人
- 发射很多子弹
- 满天星星效果
- 下雨、下雪效果

#### 2.2 克隆积木

| 积木 | 功能 |
|:---|:---|
| `克隆自己` | 创建一个克隆体 |
| `当作为克隆体启动时` | 克隆体启动时执行的代码 |
| `删除此克隆体` | 删除这个克隆体 |

**重要规则**：
- 克隆体继承原角色的所有属性（位置、大小、方向等）
- 克隆体启动时，执行「当作为克隆体启动时」下面的代码
- 克隆体用完要删除，否则会越来越多，程序变慢

#### 2.3 克隆案例

**案例一：发射子弹**

```scratch
// 太空飞船的代码

当按下空格键
克隆自己

当作为克隆体启动时
显示
重复执行
    移动 10 步
    如果碰到边缘 那么
        删除此克隆体
    结束
结束
```

**案例二：敌人军团**

```scratch
// 敌人角色的代码

当绿旗被点击
隐藏  // 本体隐藏，只显示克隆体
重复执行
    等待 2 秒
    克隆自己
结束

当作为克隆体启动时
移到 x: 随机数(-200, 200) y: 180
显示
重复执行
    y坐标减少 5
    如果 y坐标 < -180 那么
        删除此克隆体
    结束
    等待 0.1 秒
结束
```

**案例三：满天下星星**

```scratch
// 星星角色的代码

当绿旗被点击
隐藏
重复 50 次
    克隆自己
结束

当作为克隆体启动时
移到 x: 随机数(-240, 240) y: 随机数(-180, 180)
将大小设为 随机数(10, 50)
显示
重复执行
    将虚像特效增加 5
    等待 0.1 秒
    如果虚像特效 > 90 那么
        将虚像特效设为 0
    结束
结束
```

### 模块三：逻辑运算

#### 3.1 什么是逻辑运算？

**逻辑运算就是判断多个条件。**

**生活中的逻辑**：
- 「今天晴天」**而且**「我有空」→ 去公园
- 「下雨」**或者**「下雪」→ 带伞
- **不**「今天周末」→ 要上学

#### 3.2 三种逻辑运算

**与（and）**：两个条件都满足
```scratch
如果 <<碰到鼠标指针> 与 <按下鼠标>> 那么
    // 鼠标碰到角色，并且按下了鼠标
    说抓住你了！
结束
```

**或（or）**：满足其中一个条件
```scratch
如果 <<按下空格键> 或 <碰到颜色 #FF0000>> 那么
    // 按了空格键，或者碰到了红色
    播放声音跳跃
结束
```

**不成立（not）**：条件反过来
```scratch
如果 <不成立 <碰到颜色 #000000>> 那么
    // 没有碰到黑色（地面）
    y坐标减少 5  // 下落
结束
```

#### 3.3 逻辑运算组合

**复杂的条件**：
```scratch
// 游戏胜利条件：分数>=100 且 生命>0 且 时间>0
如果 <<<分数 >= 100> 与 <生命 > 0>> 与 <时间 > 0>> 那么
    说胜利！
    停止全部
结束
```

### 模块四：更多侦测积木

#### 4.1 距离侦测

```scratch
到鼠标指针的距离
到角色的距离  // 如：到小猫的距离
```

**应用**：敌人只在距离近时才追击
```scratch
重复执行
    如果 <到小猫的距离 < 100> 那么
        面向小猫
        移动 3 步
    结束
结束
```

#### 4.2 询问与回答

```scratch
询问你叫什么名字？并等待
说你好，回答！2秒

询问5+3=？并等待
如果 <回答 = 8> 那么
    说答对了！
否则
    说再想想...
结束
```

#### 4.3 计时器

```scratch
计时器归零
如果 <计时器 > 10> 那么
    // 过了10秒
结束
```

**注意**：这是 Scratch 内置的计时器，和变量做的计时器不一样！

### 模块五：变量的高级应用

#### 5.1 私有变量 vs 公有变量

```scratch
// 公有变量：所有角色共享
适用于所有角色的变量：总分

// 私有变量：每个角色独立
仅适用于当前角色的变量：生命、速度
```

#### 5.2 云变量（需要登录）

云变量可以保存到服务器，即使关闭 Scratch 也不会丢失！

用途：
- 排行榜
- 存档功能
- 多人在线数据同步

#### 5.3 变量滑块

右键点击变量，选择「滑杆」，可以直接拖动改变变量值！

用途：
- 调节速度
- 调节大小
- 调节音量

## 🎯 三经典智慧

### 《易经》· 穷则变，变则通

**变量的本质就是「变化」。**

程序运行时，变量的值不断变化：
- 分数增加
- 时间减少
- 位置移动

学会了变量，就掌握了编程的核心——**处理变化**。

### 《道德经》· 一生二，二生三，三生万物

**克隆技术就是「一生多」。**

一个角色，可以克隆出千千万万个。就像：
- 一颗种子长出一片森林
- 一个细胞分裂成生命
- 一个简单的规则，产生复杂的现象

### 《论语》· 学而不思则罔

学变量和克隆，要多思考：
- 这个变量应该在什么时候改变？
- 克隆体应该在什么时候删除？
- 多个角色如何配合？
