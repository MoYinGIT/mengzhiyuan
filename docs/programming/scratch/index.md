# Scratch 图形化编程

> **用搭积木的方式学编程** · 适合 6-10 岁

---

## 🎯 为什么选择 Scratch？

### 图形化编程的优势

| 传统代码 | Scratch |
|:---|:---|
| `print("Hello")` | 拖拽积木「说 Hello」 |
| 需要记住语法 | 像搭积木一样直观 |
| 容易出错 | 积木卡扣，拼错拼不上 |
| 抽象难懂 | 所见即所得 |

### Scratch 适合谁？

- ✅ 6-10岁零基础儿童
- ✅ 培养逻辑思维的启蒙阶段
- ✅ 对编程感兴趣但怕难的孩子
- ✅ 想为代码编程打基础的学生

## 🚀 快速开始

### 第一步：访问 Scratch

- **在线版**（推荐）：https://scratch.mit.edu/create
- **离线版**：下载 [Scratch 桌面版](https://scratch.mit.edu/download)

### 第二步：创建第一个作品

5分钟完成「小猫打招呼」：

```scratch
当绿旗被点击
说"你好，世界！"2秒
播放声音 [喵]
```

**操作步骤**：
1. 打开 Scratch
2. 从「事件」模块拖出「当绿旗被点击」
3. 从「外观」模块拖出「说...2秒」
4. 输入文字「你好，世界！」
5. 从「声音」模块拖出「播放声音」
6. 点击绿旗运行

🎉 恭喜你！这是你写的第一个程序！

### 第三步：继续学习

- 📖 阅读 [L1 一级课程](./l1.md)
- 🎮 完成课后练习
- 🏆 挑战综合项目

## 🏛️ 三经典与 Scratch

### 《易经》· 变化之道

Scratch 的角色在舞台上移动、旋转、变色，这就是「变化」。

通过编程，孩子理解：
- 改变坐标 → 位置变化
- 改变角度 → 方向变化  
- 改变颜色 → 外观变化

### 《道德经》· 道法自然

Scratch 的积木设计符合儿童认知规律：
- 颜色区分类别（运动蓝、外观紫、事件黄）
- 形状提示用法（帽子形的事件积木只能放顶部）
- 拼插即正确，不会拼错

### 《论语》· 有教无类

Scratch 是免费的，全球的孩子都能用同样的工具学习编程。

## ❓ 常见问题

**Q1: Scratch 要学多久？**  
A: 完成 L1-L4 四级大约需要 8-12 个月。

**Q2: 学完 Scratch 还要学 Python 吗？**  
A: 强烈建议学习。Python 是真正代码编程的桥梁。

**Q3: 孩子年龄小，能学会吗？**  
A: 6 岁以上完全可以，Scratch 就是专为儿童设计的。

**Q4: 必须考级吗？**  
A: 不强制，但考级是检验学习成果的好方法。

**Q5: 没有电脑能学吗？**  
A: Scratch 有平板版（iPad/Android），但电脑版功能更完整。
