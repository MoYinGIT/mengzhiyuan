# Scratch 图形化编程

> **用搭积木的方式学编程** · 适合 6-9 岁 · **启蒙阶段**

---

## 🎯 Scratch 的定位

**Scratch 不是终点，而是起点。**

Scratch 是编程启蒙工具，帮助孩子：
- ✅ 建立编程思维（顺序、分支、循环）
- ✅ 理解核心概念（变量、函数、模块化）
- ✅ 培养兴趣和信心
- ✅ **为 C++ 代码编程打基础**

> 💡 **重要**：Scratch 学到 L3 即可，L4 是向 C++ 的过渡，不是继续深入 Scratch。

---

## 📚 三级启蒙 + 一级过渡

| 级别 | 适合年龄 | 核心内容 | 去向 |
|:---:|:---:|:---|:---|
| **L1 一级** | 6-7岁 | 编程基础、坐标系、三大结构 | → L2 |
| **L2 二级** | 7-8岁 | 变量、克隆、逻辑运算 | → L3 |
| **L3 三级** | 8-9岁 | 列表、函数、模块化 | **→ C++ L1 或 L4过渡** |
| **L4 四级** | 9-10岁 | **向C++过渡**、代码对照学习 | **→ C++ L1** |

### 学习时长

| 级别 | 建议时长 | 说明 |
|:---:|:---:|:---|
| L1 | 3-4 个月 | 培养兴趣为主 |
| L2 | 3-4 个月 | 理解变量和逻辑 |
| L3 | 3-4 个月 | 掌握函数和模块化 |
| L4 | 2 个月 | **过渡性质**，可选 |

**总计**：L1-L3 约 1 年，L4 约 2 个月

## 🚀 快速开始

### 第一步：访问 Scratch

Scratch 是在线平台，无需安装：
- 打开浏览器，搜索「Scratch 在线版」
- 注册免费账号
- 开始创作

### 第二步：创建第一个作品

5分钟完成「小猫打招呼」：

```scratch
当绿旗被点击
说"你好，世界！"2秒
```

### 第三步：规划学习路径

**根据年龄规划**：

| 当前年龄 | 学习路径 |
|:---:|:---|
| 6-7岁 | L1 → L2 → L3 → **C++** |
| 8-9岁 | L2/L3（评估后）→ **C++** |
| 9-10岁 | **直接 C++ L1** 或 L4 过渡 |

## 🎯 为什么不要一直学 Scratch？

### Scratch 的局限

| 局限 | 说明 | 影响 |
|:---|:---|:---|
| **无法参加竞赛** | 信奥竞赛只用 C++ | 升学无优势 |
| **性能受限** | 做不了复杂算法 | 天花板低 |
| **就业无关** | 真正的开发用代码 | 学了白学 |

### 最佳策略

```
Scratch L1-L3（1年） → C++ L1-L8（2-3年） → 竞赛/升学
       ↑                                    ↓
   启蒙兴趣                          核心竞争力
```

**9岁（四年级）是转折点**，必须开始 C++。

## 💡 学习建议

### 学习方法

1. **L1-L2**：培养兴趣，多做多玩
2. **L3**：理解函数，为 C++ 打基础
3. **L4**（可选）：熟悉代码概念，消除恐惧
4. **然后**：果断进入 C++，不要留恋 Scratch

### 跳级建议

| 情况 | 建议 |
|:---|:---|
| 8岁，数学好，理解力强 | 学完 L3 **直接 C++**，跳过 L4 |
| 9岁，时间紧 | **直接 C++ L1**，L4 可不看 |
| 需要缓冲，对代码恐惧 | L4 过渡，建立信心 |

### 家长建议

- 👨‍👩‍👧 **不要让孩子沉迷 Scratch**，9岁必须转 C++
- 🎯 **Scratch 是玩具，C++ 是工具**
- ⏰ **控制时间**：Scratch 阶段每天不超过 1 小时
- 🚀 **推动进阶**：不要让孩子在舒适区停留太久

## 🏛️ 三经典与 Scratch

### 《易经》· 启蒙之道

**蒙以养正，圣功也。**

Scratch 就是「蒙」——启蒙阶段。
- 让孩子对编程产生兴趣
- 建立正确的编程思维
- **但蒙阶段不能一直停留**

### 《道德经》· 知止不殆

**知道什么时候该停止，才不会危险。**

Scratch 学到 L3 就该「止」：
- 再深入就是浪费时间
- 9岁必须开始 C++
- 不要让孩子在图形化里「躺平」

### 《论语》· 有教无类

Scratch 让每个孩子都能接触编程，但：
- **不是每个孩子都要学很久 Scratch**
- 有的孩子 L2 就可以转 C++
- 因材施教，不要一刀切

## ❓ 常见问题

**Q1: Scratch 要学多久？**  
A: **L1-L3 约 1 年**，L4 可选。不要学超过 1.5 年。

**Q2: 学完 Scratch 学什么？**  
A: **C++**，不是继续 Scratch。Scratch L4 只是过渡。

**Q3: 可以不学 Scratch 直接 C++ 吗？**  
A: **9岁以上可以**。9岁以下建议先 Scratch 培养兴趣。

**Q4: Scratch 考级有用吗？**  
A: 作用有限。**GESP C++ 考级才有含金量**。

**Q5: 孩子喜欢 Scratch 不想学 C++ 怎么办？**  
A: **家长要推动**。Scratch 是玩具，C++ 是技能。9岁必须转。
