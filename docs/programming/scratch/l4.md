## 🔄 从 Scratch 到 C++ 的对比

### 变量

| Scratch | C++ | 说明 |
|:---|:---|:---|
| `将x设为10` | `int x = 10;` | 整数变量 |
| `将name设为小明` | `string name = "小明";` | 字符串变量 |
| `将pi设为3.14` | `double pi = 3.14;` | 小数变量 |

### 输出（说/打印）

**Scratch**：
```scratch
说 "你好，世界！"
```

**C++**：
```cpp
cout << "你好，世界！" << endl;
```

### 循环

**Scratch**：
```scratch
重复执行 (10) 次
    移动 (10) 步
结束
```

**C++**：
```cpp
for (int i = 0; i < 10; i++) {
    cout << "*";
}
```

### 条件判断

**Scratch**：
```scratch
如果 <x > 0> 那么
    说 "正数"
否则
    说 "负数或零"
结束
```

**C++**：
```cpp
if (x > 0) {
    cout << "正数" << endl;
} else {
    cout << "负数或零" << endl;
}
```

## 📖 知识模块

### 模块一：为什么要学 C++？

#### 1.1 Scratch 的局限

| 局限 | 说明 |
|:---|:---|
| **性能** | Scratch 运行慢，做不了复杂程序 |
| **扩展性** | 无法访问硬件、网络等底层功能 |
| **竞赛** | 信奥竞赛只用 C++，不用 Scratch |
| **职业** | 真正的软件开发用代码，不用积木 |

#### 1.2 C++ 的优势

- 🏆 **竞赛标准**：NOIP、CSP-J/S 唯一指定语言
- ⚡ **性能卓越**：运行速度快，适合算法竞赛
- 🔧 **工业级**：游戏引擎、操作系统都用 C++
- 📈 **学习路径**：C++ → 算法 → 竞赛 → 名校

### 模块二：C++ 初体验

#### 2.1 第一个 C++ 程序

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "你好，世界！" << endl;
    return 0;
}
```

**对比 Scratch**：
```scratch
当绿旗被点击
说 "你好，世界！"
```

**相同点**：都是让计算机输出一句话
**不同点**：C++ 需要写更多"规矩"（语法）

#### 2.2 C++ 程序结构

```cpp
// 第1行：包含头文件（工具箱）
#include <iostream>

// 第2行：使用标准命名空间
using namespace std;

// 主函数（程序从这里开始）
int main() {
    // 你的代码写在这里
    
    return 0;  // 程序结束
}
```

| 部分 | 作用 | 类比 Scratch |
|:---|:---|:---|
| `#include` | 加载工具 | 添加扩展模块 |
| `main()` | 程序入口 | 绿旗被点击 |
| `cout` | 输出文字 | 说积木 |
| `return 0` | 正常结束 | 停止全部 |

### 模块三：算法基础（Scratch + C++ 对照）

#### 3.1 求和程序

**Scratch**：
```scratch
当绿旗被点击
将 和 设为 0
重复执行 (100) 次
    将 和 增加 计数器
结束
说 和
```

**C++**：
```cpp
#include <iostream>
using namespace std;

int main() {
    int sum = 0;  // 和设为0
    
    for (int i = 1; i <= 100; i++) {
        sum = sum + i;  // 和增加i
    }
    
    cout << sum << endl;  // 说和
    return 0;
}
```

#### 3.2 判断奇偶

**Scratch**：
```scratch
如果 <(数字) 除以 (2) 的余数 = 0> 那么
    说 "偶数"
否则
    说 "奇数"
结束
```

**C++**：
```cpp
int num;
cin >> num;  // 输入数字

if (num % 2 == 0) {
    cout << "偶数" << endl;
} else {
    cout << "奇数" << endl;
}
```

#### 3.3 找最大值

**Scratch**：
```scratch
将 最大值 设为 列表的第1项
重复执行 (列表长度 - 1) 次
    如果 (列表的第(计数器 + 1)项) > (最大值) 那么
        将 最大值 设为 (列表的第(计数器 + 1)项)
    结束
结束
```

**C++**：
```cpp
int a[5] = {5, 3, 8, 1, 2};
int maxVal = a[0];  // 第一个设为最大

for (int i = 1; i < 5; i++) {
    if (a[i] > maxVal) {
        maxVal = a[i];  // 发现更大的
    }
}

cout << "最大值：" << maxVal << endl;
```

## 🎮 综合项目

### 项目：简易计算器（Scratch + C++ 双版本）

**功能**：输入两个数，输出四则运算结果

**Scratch 版**：
```scratch
当绿旗被点击
询问 "请输入第一个数："
将 a 设为 回答
询问 "请输入第二个数："
将 b 设为 回答

说 (连接 "和：" 和 (a + b))
说 (连接 "差：" 和 (a - b))
说 (连接 "积：" 和 (a * b))
说 (连接 "商：" 和 (a / b))
```

**C++ 版**：
```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b;
    
    cout << "请输入第一个数：";
    cin >> a;
    cout << "请输入第二个数：";
    cin >> b;
    
    cout << "和：" << a + b << endl;
    cout << "差：" << a - b << endl;
    cout << "积：" << a * b << endl;
    cout << "商：" << a / b << endl;
    
    return 0;
}
```
