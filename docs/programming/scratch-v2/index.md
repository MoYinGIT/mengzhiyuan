## 🎮 为什么选择Scratch？

**Scratch是最佳的第一门编程语言：**

| 特点 | 说明 |
|:---|:---|
| **图形化** | 拖拽积木，不用记语法 |
| **即时反馈** | 点击绿旗立刻看到效果 |
| **创造性强** | 做动画、做游戏、做故事 |

## 📝 学习方法

### 给孩子
1. **先玩**：打开Scratch随便点一点
2. **跟做**：跟着教程一步一步做
3. **改着玩**：改数字、换角色、加功能
4. **创造**：做自己的原创项目

### 给家长
1. **陪伴**：坐在旁边，但不代劳
2. **提问**："你想实现什么效果？"
3. **鼓励**："这个创意很棒！"
4. **不焦虑**：进度慢没关系，保持兴趣最重要
