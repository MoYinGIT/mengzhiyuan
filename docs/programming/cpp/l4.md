## 📚 模块一：递归

### 1.1 什么是递归？

**递归**：函数调用自己。

**递归的两个要素**：
1. **基准情况**（Base Case）：停止递归的条件
2. **递归步骤**（Recursive Step）：向基准情况靠近

**类比**：
- 俄罗斯套娃
- 镜子里的镜子
- 故事里的"从前有座山，山里有座庙..."

### 1.2 阶乘

```cpp
// n! = n × (n-1) × (n-2) × ... × 1
// 5! = 5 × 4 × 3 × 2 × 1 = 120

#include <iostream>
using namespace std;

// 递归计算阶乘
int factorial(int n) {
    // 基准情况
    if (n <= 1) {
        return 1;
    }
    // 递归步骤
    return n * factorial(n - 1);
}

int main() {
    cout << factorial(5) << endl;  // 120
    return 0;
}
```

**递归过程可视化**：
```
factorial(5)
= 5 × factorial(4)
= 5 × (4 × factorial(3))
= 5 × (4 × (3 × factorial(2)))
= 5 × (4 × (3 × (2 × factorial(1))))
= 5 × (4 × (3 × (2 × 1)))
= 5 × (4 × (3 × 2))
= 5 × (4 × 6)
= 5 × 24
= 120
```

### 1.3 斐波那契数列

```cpp
// F(n) = F(n-1) + F(n-2)
// F(1) = F(2) = 1
// 1, 1, 2, 3, 5, 8, 13, 21...

int fib(int n) {
    if (n <= 2) return 1;  // 基准情况
    return fib(n - 1) + fib(n - 2);
}

// 优化版本（记忆化）
int memo[100] = {0};
int fib_fast(int n) {
    if (n <= 2) return 1;
    if (memo[n] != 0) return memo[n];  // 已经计算过
    return memo[n] = fib_fast(n - 1) + fib_fast(n - 2);  // 保存结果
}
```

### 1.4 汉诺塔

**问题**：将n个盘子从A柱移到C柱，每次只能移一个，大盘不能压小盘。

```cpp
#include <iostream>
using namespace std;

// 将n个盘子从a柱移到c柱，借助b柱
void hanoi(int n, char a, char b, char c) {
    if (n == 1) {
        // 基准情况：只有一个盘子，直接移动
        cout << a << " -> " << c << endl;
        return;
    }
    // 递归步骤
    hanoi(n - 1, a, c, b);  // 将n-1个盘子从a移到b
    cout << a << " -> " << c << endl;  // 将第n个盘子从a移到c
    hanoi(n - 1, b, a, c);  // 将n-1个盘子从b移到c
}

int main() {
    hanoi(3, 'A', 'B', 'C');
    return 0;
}
```

### 1.5 全排列

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

void permutation(string s, int start) {
    if (start == s.length() - 1) {
        cout << s << endl;
        return;
    }
    for (int i = start; i < s.length(); i++) {
        swap(s[start], s[i]);          // 交换
        permutation(s, start + 1);     // 递归
        swap(s[start], s[i]);          // 恢复（回溯）
    }
}

int main() {
    permutation("abc", 0);
    return 0;
}
```

## 📚 模块二：排序算法

### 2.1 选择排序

**思想**：每次选择最小的，放到前面。

```cpp
void selection_sort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        int min_idx = i;
        // 找最小值的位置
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[min_idx]) {
                min_idx = j;
            }
        }
        // 交换
        swap(arr[i], arr[min_idx]);
    }
}
// 时间复杂度：O(n²)
```

### 2.2 插入排序

**思想**：像整理扑克牌，把牌插入到正确位置。

```cpp
void insertion_sort(int arr[], int n) {
    for (int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;
        // 把比key大的元素后移
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
}
// 时间复杂度：O(n²)，但常数小，对小数组很快
```

### 2.3 归并排序

**思想**：分而治之，先分成两半排序，再合并。

```cpp
// 合并两个有序数组
void merge(int arr[], int left, int mid, int right) {
    int n1 = mid - left + 1;
    int n2 = right - mid;
    
    int L[n1], R[n2];
    for (int i = 0; i < n1; i++) L[i] = arr[left + i];
    for (int i = 0; i < n2; i++) R[i] = arr[mid + 1 + i];
    
    int i = 0, j = 0, k = left;
    while (i < n1 && j < n2) {
        if (L[i] <= R[j]) arr[k++] = L[i++];
        else arr[k++] = R[j++];
    }
    while (i < n1) arr[k++] = L[i++];
    while (j < n2) arr[k++] = R[j++];
}

void merge_sort(int arr[], int left, int right) {
    if (left >= right) return;
    int mid = left + (right - left) / 2;
    merge_sort(arr, left, mid);
    merge_sort(arr, mid + 1, right);
    merge(arr, left, mid, right);
}
// 时间复杂度：O(n log n)
```

### 2.4 快速排序

**思想**：选一个基准，小的放左边，大的放右边，递归排序。

```cpp
int partition(int arr[], int low, int high) {
    int pivot = arr[high];  // 选最后一个为基准
    int i = low - 1;
    for (int j = low; j < high; j++) {
        if (arr[j] < pivot) {
            i++;
            swap(arr[i], arr[j]);
        }
    }
    swap(arr[i + 1], arr[high]);
    return i + 1;
}

void quick_sort(int arr[], int low, int high) {
    if (low < high) {
        int pi = partition(arr, low, high);
        quick_sort(arr, low, pi - 1);
        quick_sort(arr, pi + 1, high);
    }
}
// 时间复杂度：平均O(n log n)，最坏O(n²)
```

### 2.5 排序算法对比

| 算法 | 平均时间 | 最坏时间 | 空间 | 稳定性 |
|:---|:---:|:---:|:---:|:---:|
| 选择排序 | O(n²) | O(n²) | O(1) | ❌ |
| 插入排序 | O(n²) | O(n²) | O(1) | ✅ |
| 冒泡排序 | O(n²) | O(n²) | O(1) | ✅ |
| 归并排序 | O(n log n) | O(n log n) | O(n) | ✅ |
| 快速排序 | O(n log n) | O(n²) | O(log n) | ❌ |

**稳定性**：相等元素的相对顺序是否保持不变。

## 📚 模块三：贪心算法

### 3.1 什么是贪心？

**贪心算法**：每一步都选择当前最优解，希望最终得到全局最优。

**适用条件**：
- 问题具有**最优子结构**
- 每一步的局部最优能导致全局最优

### 3.2 活动选择问题

**问题**：有n个活动，每个有开始和结束时间，选择最多不重叠的活动。

**贪心策略**：每次选择结束时间最早的。

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

struct Activity {
    int start, end;
};

bool cmp(Activity a, Activity b) {
    return a.end < b.end;  // 按结束时间排序
}

int main() {
    Activity acts[] = {{1, 4}, {3, 5}, {0, 6}, {5, 7}, {3, 8}};
    int n = 5;
    
    sort(acts, acts + n, cmp);
    
    int count = 1;
    int last_end = acts[0].end;
    
    for (int i = 1; i < n; i++) {
        if (acts[i].start >= last_end) {
            count++;
            last_end = acts[i].end;
        }
    }
    
    cout << "最多可以参加 " << count << " 个活动" << endl;
    return 0;
}
```

### 3.3 钱币找零

**问题**：用最少数量的钱币凑出金额。

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int coins[] = {100, 50, 20, 10, 5, 1};  // 面额

void make_change(int amount) {
    int count = 0;
    for (int i = 0; i < 6; i++) {
        while (amount >= coins[i]) {
            amount -= coins[i];
            count++;
            cout << coins[i] << " ";
        }
    }
    cout << "\n共 " << count << " 张" << endl;
}

int main() {
    make_change(186);  // 100 50 20 10 5 1
    return 0;
}
```

## 📚 模块四：时间复杂度

### 4.1 什么是时间复杂度？

**时间复杂度**：算法运行时间随输入规模增长的变化趋势。

**大O表示法**：描述算法在最坏情况下的上界。

### 4.2 常见时间复杂度

| 复杂度 | 名称 | n=1000时的运算次数 |
|:---:|:---|:---:|
| O(1) | 常数 | 1 |
| O(log n) | 对数 | ~10 |
| O(n) | 线性 | 1000 |
| O(n log n) | 线性对数 | ~10000 |
| O(n²) | 平方 | 1,000,000 |
| O(2ⁿ) | 指数 | 约10³⁰⁰ |

### 4.3 计算时间复杂度

**规则**：
1. 忽略常数：`O(2n)` = `O(n)`
2. 取最高阶：`O(n² + n)` = `O(n²)`
3. 嵌套相乘：双重循环 `O(n²)`

**示例**：
```cpp
// O(1)
int x = 10;

// O(n)
for (int i = 0; i < n; i++) { ... }

// O(n²)
for (int i = 0; i < n; i++)
    for (int j = 0; j < n; j++) { ... }

// O(log n)
while (n > 0) { n /= 2; }

// O(n log n) - 归并排序
merge_sort(arr, 0, n - 1);
```

## 🎮 综合项目：图书馆管理系统

```cpp
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

const int MAX = 1000;

struct Book {
    int id;
    string title;
    string author;
    bool borrowed;
};

Book library[MAX];
int book_count = 0;

// 添加书籍
void add_book() {
    if (book_count >= MAX) {
        cout << "图书馆已满！\n";
        return;
    }
    Book &b = library[book_count];
    cout << "书号: ";
    cin >> b.id;
    cin.ignore();
    cout << "书名: ";
    getline(cin, b.title);
    cout << "作者: ";
    getline(cin, b.author);
    b.borrowed = false;
    book_count++;
    cout << "添加成功！\n";
}

// 按书名排序（用于二分查找）
bool cmp_book(const Book &a, const Book &b) {
    return a.title < b.title;
}

void sort_books() {
    sort(library, library + book_count, cmp_book);
    cout << "已按书名排序！\n";
}

// 二分查找书籍
int binary_search_book(string title) {
    int left = 0, right = book_count - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (library[mid].title == title) return mid;
        if (library[mid].title < title) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}

// 显示所有书籍
void display_books() {
    cout << "\n======== 图书列表 ========\n";
    for (int i = 0; i < book_count; i++) {
        cout << library[i].id << "\t" 
             << library[i].title << "\t"
             << library[i].author << "\t"
             << (library[i].borrowed ? "已借出" : "可借阅") << endl;
    }
}

int main() {
    int choice;
    do {
        cout << "\n===== 图书馆管理系统 =====\n";
        cout << "1. 添加书籍\n";
        cout << "2. 排序书籍\n";
        cout << "3. 查找书籍\n";
        cout << "4. 显示所有\n";
        cout << "0. 退出\n";
        cout << "请选择: ";
        cin >> choice;
        
        switch (choice) {
            case 1: add_book(); break;
            case 2: sort_books(); break;
            case 3: {
                cin.ignore();
                string title;
                cout << "书名: ";
                getline(cin, title);
                int idx = binary_search_book(title);
                if (idx != -1) cout << "找到！索引: " << idx << endl;
                else cout << "未找到\n";
                break;
            }
            case 4: display_books(); break;
        }
    } while (choice != 0);
    
    return 0;
}
```

## 🏛️ 三经典智慧

### 《易经》· 递归与循环

**递归就是「反复其道」**：
- 从简单到复杂（基准→递归）
- 从复杂回归简单（递归→基准）
- 一阴一阳，循环往复

### 《道德经》· 治大国若烹小鲜

**分治思想**：
- 大问题分解为小问题
- 小问题解决了，大问题自然解决
- 归并排序、快速排序皆是此理

### 《论语》· 工欲善其事

**算法的重要性**：
- O(n²) vs O(n log n)，天壤之别
- 选择正确的算法，事半功倍
- 算法是程序员的"器"
