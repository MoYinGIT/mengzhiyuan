## 📚 模块一：指针基础

### 1.1 什么是指针？

**指针**是存储**内存地址**的变量。

**类比**：
- 变量 = 房间里的物品
- 地址 = 房间门牌号
- 指针 = 记录门牌号的纸条

### 1.2 指针的声明与使用

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 10;        // 普通变量
    int *p = &x;      // 指针变量，存储x的地址
    
    cout << "x的值: " << x << endl;       // 10
    cout << "x的地址: " << &x << endl;   // 0x7ffee...
    cout << "p的值: " << p << endl;       // 0x7ffee...（x的地址）
    cout << "*p的值: " << *p << endl;     // 10（解引用）
    
    // 通过指针修改值
    *p = 20;
    cout << "修改后x的值: " << x << endl;  // 20
    
    return 0;
}
```

**符号说明**：
| 符号 | 含义 | 示例 |
|:---|:---|:---|
| `*` | 声明指针 | `int *p` |
| `&` | 取地址 | `&x` |
| `*` | 解引用 | `*p` |

### 1.3 指针与数组

**数组名就是指针**：

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[5] = {10, 20, 30, 40, 50};
    int *p = arr;  // 等价于 &arr[0]
    
    // 通过指针访问数组
    cout << p[0] << endl;   // 10
    cout << *(p+1) << endl; // 20
    cout << *(p+2) << endl; // 30
    
    // 指针运算
    p++;  // 移动到下一个元素
    cout << *p << endl;  // 20
    
    p += 2;
    cout << *p << endl;  // 40
    
    return 0;
}
```

**重要**：`*(p+i)` 等价于 `p[i]` 等价于 `arr[i]`

### 1.4 指针与函数

**传递指针可以修改原变量**：

```cpp
#include <iostream>
using namespace std;

// 值传递（不能修改原变量）
void swap_by_value(int a, int b) {
    int temp = a;
    a = b;
    b = temp;
}

// 指针传递（可以修改原变量）
void swap_by_pointer(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    int x = 3, y = 5;
    
    swap_by_value(x, y);
    cout << x << " " << y << endl;  // 3 5（没变）
    
    swap_by_pointer(&x, &y);
    cout << x << " " << y << endl;  // 5 3（交换了）
    
    return 0;
}
```

## 📚 模块二：引用

### 2.1 什么是引用？

**引用**是变量的**别名**。

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 10;
    int &ref = x;  // ref 是 x 的引用（别名）
    
    cout << x << endl;     // 10
    cout << ref << endl;   // 10
    
    ref = 20;  // 修改ref就是修改x
    cout << x << endl;     // 20
    
    return 0;
}
```

### 2.2 引用 vs 指针

| 特性 | 指针 | 引用 |
|:---|:---|:---|
| 声明 | `int *p` | `int &r` |
| 初始化 | 可以不初始化 | 必须初始化 |
| 重新绑定 | 可以指向其他变量 | 不能改变 |
| 空值 | 可以为 nullptr | 不能为null |
| 使用 | 需要解引用 `*p` | 直接使用 `r` |

### 2.3 引用传递（推荐）

```cpp
// 引用传递（更简洁，效果同指针）
void swap_by_reference(int &a, int &b) {
    int temp = a;
    a = b;
    b = temp;
}

int main() {
    int x = 3, y = 5;
    swap_by_reference(x, y);  // 不需要取地址
    cout << x << " " << y << endl;  // 5 3
    return 0;
}
```

**建议**：能用引用的地方优先用引用。

## 📚 模块三：结构体

### 3.1 为什么需要结构体？

**问题**：如何表示一个学生？
- 姓名（string）
- 年龄（int）
- 成绩（double）

**不用结构体**：
```cpp
string name1, name2, name3;
int age1, age2, age3;
double score1, score2, score3;
// 混乱！难以管理
```

**用结构体**：
```cpp
struct Student {
    string name;
    int age;
    double score;
};

Student s1, s2, s3;  // 清晰！一个变量包含所有信息
```

### 3.2 结构体的定义与使用

```cpp
#include <iostream>
#include <string>
using namespace std;

// 定义结构体
struct Student {
    string name;
    int age;
    double score;
};

int main() {
    // 创建结构体变量
    Student s1;
    s1.name = "小明";
    s1.age = 12;
    s1.score = 89.5;
    
    // 输出
    cout << "姓名: " << s1.name << endl;
    cout << "年龄: " << s1.age << endl;
    cout << "成绩: " << s1.score << endl;
    
    // 初始化方式2
    Student s2 = {"小红", 11, 92.0};
    
    // 初始化方式3（C++11）
    Student s3{"小刚", 13, 85.5};
    
    return 0;
}
```

### 3.3 结构体数组

```cpp
#include <iostream>
using namespace std;

struct Student {
    string name;
    int age;
    double score;
};

int main() {
    // 结构体数组
    Student students[3] = {
        {"小明", 12, 89.5},
        {"小红", 11, 92.0},
        {"小刚", 13, 85.5}
    };
    
    // 计算平均分
    double sum = 0;
    for (int i = 0; i < 3; i++) {
        sum += students[i].score;
        cout << students[i].name << ": " 
             << students[i].score << endl;
    }
    
    cout << "平均分: " << sum / 3 << endl;
    
    return 0;
}
```

### 3.4 结构体与指针

```cpp
#include <iostream>
using namespace std;

struct Student {
    string name;
    int age;
};

int main() {
    Student s = {"小明", 12};
    Student *p = &s;
    
    // 通过指针访问成员
    cout << (*p).name << endl;   // 小明
    cout << p->name << endl;     // 小明（更简洁）
    cout << p->age << endl;      // 12
    
    // 修改
    p->age = 13;
    cout << s.age << endl;  // 13
    
    return 0;
}
```

**访问结构体成员的两种方式**：
- 变量名.成员：`s.name`
- 指针->成员：`p->name`（等价于 `(*p).name`）

### 3.5 结构体作为函数参数

```cpp
#include <iostream>
using namespace std;

struct Student {
    string name;
    int age;
    double score;
};

// 值传递（会复制整个结构体）
void print_student(Student s) {
    cout << s.name << " " << s.age << " " << s.score << endl;
}

// 引用传递（推荐，效率高）
void input_student(Student &s) {
    cin >> s.name >> s.age >> s.score;
}

int main() {
    Student s;
    input_student(s);
    print_student(s);
    return 0;
}
```

## 📚 模块四：进制与位运算

### 4.1 进制转换

| 进制 | 前缀 | 示例 |
|:---|:---|:---|
| 二进制 | `0b` | `0b1010` = 10 |
| 八进制 | `0` | `012` = 10 |
| 十进制 | 无 | `10` = 10 |
| 十六进制 | `0x` | `0xA` = 10 |

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 0b1010;   // 二进制
    int b = 012;      // 八进制
    int c = 10;       // 十进制
    int d = 0xA;      // 十六进制
    
    cout << a << " " << b << " " << c << " " << d << endl;
    // 输出: 10 10 10 10
    
    return 0;
}
```

### 4.2 位运算

| 运算符 | 名称 | 示例 | 结果 |
|:---|:---|:---|:---|
| `&` | 与 | `5 & 3` | `1` |
| `|` | 或 | `5 | 3` | `7` |
| `^` | 异或 | `5 ^ 3` | `6` |
| `~` | 非 | `~5` | `-6` |
| `<<` | 左移 | `5 << 1` | `10` |
| `>>` | 右移 | `5 >> 1` | `2` |

**位运算示例**：

```cpp
#include <iostream>
using namespace std;

int main() {
    // 与运算：两位都为1才为1
    // 5 = 101
    // 3 = 011
    // --------- &
    // 1 = 001
    cout << (5 & 3) << endl;  // 1
    
    // 或运算：有一位为1就为1
    // 5 = 101
    // 3 = 011
    // --------- |
    // 7 = 111
    cout << (5 | 3) << endl;   // 7
    
    // 异或：不同为1，相同为0
    // 5 = 101
    // 3 = 011
    // --------- ^
    // 6 = 110
    cout << (5 ^ 3) << endl;   // 6
    
    // 左移：乘以2
    cout << (5 << 1) << endl;  // 10
    cout << (5 << 2) << endl;  // 20
    
    // 右移：除以2
    cout << (10 >> 1) << endl; // 5
    
    return 0;
}
```

**实际应用**：

```cpp
// 判断奇偶
bool is_odd(int n) {
    return n & 1;  // 看最后一位
}

// 交换两个数（不用临时变量）
void swap(int &a, int &b) {
    a = a ^ b;
    b = a ^ b;
    a = a ^ b;
}

// 判断2的幂
bool is_power_of_2(int n) {
    return n > 0 && (n & (n - 1)) == 0;
}
```

## 🎮 综合项目：学生成绩管理系统 V2

使用结构体和指针重写学生管理系统：

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

const int MAX = 100;

struct Student {
    string name;
    int age;
    double scores[3];  // 三科成绩
    double avg;
};

// 输入学生信息
void input(Student &s) {
    cout << "姓名: ";
    cin >> s.name;
    cout << "年龄: ";
    cin >> s.age;
    cout << "三科成绩: ";
    cin >> s.scores[0] >> s.scores[1] >> s.scores[2];
    
    s.avg = (s.scores[0] + s.scores[1] + s.scores[2]) / 3.0;
}

// 输出学生信息
void output(const Student &s) {
    cout << left << setw(10) << s.name << " ";
    cout << s.age << " ";
    cout << fixed << setprecision(1);
    for (int i = 0; i < 3; i++) {
        cout << setw(6) << s.scores[i];
    }
    cout << setw(6) << s.avg << endl;
}

// 按平均分排序
void sort(Student arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j].avg < arr[j + 1].avg) {
                Student temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}

int main() {
    Student students[MAX];
    int n;
    
    cout << "学生人数: ";
    cin >> n;
    
    // 输入
    for (int i = 0; i < n; i++) {
        cout << "\n第" << i + 1 << "个学生:\n";
        input(students[i]);
    }
    
    // 排序
    sort(students, n);
    
    // 输出
    cout << "\n======== 成绩表 ========\n";
    cout << left << setw(10) << "姓名" << " 年龄  语文  数学  英语  平均\n";
    for (int i = 0; i < n; i++) {
        output(students[i]);
    }
    
    return 0;
}
```

## 🏛️ 三经典智慧

### 《易经》· 象数思维

**结构体就是「象」**：
- 把相关的数据组合在一起
- 形成有意义的整体
- 如同八卦组合成万物

### 《道德经》· 有无相生

**指针体现了「有无」**：
- 指针本身（有）存储的是地址
- 指向的内容（有/无）可以是空
- `nullptr` 就是「无」

### 《论语》· 名正言顺

**结构体命名的重要性**：
- 好的命名：`Student`, `Book`, `Point`
- 一目了然，见名知意
- 这是编程的「礼」
