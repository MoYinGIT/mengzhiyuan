## 📚 L1 基础练习题

### 入门题

#### 题目1：Hello Name

**题目**：输入一个名字，输出 "Hello, 名字!"

**示例**：
```
输入: 小明
输出: Hello, 小明!
```

**参考代码**：
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string name;
    cin >> name;
    cout << "Hello, " << name << "!" << endl;
    return 0;
}
```

### 函数题

#### 题目16：最大公约数

**题目**：编写函数 `gcd(a, b)`，计算a和b的最大公约数

**方法**：辗转相除法

```cpp
int gcd(int a, int b) {
    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}
```

## 💡 解题技巧

### 调试技巧

1. **输出调试**
   ```cpp
   cout << "DEBUG: i=" << i << ", val=" << val << endl;
   ```

2. **分段测试**
   - 先测试输入是否正确
   - 再测试中间结果
   - 最后验证输出

3. **边界测试**
   - 最小值
   - 最大值
   - 特殊值（0、负数等）

### 常见错误

| 错误 | 原因 | 解决 |
|:---|:---|:---|
| 数组越界 | 索引 >= 数组长度 | 检查循环条件 |
| 整数溢出 | 结果超过int范围 | 使用long long |
| 除零错误 | 除数为0 | 添加判断 |
| 死循环 | 循环条件永远为真 | 检查更新条件 |
| 忘记换行 | 缺少endl或\n | 添加换行 |
