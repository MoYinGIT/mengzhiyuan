## 📚 模块一：动态规划入门

### 1.1 什么是动态规划？

**动态规划**（Dynamic Programming，简称 DP）是一种**分阶段求解决策问题的数学思想**。

**核心思想**：
- **最优子结构**：大问题由子问题组成，大问题的最优解包含子问题的最优解
- **重叠子问题**：子问题重复出现，可以记忆化存储
- **状态转移**：通过状态转移方程，从已知推未知

**类比**：爬楼梯
```
到第 n 阶的方法 = 到第 n-1 阶的方法 + 到第 n-2 阶的方法
```

### 1.2 爬楼梯问题

**问题**：每次可以爬 1 阶或 2 阶，到第 n 阶有多少种方法？

**分析**：
```
f(1) = 1  // 只有1种：1
f(2) = 2  // 两种：1+1, 2
f(3) = f(2) + f(1) = 3  // 从2阶跨1步，或从1阶跨2步
f(n) = f(n-1) + f(n-2)  // 递推公式
```

**代码实现**：

```cpp
#include <iostream>
using namespace std;

const int MAX = 100;
long long dp[MAX];  // dp[i] 表示到第 i 阶的方法数

long long climb_stairs(int n) {
    // 基准情况
    dp[1] = 1;
    dp[2] = 2;
    
    // 状态转移
    for (int i = 3; i <= n; i++) {
        dp[i] = dp[i-1] + dp[i-2];
    }
    
    return dp[n];
}

int main() {
    int n;
    cin >> n;
    cout << climb_stairs(n) << endl;
    return 0;
}
```

### 1.3 0/1 背包问题

**问题**：有 n 个物品，每个有重量 w[i] 和价值 v[i]，背包容量为 W，求最大价值。

**状态定义**：
- `dp[i][j]`：前 i 个物品，容量为 j 时的最大价值

**状态转移**：
```
如果不选第 i 个物品：dp[i][j] = dp[i-1][j]
如果选第 i 个物品（需要 j >= w[i]）：dp[i][j] = dp[i-1][j-w[i]] + v[i]

dp[i][j] = max(不选, 选)
```

**代码实现**：

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

const int MAXN = 100;
const int MAXW = 10000;

int dp[MAXN][MAXW];
int w[MAXN], v[MAXN];

int knapsack(int n, int W) {
    // 初始化：0个物品或0容量，价值为0
    for (int i = 0; i <= n; i++) dp[i][0] = 0;
    for (int j = 0; j <= W; j++) dp[0][j] = 0;
    
    // 状态转移
    for (int i = 1; i <= n; i++) {
        for (int j = 0; j <= W; j++) {
            // 不选第i个物品
            dp[i][j] = dp[i-1][j];
            
            // 选第i个物品
            if (j >= w[i]) {
                dp[i][j] = max(dp[i][j], dp[i-1][j-w[i]] + v[i]);
            }
        }
    }
    
    return dp[n][W];
}

// 空间优化版本
int knapsack_optimized(int n, int W) {
    int dp[MAXW] = {0};
    
    for (int i = 1; i <= n; i++) {
        // 必须倒序遍历！
        for (int j = W; j >= w[i]; j--) {
            dp[j] = max(dp[j], dp[j-w[i]] + v[i]);
        }
    }
    
    return dp[W];
}

int main() {
    int n, W;
    cin >> n >> W;
    for (int i = 1; i <= n; i++) {
        cin >> w[i] >> v[i];
    }
    cout << knapsack(n, W) << endl;
    return 0;
}
```

### 1.4 最长递增子序列（LIS）

**问题**：给定序列，求最长递增子序列的长度

**示例**：`[10, 9, 2, 5, 3, 7, 101, 18]` → LIS 是 `[2, 3, 7, 101]`，长度为 4

**状态定义**：
- `dp[i]`：以第 i 个元素结尾的最长递增子序列长度

**状态转移**：
```cpp
int LIS(int arr[], int n) {
    int dp[MAXN];
    int ans = 1;
    
    for (int i = 0; i < n; i++) {
        dp[i] = 1;  // 至少包含自己
        for (int j = 0; j < i; j++) {
            if (arr[j] < arr[i]) {
                dp[i] = max(dp[i], dp[j] + 1);
            }
        }
        ans = max(ans, dp[i]);
    }
    
    return ans;
}
// 时间复杂度：O(n²)
```

### 1.5 最长公共子序列（LCS）

**问题**：两个序列的最长公共子序列长度

**示例**：`ABCD` 和 `AEBD` 的 LCS 是 `ABD`，长度为 3

**状态定义**：
- `dp[i][j]`：序列1前 i 个和序列2前 j 个的 LCS 长度

**状态转移**：
```cpp
int LCS(string s1, string s2) {
    int n = s1.length(), m = s2.length();
    int dp[MAXN][MAXN] = {0};
    
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= m; j++) {
            if (s1[i-1] == s2[j-1]) {
                dp[i][j] = dp[i-1][j-1] + 1;
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i][j-1]);
            }
        }
    }
    
    return dp[n][m];
}
```

## 📚 模块二：图论基础

### 2.1 图的基本概念

**图**（Graph）：由顶点（Vertex）和边（Edge）组成的数据结构

| 术语 | 说明 |
|:---|:---|
| **顶点** | 图中的节点 |
| **边** | 连接两个顶点的线 |
| **有向图** | 边有方向 |
| **无向图** | 边无方向 |
| **权** | 边上的数值（如距离、成本）|
| **度** | 与顶点相连的边数 |

### 2.2 图的存储

#### 邻接矩阵

```cpp
const int MAXN = 100;
int graph[MAXN][MAXN];  // graph[i][j] = 1 表示 i 到 j 有边

// 初始化
memset(graph, 0, sizeof(graph));

// 添加边
graph[1][2] = 1;  // 1 到 2 有边
graph[2][1] = 1;  // 无向图要加反向边
```

**优点**：查询快 O(1)  
**缺点**：空间 O(n²)，稀疏图浪费空间

#### 邻接表

```cpp
#include <vector>
using namespace std;

const int MAXN = 100;
vector<int> graph[MAXN];  // graph[i] 存储与 i 相邻的顶点

// 添加边
void add_edge(int u, int v) {
    graph[u].push_back(v);
    graph[v].push_back(u);  // 无向图
}

// 遍历与 u 相邻的顶点
for (int v : graph[u]) {
    cout << v << " ";
}
```

**优点**：空间 O(n+m)，适合稀疏图  
**缺点**：查询需要遍历

### 2.3 深度优先搜索（DFS）

**思想**：一条路走到黑，走不通再回头

```cpp
const int MAXN = 100;
vector<int> graph[MAXN];
bool visited[MAXN];

void dfs(int u) {
    visited[u] = true;
    cout << u << " ";  // 访问顶点
    
    for (int v : graph[u]) {
        if (!visited[v]) {
            dfs(v);
        }
    }
}

// 使用
int main() {
    // 建图...
    memset(visited, false, sizeof(visited));
    dfs(1);  // 从顶点 1 开始遍历
    return 0;
}
```

**应用场景**：
- 图的遍历
- 连通分量
- 判断环路
- 拓扑排序

### 2.4 广度优先搜索（BFS）

**思想**：层层推进，像水波纹扩散

```cpp
#include <queue>

void bfs(int start) {
    queue<int> q;
    bool visited[MAXN] = {false};
    
    q.push(start);
    visited[start] = true;
    
    while (!q.empty()) {
        int u = q.front();
        q.pop();
        cout << u << " ";  // 访问顶点
        
        for (int v : graph[u]) {
            if (!visited[v]) {
                visited[v] = true;
                q.push(v);
            }
        }
    }
}
```

**应用场景**：
- 最短路径（无权图）
- 层次遍历
- 最小步数问题

### 2.5 最短路径 - Dijkstra 算法

**问题**：带权图中，求一个顶点到其他所有顶点的最短路径

**要求**：边权非负

```cpp
#include <queue>
#include <vector>
#include <climits>
using namespace std;

const int MAXN = 100;
const int INF = INT_MAX;

// 邻接表存储，pair<权值, 终点>
vector<pair<int, int>> graph[MAXN];
int dist[MAXN];
bool visited[MAXN];

void dijkstra(int start) {
    // 初始化
    for (int i = 0; i < MAXN; i++) {
        dist[i] = INF;
        visited[i] = false;
    }
    dist[start] = 0;
    
    // 优先队列，按距离从小到大
    priority_queue<pair<int, int>, vector<pair<int, int>>, greater<>> pq;
    pq.push({0, start});
    
    while (!pq.empty()) {
        int u = pq.top().second;
        pq.pop();
        
        if (visited[u]) continue;
        visited[u] = true;
        
        for (auto &edge : graph[u]) {
            int v = edge.second;
            int w = edge.first;
            
            if (dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
                pq.push({dist[v], v});
            }
        }
    }
}
```

## 📚 模块三：综合算法

### 3.1 记忆化搜索

**思想**：递归 + 存储，避免重复计算

```cpp
// 斐波那契记忆化
long long memo[100];

long long fib(int n) {
    if (n <= 1) return n;
    if (memo[n] != 0) return memo[n];
    return memo[n] = fib(n-1) + fib(n-2);
}
```

### 3.2 贪心 + DP 对比

| 特点 | 贪心 | 动态规划 |
|:---|:---|:---|
| **局部最优** | 是 | 不一定 |
| **全局最优** | 不一定 | 是 |
| **适用** | 特定问题 | 更广泛 |
| **复杂度** | 通常较低 | 通常较高 |
| **典型问题** | 活动选择、背包分数 | 0/1背包、LIS |

## 🎮 综合项目：迷宫求解

```cpp
#include <iostream>
#include <queue>
#include <utility>
using namespace std;

const int MAX = 100;
int maze[MAX][MAX];      // 0=路, 1=墙
bool visited[MAX][MAX];
int dist[MAX][MAX];      // 到每个点的最短距离
int n, m;

// 四个方向
int dx[] = {-1, 1, 0, 0};
int dy[] = {0, 0, -1, 1};

// BFS 求最短路径
int bfs(int sx, int sy, int ex, int ey) {
    queue<pair<int, int>> q;
    q.push({sx, sy});
    visited[sx][sy] = true;
    dist[sx][sy] = 0;
    
    while (!q.empty()) {
        auto [x, y] = q.front();
        q.pop();
        
        if (x == ex && y == ey) {
            return dist[x][y];
        }
        
        for (int i = 0; i < 4; i++) {
            int nx = x + dx[i];
            int ny = y + dy[i];
            
            if (nx >= 0 && nx < n && ny >= 0 && ny < m
                && !maze[nx][ny] && !visited[nx][ny]) {
                visited[nx][ny] = true;
                dist[nx][ny] = dist[x][y] + 1;
                q.push({nx, ny});
            }
        }
    }
    
    return -1;  // 无法到达
}

int main() {
    cin >> n >> m;
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            cin >> maze[i][j];
    
    int result = bfs(0, 0, n-1, m-1);
    cout << "最短路径长度: " << result << endl;
    
    return 0;
}
```
