# C++ 竞赛编程

> **信奥竞赛唯一语言** · 适合 9 岁+ · **主线推荐**

---

## 🎯 C++ 的定位：主线推荐

**C++ 是信息学竞赛的唯一指定语言，是主线中的主线。**

| 路线 | 路径 | 适合人群 |
|:---|:---|:---|
| **主线** ⭐ | Scratch → **C++** → CSP-J/S → NOIP | 目标竞赛/升学 |
| 支线 | Scratch → Python → AI/数据 | 兴趣导向 |

### 为什么选 C++？

✅ **竞赛唯一**：CSP-J/S、NOIP、NOI 只用 C++  
✅ **升学优势**：科技特长生、强基计划、保送  
✅ **基础扎实**：语法严格，培养严谨思维  
✅ **职业广阔**：算法工程师、系统开发、游戏引擎

## 🚀 最佳起点：四年级（9-10岁）

### 为什么四年级开始？

| 年级 | 学习内容 | 目标 |
|:---:|:---|:---|
| **四年级** | **C++ L1-L2** | 语法入门 |
| 五年级 | C++ L3-L4 | 数据结构+基础算法 |
| 六年级 | C++ L5-L6 | **CSP-J 一等奖** |
| 初中 | C++ L7-L8 | **CSP-S 省一/省队** |

**关键**：四年级开始，六年级有足够时间准备 CSP-J。

### 从哪开始？

| 情况 | 起点 |
|:---|:---|
| 9-10岁，有Scratch基础 | **直接 C++ L1** |
| 9-10岁，零基础 | **C++ L1**（加把劲能跟上） |
| 需要缓冲 | [Scratch L4](../scratch/l4.md) 过渡2个月 |

## 📚 八级竞赛体系

### 基础阶段（语法入门）

| 级别 | 年级 | 核心内容 | 目标 |
|:---:|:---:|:---|:---|
| **L1 一级** | 四上 | 语法基础、输入输出、简单计算 | 熟悉C++语法 |
| **L2 二级** | 四下 | 数组、字符串、函数 | 掌握基础数据结构 |

### 进阶阶段（算法起步）

| 级别 | 年级 | 核心内容 | 目标 |
|:---:|:---:|:---|:---|
| **L3 三级** | 五上 | 进制、位运算、指针、结构体 | GESP L3 认证 |
| **L4 四级** | 五下 | 递归、排序、贪心 | GESP L4 认证 |

### 强化阶段（竞赛准备）

| 级别 | 年级 | 核心内容 | 竞赛节点 |
|:---:|:---:|:---|:---|
| **L5 五级** | 六上 | 数论、链表、分治 | **CSP-J 初赛** |
| **L6 六级** | 六下 | 动态规划、图搜索 | **CSP-J 复赛** |
| **L7 七级** | 初一 | 复杂DP、哈希、MST、最短路径 | **CSP-S** |
| **L8 八级** | 初二+ | 组合数学、图论进阶 | **NOIP/省队** |

## 🆚 C++ vs Python：为什么选 C++？

| 对比项 | C++ | Python |
|:---|:---|:---|
| **语法难度** | ⭐⭐⭐⭐ 复杂 | ⭐⭐ 简单 |
| **竞赛支持** | ✅ 唯一指定语言 | ❌ 不能用 |
| **升学帮助** | ✅ 科技特长生、保送 | ❌ 无直接帮助 |
| **执行速度** | ⚡ 极快 | 🐢 较慢 |
| **适合年龄** | **9岁+** | 10岁+ |
| **学习时长** | 2-3年到竞赛 | 兴趣导向，无固定 |

### 关键结论

- **目标竞赛/升学** → **必须 C++**，Python 无用
- **兴趣/AI** → 可选 Python，但要接受无法竞赛
- **不确定** → **选 C++**，保留选择权，基础牢了转 Python 很快

## 🚀 快速开始

### 第一个 C++ 程序

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "你好，世界！" << endl;
    return 0;
}
```

**输出**：
```
你好，世界！
```

### 编译运行

```bash
# 编译
g++ hello.cpp -o hello

# 运行
./hello
```

## 💡 Scratch  vs C++ 对照

| 概念 | Scratch | C++ |
|:---|:---|:---|
| 输出 | `说"你好"` | `cout << "你好";` |
| 变量 | 创建变量积木 | `int x = 10;` |
| 循环 | `重复执行10次` | `for(int i=0; i<10; i++)` |
| 条件 | `如果...那么...` | `if (x > 0)` |

**学 C++ 不难**，有 Scratch 基础的孩子，1-2个月就能上手。

## ❓ 常见问题

**Q1: 四年级学 C++ 会不会太早？**  
A: **不会**。9岁抽象思维快速发展，是代码编程的最佳起点。

**Q2: C++ 比 Python 难，为什么不学 Python？**  
A: **竞赛必须用 C++**。Python 简单但无法参加信奥，对升学无直接帮助。

**Q3: 没学过 Scratch 可以直接学 C++ 吗？**  
A: **9岁以上可以**。Scratch 是培养兴趣，不是必须前置。

**Q4: 学多久可以参加 CSP-J？**  
A: 四年级开始，六年级（约2年）可以参加 CSP-J 入门组。

**Q5: 女孩适合学 C++ 竞赛吗？**  
A: **非常适合**。NOI 有女生专项名额，竞争更小，机会更大。
