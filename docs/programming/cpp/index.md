## 🎯 为什么选择 C++？

### C++ 的特点

| 特点 | 说明 |
|:---|:---|
| **性能卓越** | 运行速度极快，适合竞赛 |

| **竞赛标准** | NOIP、CSP-J/S 官方指定语言 |
| **底层控制** | 深入理解计算机原理 |

- ✅ 12岁以上，有编程基础

## 📝 基础语法预览

### 变量与类型

```cpp
#include <iostream>
using namespace std;

int main() {
    int age = 12;           // 整数
    double height = 1.55;   // 小数
    char grade = 'A';       // 字符
    string name = "小明";   // 字符串
    bool is_student = true; // 布尔
    
    cout << "姓名：" << name << endl;
    cout << "年龄：" << age << endl;
    
    return 0;
}
```

### 条件与循环

```cpp
// if-else
int score = 85;
if (score >= 90) {
    cout << "优秀" << endl;
} else if (score >= 80) {
    cout << "良好" << endl;
} else {
    cout << "加油" << endl;
}

// for 循环
for (int i = 0; i < 5; i++) {
    cout << i << " ";
}
// 输出：0 1 2 3 4
```

### 数组

```cpp
int scores[5] = {85, 92, 78, 90, 88};

for (int i = 0; i < 5; i++) {
    cout << scores[i] << " ";
}
```

## 🏆 信息学竞赛

### 竞赛体系

```
CSP-J 入门组 → CSP-S 提高组 → NOIP → NOI → IOI
    ↓              ↓
  初中阶段      高中阶段
```

### 重要竞赛

| 竞赛 | 级别 | 说明 |
|:---|:---|:---|
| **CSP-J** | 入门级 | 初中生参加 |
| **CSP-S** | 提高级 | 高中生参加 |
| **NOIP** | 省级 | 省联赛 |
| **NOI** | 国家级 | 全国赛，金牌保送清华北大 |
| **IOI** | 国际级 | 国际奥赛 |

### 升学优势

- CSP-J/S 二等奖以上：科技特长生招生
- NOI 金牌：保送清华北大
- NOI 银牌：强基计划优惠

## 🏛️ 三经典与 C++

### 《易经》· 穷变通久

学 C++ 遇到难题是常态：
- **指针** 搞不懂？换个方式理解（把它当作地址标签）
- **递归** 想不通？画图分解步骤
- **算法** 学不会？从简单例子入手

穷则变，变则通，通则久。

### 《道德经》· 知常曰明

C++ 让你「知其然，更知其所以然」：
- 内存怎么分配？
- 指针怎么工作？
- 程序怎么运行？

深入底层，才能真正理解计算机。
